Inversore di Numeri

Ti viene dato un numero e devi invertire le sue cifre.

Task

Crea una funzione che prenda in input un numero e restituisca il numero con le sue cifre invertite.

Input

L'input sarà un numero intero positivo.

Output

L'output sarà il numero di input con le sue cifre invertite.

Example

Input

"472"

Output

"274"
