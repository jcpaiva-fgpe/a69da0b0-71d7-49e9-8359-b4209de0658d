Odwracacz Liczb

Dostajesz liczbę i musisz odwrócić jej cyfry.

Task

Stwórz funkcję, która przyjmuje liczbę i zwraca liczbę odwróconą o jej cyfry.

Input

Wejściem będzie dodatnia liczba całkowita.

Output

Wyjściem będzie liczba wejściowa z odwróconymi cyframi.

Example

Input

"472"

Output

"274"
