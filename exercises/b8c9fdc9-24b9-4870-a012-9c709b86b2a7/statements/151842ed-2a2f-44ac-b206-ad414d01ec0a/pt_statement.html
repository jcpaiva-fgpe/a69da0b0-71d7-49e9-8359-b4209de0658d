Chapéu Seletor

É seu primeiro dia em Hogwarts e você está animado para descobrir a qual Casa Hogwarts você pertence. Para descobrir em qual casa você pertence, você deve usar o Chapéu Seletor. O Chapéu Seletor analisa seus traços de personalidade e decide qual casa combina com você: Grifinória, Lufa-Lufa, Corvinal, ou Sonserina. Sua missão é criar um programa que simule o processo de decisão do Chapéu Seletor.

Task

Crie uma função que receba um número inteiro de 1 a 100 e retorne uma string indicando a qual Casa Hogwarts a pessoa pertence, de acordo com os seguintes critérios: Se o número é: - Divisível por 4, a função deve retornar 'Grifinória' - Divisível por 3, a função deve retornar 'Lufa-Lufa' - Divisível por 2, a função deve retornar 'Corvinal' - Não divisível por 2, 3 ou 4, a função deve retornar 'Sonserina'

Input

Um número inteiro de 1 a 100.

Output

Uma string indicando a Casa Hogwarts correspondente ao número de entrada.

Example

Input

"12"

Output

"Gryffindor"
