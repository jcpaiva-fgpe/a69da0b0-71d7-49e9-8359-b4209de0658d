Sorting Hat

It's your first day at Hogwarts and you're excited to discover which Hogwarts House you belong to. To find out which house you belong, you have to wear the Sorting Hat. The Sorting Hat analyzes your personality traits and decides which house suits you best: Gryffindor, Hufflepuff, Ravenclaw, or Slytherin. Your task is to create a program that simulates the Sorting Hat's decision process.

Task

Create a function that receives an integer from 1 to 100 and returns a string indicating which Hogwarts House the person belongs to, according to the following criteria: If the number is: - Divisible by 4, the function should return 'Gryffindor' - Divisible by 3, the function should return 'Hufflepuff' - Divisible by 2, the function should return 'Ravenclaw' - Not divisible by 2, 3 or 4, the function should return 'Slytherin'

Input

An integer from 1 to 100.

Output

A string indicating the Hogwarts House corresponding to the input number.

Example

Input

"12"

Output

"Gryffindor"
