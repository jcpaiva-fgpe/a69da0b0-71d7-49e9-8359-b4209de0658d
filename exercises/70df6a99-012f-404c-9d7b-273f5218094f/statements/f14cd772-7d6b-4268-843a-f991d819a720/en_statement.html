Multiplication Table

You are tasked to create a multiplication table from 1 to n. Each number in the table should be equal to the product of the row and column index. For example, the number in row i and column j should be equal to i*j.

Task

Write a function that takes an integer n and returns a 2D list representing the multiplication table from 1 to n.

Input

The input parameter is an integer n (1 <= n <= 20).

Output

The function should return a 2D list with n rows and n columns, where each element at row i and column j is equal to i*j.

Example

Input

"3"

Output

"[[1, 2, 3], [2, 4, 6], [3, 6, 9]]"
