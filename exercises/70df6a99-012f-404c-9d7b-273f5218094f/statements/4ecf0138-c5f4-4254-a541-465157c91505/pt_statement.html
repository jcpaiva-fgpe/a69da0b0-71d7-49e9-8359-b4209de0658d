Tabela de Multiplicação

Você recebeu a tarefa de criar uma tabela de multiplicação de 1 até n. Cada número na tabela deve ser igual ao produto do índice da linha e da coluna. Por exemplo, o número na linha i e coluna j deve ser igual a i*j.

Task

Escreva uma função que recebe um inteiro n e retorne uma lista 2D representando uma tabela de multiplicação de 1 até n.

Input

O parâmetro de entrada é um inteiro n (1 <= n <= 20).

Output

A função deve retornar uma lista 2D com n linhas e n colunas, onde cada elemento na linha i e na coluna j é igual a i*j.

Example

Input

"3"

Output

"[[1, 2, 3], [2, 4, 6], [3, 6, 9]]"
