Tabellina

Devi creare una tabellina da 1 a n. Ogni numero nella tabella dovrebbe essere il prodotto dell'indice di riga e di colonna. Ad esempio, il numero nella riga i e colonna j dovrebbe essere uguale a i*j.

Task

Scrivere una funzione che prende un intero n e restituisce una lista 2D che rappresenta una tabellina da 1 a n.

Input

Il parametro di input è un intero n (1 <= n <= 20).

Output

La funzione dovrebbe restituire una lista 2D con n righe e n colonne, dove ogni elemento nella riga i e nella colonna j è uguale a i*j.

Example

Input

"3"

Output

"[[1, 2, 3], [2, 4, 6], [3, 6, 9]]"
