Tabliczka mnożenia

Twoim zadaniem jest stworzenie tabliczki mnożenia od 1 do n. Każda liczba w tablicy powinna być równa iloczynowi indeksu wiersza i kolumny. Na przykład liczba w wierszu i i kolumnie j powinna być równa i*j.

Task

Napisz funkcję, która bierze liczbę całkowitą n i zwraca listę 2D reprezentującą tabliczkę mnożenia od 1 do n.

Input

Parametrem wejściowym jest liczba całkowita n (1 <= n <= 20).

Output

Funkcja powinna zwrócić listę 2D z n wierszami i n kolumnami, gdzie każdy element w wierszu i i kolumnie j jest równy i*j.

Example

Input

"3"

Output

"[[1, 2, 3], [2, 4, 6], [3, 6, 9]]"
