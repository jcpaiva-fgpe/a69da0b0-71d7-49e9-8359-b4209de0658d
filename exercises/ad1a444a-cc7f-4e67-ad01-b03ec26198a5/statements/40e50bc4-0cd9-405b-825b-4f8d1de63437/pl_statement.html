Największa liczba całkowita na liście

Musisz znaleźć największą liczbę całkowitą na liście.

Task

Napisz funkcję, która przyjmuje listę liczb całkowitych i zwraca największą liczbę całkowitą.

Input

Lista liczb całkowitych

Output

Największa liczba całkowita na liście.

Example

Input

"[1,2,3,4,5,6]"

Output

"6"
