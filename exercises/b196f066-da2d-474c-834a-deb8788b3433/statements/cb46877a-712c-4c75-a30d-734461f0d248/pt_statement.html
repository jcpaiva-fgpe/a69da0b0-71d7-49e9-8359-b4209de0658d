Checador de Fibonacci

A sequência de Fibonacci é uma série de números, em que cada número é a soma dos dois números anteriores

Task

Escreva uma função que recebe um inteiro e retorna se é ou não um número de Fibonacci

Input

Um inteiro n para checar se é um número de Fibonacci

Output

Verdadeiro se o número for um número de Fibonacci, falso caso contrário

Example

Input

"13"

Output

"True"
