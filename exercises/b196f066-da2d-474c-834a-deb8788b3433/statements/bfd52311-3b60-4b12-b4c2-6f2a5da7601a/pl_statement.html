Sprawdzacz ciągu Fibonacciego

Ciąg Fibonacciego to seria liczb, w której każda liczba jest sumą dwóch poprzednich liczb

Task

Napisz funkcję, która otrzymuje liczbę całkowitą i zwraca, czy jest ona liczbą Fibonacciego, czy nie

Input

Liczba całkowita n do sprawdzenia, czy jest ona liczbą Fibonacciego

Output

Prawda, jeśli liczba jest liczbą Fibonacciego, fałsz w przeciwnym razie

Example

Input

"13"

Output

"True"
