Possibilità di Palindromo

Un palindromo è una parola che si scrive allo stesso modo in avanti e all'indietro. Devi creare una funzione che prende in input una stringa e restituisce un valore booleano che indica se è possibile o meno creare un palindromo utilizzando tutti i caratteri nella stringa di input.

Task

Crea una funzione che prende una stringa e controlla se è possibile creare un palindromo utilizzando tutti i caratteri nella stringa. La funzione dovrebbe restituire True se è possibile, altrimenti restituire False.

Input

L'input consiste in una stringa di caratteri ASCII stampabili.

Output

La funzione dovrebbe restituire un valore booleano che indica se è possibile o meno creare un palindromo utilizzando tutti i caratteri nella stringa di input.

Example

Input

"hello"

Output

"False"
