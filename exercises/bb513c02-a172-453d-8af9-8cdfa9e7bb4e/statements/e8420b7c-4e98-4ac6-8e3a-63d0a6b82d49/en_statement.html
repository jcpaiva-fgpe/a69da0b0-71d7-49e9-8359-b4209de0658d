Palindrome Possibility

A palindrome is a word that is spelled the same way forwards and backwards. You need to create a function that takes in a string and returns a boolean value that indicates whether or not it is possible to create a palindrome using all of the characters in the input string.

Task

Create a function that takes a string and checks if it is possible to create a palindrome using all the characters in the string. The function should return True if it is possible, otherwise return False.

Input

The input will consist of a string of printable ASCII characters.

Output

The function should return a boolean value that indicates whether or not it is possible to create a palindrome using all of the characters in the input string.

Example

Input

"hello"

Output

"False"
