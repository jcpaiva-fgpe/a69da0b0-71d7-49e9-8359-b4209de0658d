Possibilidade de Palíndromo

Um palíndromo é uma palavra que é soletrada da mesma forma de frente para trás. Você precisa criar uma função que recebe uma string e retorna um valor booleano que indica se é possível ou não criar um palíndromo usando todos os caracteres na string de entrada.

Task

Crie uma função que recebe uma string e verifica se é possível criar um palíndromo usando todos os caracteres da string. A função deve retornar True se for possível, caso contrário, retornará False.

Input

A entrada consistirá em uma string de caracteres ASCII imprimíveis.

Output

A função deve retornar um valor booleano que indica se é possível ou não criar um palíndromo usando todos os caracteres na string de entrada.

Example

Input

"hello"

Output

"False"
