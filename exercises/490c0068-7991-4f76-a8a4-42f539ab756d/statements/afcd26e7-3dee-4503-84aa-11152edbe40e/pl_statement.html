Silnia

W matematyce, silnią liczby całkowitej nieujemnej n oznacza się n! i definiuje jako iloczyn wszystkich liczb naturalnych dodatnich do n włączenie. Na przykład, 5! = 5 x 4 x 3 x 2 x 1 = 120

Task

Napisz funkcję, która pobiera nieujemną liczbę całkowitą n i zwraca jej silnię

Input

Nieujemna liczba całkowita n

Output

Silnia danej liczby n

Example

Input

"5"

Output

"120"
