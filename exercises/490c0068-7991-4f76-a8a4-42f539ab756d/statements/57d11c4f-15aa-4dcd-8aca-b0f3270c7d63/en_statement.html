Factorial

In maths, factorial of a non-negative integer number n is represented by n! and is defined as the product of all positive integers up to n. For example, 5! = 5 x 4 x 3 x 2 x 1 = 120

Task

Create a function that receives a non-negative integer n and returns its factorial

Input

A non-negative integer number n

Output

The factorial of the given number n

Example

Input

"5"

Output

"120"
