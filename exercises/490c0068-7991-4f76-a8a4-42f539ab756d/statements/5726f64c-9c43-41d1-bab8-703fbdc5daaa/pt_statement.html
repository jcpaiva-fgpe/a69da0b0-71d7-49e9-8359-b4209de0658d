Fatorial

Em matemática, o fatorial de um número inteiro não negativo n é representado por n! e é definido como o produto de todos os inteiros positivos até n. Por exemplo, 5! = 5 x 4 x 3 x 2 x 1 = 120

Task

Crie uma função que receba um inteiro não negativo n e retorne seu fatorial

Input

Um número inteiro não negativo n

Output

O fatorial do número dado n

Example

Input

"5"

Output

"120"
