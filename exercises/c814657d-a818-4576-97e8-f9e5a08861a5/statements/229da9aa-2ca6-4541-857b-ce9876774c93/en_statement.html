Finding Highest and Lowest Numbers

You are required to write a function that takes a list as an argument and returns the highest and lowest numbers of the list.

Task

Write a function that takes a list of integers and returns a tuple containing the highest and lowest number.

Input

A list of integers separated by space

Output

A tuple containing two integers - the highest and lowest numbers of the list

Example

Input

"1 2 3 4 5"

Output

"(5, 1)"
