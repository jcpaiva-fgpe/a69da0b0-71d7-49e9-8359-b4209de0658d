Znajdź najwyższe i najniższe liczby

Należy napisać funkcję, która przyjmuje listę jako argument i zwraca najwyższą i najniższą liczbę z listy.

Task

Napisz funkcję, która przyjmuje listę liczb całkowitych i zwraca krotkę zawierającą najwyższą i najniższą liczbę.

Input

Lista liczb całkowitych oddzielonych spacjami

Output

Krotka zawierająca dwie liczby całkowite - najwyższą i najniższą liczbę z listy

Example

Input

"1 2 3 4 5"

Output

"(5, 1)"
