Gerador de Tabela de Multiplicação

Como desenvolvedor, você precisa gerar tabelas de multiplicações para ajudar estudantes com exercícios de cálculo. Como há diferentes níveis de aprendizado, o programa deve receber como entrada o tamanho da tabela de multiplicação.

Task

Crie uma função que gere uma tabela de multiplicação com base no tamanho recebido como entrada.

Input

A função recebe um inteiro n (1 <= n <= 20), que representa o tamanho da tabela de multiplicação

Output

A função deve retornar uma lista de listas que representa a tabela de multiplicação, onde cada elemento [i][j] é a multiplicação de i e j

Example

Input

"3"

Output

"[[0, 0, 0], [0, 1, 2], [0, 2, 4]]"
