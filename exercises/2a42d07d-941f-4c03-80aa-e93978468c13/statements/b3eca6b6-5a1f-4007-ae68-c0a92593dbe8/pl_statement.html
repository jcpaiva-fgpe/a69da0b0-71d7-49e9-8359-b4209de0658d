Generator Tabliczki Mnożenia

Jako deweloper potrzebujesz generować tablice mnożenia, aby pomóc uczniom w ćwiczeniach związanych z obliczeniami. Ponieważ istnieją różne poziomy nauki, program musi przyjmować jako wejście rozmiar tabliczki mnożenia.

Task

Stwórz funkcję, która generuje tabliczkę mnożenia na podstawie rozmiaru otrzymanego jako dane wejściowe.

Input

Funkcja przyjmuje liczbę całkowitą n (1 <= n <= 20), która reprezentuje rozmiar tabliczki mnożenia

Output

Funkcja zwraca listę list reprezentującą tabliczkę mnożenia, w której każdy element [i][j] to iloczyn i i j

Example

Input

"3"

Output

"[[0, 0, 0], [0, 1, 2], [0, 2, 4]]"
