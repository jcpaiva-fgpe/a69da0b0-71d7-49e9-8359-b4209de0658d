Generatore di Tabella di Moltiplicazione

Come sviluppatore, devi generare le tabelle di moltiplicazione per aiutare gli studenti con gli esercizi di calcolo. Poiché ci sono diversi livelli di apprendimento, il programma deve ricevere la dimensione della tabella di moltiplicazione in input.

Task

Creare una funzione che genera una tabella di moltiplicazione in base alla dimensione ricevuta in input.

Input

La funzione riceve un intero n (1 <= n <= 20), che rappresenta la dimensione della tabella di moltiplicazione

Output

La funzione deve restituire una lista di liste che rappresenta la tabella di moltiplicazione, dove ogni elemento [i][j] è la moltiplicazione di i e j

Example

Input

"3"

Output

"[[0, 0, 0], [0, 1, 2], [0, 2, 4]]"
