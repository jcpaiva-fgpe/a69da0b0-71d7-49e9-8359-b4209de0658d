Multiplication Table Generator

As a developer, you need to generate multiplication tables to help students with calculation exercises. As there are different learning levels, the program must receive the size of the multiplication table as an input.

Task

Create a function that generates a multiplication table based on the size received as input.

Input

The function receives an integer n (1 <= n <= 20), which represents the size of the multiplication table

Output

The function must return a list of lists representing the multiplication table, where each element [i][j] is the multiplication of i and j

Example

Input

"3"

Output

"[[0, 0, 0], [0, 1, 2], [0, 2, 4]]"
