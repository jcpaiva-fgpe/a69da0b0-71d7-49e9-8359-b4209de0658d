Walidator Złożoności Hasła

Masz za zadanie stworzyć funkcję walidującą złożoność hasła dla strony internetowej. Strona wymaga, aby użytkownicy tworzyli hasło, używając następujących reguł: - Hasło musi zawierać co najmniej 8 znaków. - Hasło musi zawierać co najmniej jedną wielką literę i jedną małą literę. - Hasło musi zawierać co najmniej jedną cyfrę. - Hasło musi zawierać co najmniej jeden znak specjalny (@, #, $, %, &, *).

Task

Stwórz funkcję, która przyjmie hasło typu string i zwróci wartość logiczną wskazującą, czy hasło spełnia zasady strony dotyczące złożoności hasła.

Input

Ciąg znaków hasła.

Output

Wartość logiczna wskazująca, czy hasło spełnia zasady strony dotyczące złożoności hasła.

Example

Input

"P@ssword123"

Output

"True"
