Password Complexity Validator

You are tasked to create a password complexity validator function for a website. The website requires its users to create a password using the following rules: - The password must contain at least 8 characters. - The password must contain at least one uppercase and one lowercase letter. - The password must contain at least one number. - The password must contain at least one special character (@, #, $, %, &, *).

Task

Create a function that will take in a string password and return a boolean value indicating whether or not the password meets the website's password complexity rules.

Input

A password string.

Output

A boolean value indicating whether or not the password meets the website's password complexity rules.

Example

Input

"P@ssword123"

Output

"True"
