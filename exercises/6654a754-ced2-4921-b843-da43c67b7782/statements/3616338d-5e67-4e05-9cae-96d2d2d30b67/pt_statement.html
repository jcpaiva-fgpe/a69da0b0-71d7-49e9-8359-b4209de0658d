Validador de Complexidade de Senha

Você tem a tarefa de criar uma função validadora de complexidade de senha para um site. O site exige que seus usuários criem senhas seguindo as seguintes regras: - A senha deve conter pelo menos 8 caracteres. - A senha deve conter pelo menos uma letra maiúscula e uma letra minúscula. - A senha deve conter pelo menos um número. - A senha deve conter pelo menos um caractere especial (@, #, $, %, &, *).

Task

Crie uma função que recebe uma senha do tipo string e retorna um valor booleano indicando se a senha atende ou não às regras de complexidade de senha do site.

Input

Uma string de senha.

Output

Um valor booleano indicando se a senha atende ou não às regras de complexidade de senha do site.

Example

Input

"P@ssword123"

Output

"True"
