Znalezienie najdłuższego wspólnego prefiksu

Dany jest tablica stringów, znaleźć najdłuższy wspólny prefiks.

Task

Napisz funkcję, która przyjmuje tablicę łańcuchów i zwraca najdłuższy wspólny prefiks.

Input

Tablica łańcuchów znaków.

Output

Łańcuch znaków.

Example

Input

"[\"flower\",\"flow\",\"flight\"]"

Output

"\"fl\""
