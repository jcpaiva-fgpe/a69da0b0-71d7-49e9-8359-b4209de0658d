Encontrando o Prefixo Comum mais Longo

Dado um array de strings, encontre o prefixo comum mais longo entre eles.

Task

Escreva uma função que receba um array de strings e retorne o prefixo comum mais longo.

Input

Um array de strings.

Output

Uma string.

Example

Input

"[\"flower\",\"flow\",\"flight\"]"

Output

"\"fl\""
