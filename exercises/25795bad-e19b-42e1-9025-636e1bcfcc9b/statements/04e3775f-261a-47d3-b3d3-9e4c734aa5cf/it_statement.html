Somma delle cifre

Viene dato un intero e devi trovare la somma delle sue cifre.

Task

Crea una funzione che prende un intero in input e restituisce la somma delle sue cifre.

Input

Un intero n (0 <= n <= 10^9)

Output

L'intero che rappresenta la somma delle cifre del numero in input.

Example

Input

"357"

Output

"15"
