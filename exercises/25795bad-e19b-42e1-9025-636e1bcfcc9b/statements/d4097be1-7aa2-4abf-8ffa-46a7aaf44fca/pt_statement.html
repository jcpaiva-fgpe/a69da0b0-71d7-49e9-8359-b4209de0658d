Soma dos Dígitos

Você recebe um número inteiro e precisa encontrar a soma de seus dígitos.

Task

Crie uma função que receba um número inteiro como entrada e retorne a soma de seus dígitos.

Input

Um número inteiro n (0 <= n <= 10^9)

Output

O inteiro que representa a soma dos dígitos do número de entrada.

Example

Input

"357"

Output

"15"
