Suma cyfr

Masz liczbę całkowitą i musisz znaleźć sumę jej cyfr.

Task

Utwórz funkcję, która przyjmuje jako dane wejściowe liczbę całkowitą i zwraca sumę jej cyfr.

Input

Liczba całkowita n (0 <= n <= 10^9)

Output

Liczba całkowita reprezentująca sumę cyfr danej liczby.

Example

Input

"357"

Output

"15"
