Soma de um Array

Você recebeu um array de inteiros e sua tarefa é encontrar a soma de todos os elementos no array.

Task

Escreva uma função que receba um array de inteiros e retorne a soma de todos os elementos no array.

Input

A entrada será um array de inteiros.

Output

A saída será um único inteiro representando a soma de todos os elementos no array.

Example

Input

[1,2,3,4,5]

Output

15
