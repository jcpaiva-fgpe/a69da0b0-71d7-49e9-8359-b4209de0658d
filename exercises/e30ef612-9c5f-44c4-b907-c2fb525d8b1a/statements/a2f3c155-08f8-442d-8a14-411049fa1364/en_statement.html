Grade Calculator

You are a teacher and want to automate the process of calculating grades for your class.

Task

Write a function that takes a list of grades and returns the average grade, rounded to two decimal places.

Input

The input will be a list of integers representing grades (ranging from 0 to 100).

Output

The output will be a float representing the average grade rounded to two decimal places.

Example

Input

"[85, 90, 92, 88, 86, 89]"

Output

"88.33"
