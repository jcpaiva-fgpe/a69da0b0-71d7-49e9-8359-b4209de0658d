Calculadora de Notas

Você é um professor e quer automatizar o processo de cálculo de notas para sua turma.

Task

Escreva uma função que receba uma lista de notas e retorne a média aritmética das notas, arredondada para duas casas decimais.

Input

A entrada será uma lista de números inteiros representando notas (variando de 0 a 100).

Output

A saída será um número de ponto flutuante que representa a média aritmética das notas, arredondado para duas casas decimais.

Example

Input

"[85, 90, 92, 88, 86, 89]"

Output

"88.33"
