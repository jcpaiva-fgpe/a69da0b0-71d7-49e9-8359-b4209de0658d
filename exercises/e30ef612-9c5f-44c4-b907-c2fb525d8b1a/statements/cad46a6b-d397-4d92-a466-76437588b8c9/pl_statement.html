Kalkulator Ocen

Jesteś nauczycielem i chcesz zautomatyzować proces obliczania ocen dla twojej klasy.

Task

Napisz funkcję, która przyjmuje listę ocen i zwraca średnią ocenę, zaokrągloną do dwóch miejsc po przecinku.

Input

Wejście będzie listą liczb całkowitych reprezentujących oceny (od 0 do 100).

Output

Wyjście będzie liczbą zmiennoprzecinkową reprezentującą średnią ocenę zaokrągloną do dwóch miejsc po przecinku.

Example

Input

"[85, 90, 92, 88, 86, 89]"

Output

"88.33"
