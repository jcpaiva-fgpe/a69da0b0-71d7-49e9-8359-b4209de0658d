Znajdowanie Maksymalnych Cyfr

Dostałeś kilka liczb, teraz musisz znaleźć cyfrę o maksymalnej wartości w tej liczbie.

Task

Podana liczba całkowita, znajdź i zwróć cyfrę o maksymalnej wartości w tym numerze.

Input

Wejściem będzie nieujemna liczba całkowita n.

Output

Zwróć cyfrę o maksymalnej wartości w liczbie całkowitej.

Example

Input

"123456789"

Output

"9"
