Szyfrowanie słów

Zostałeś zatrudniony jako analityk danych w rządowej agencji o najwyższym stopniu tajności. Twoim pierwszym zadaniem jest stworzenie programu, który szyfruje poufne informacje przesyłane drogą mailową. Twój szef chce, abyś stworzył prosty program szyfrujący słowa w każdym e-mailu.

Task

Napisz funkcję, która przyjmuje zdanie jako dane wejściowe i szyfruje każde słowo w zdaniu poprzez odwrócenie kolejności znaków.

Input

Dane wejściowe to ciąg znaków składających się ze słów oddzielonych spacjami. Zdanie będzie mieć maksymalnie 1000 znaków.

Output

Wyjściem powinien być ciąg znaków składających się z zaszyfrowanych słów oddzielonych spacjami.

Example

Input

"hello world"

Output

"olleh dlrow"
