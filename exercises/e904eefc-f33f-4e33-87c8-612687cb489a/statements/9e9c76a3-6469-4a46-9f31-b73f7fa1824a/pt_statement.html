Criptografia de Palavras

Você foi contratado como analista de dados em uma agência governamental altamente secreta. Sua primeira missão é criar um programa que criptografe informações sensíveis que estão sendo enviadas por e-mail. Seu chefe quer que você crie um programa simples de criptografia que criptografe as palavras em cada e-mail.

Task

Escreva uma função que recebe como entrada uma frase e criptografa cada palavra da frase invertendo suas letras.

Input

A entrada será uma string composta de palavras separadas por espaços. A frase terá no máximo comprimento 1000.

Output

A saída deve ser uma string composta pelas palavras criptografadas separadas por espaços.

Example

Input

"hello world"

Output

"olleh dlrow"
