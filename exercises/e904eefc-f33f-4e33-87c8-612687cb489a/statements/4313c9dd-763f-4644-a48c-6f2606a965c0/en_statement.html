Word Encryption

You have been hired as a data analyst at a top secret government agency. Your first assignment is to create a program that encrypts sensitive information that is being sent over email. Your boss wants you to create a simple encryption program that encrypts the words in each email.

Task

Write a function that takes in a sentence as input and encrypts each word in the sentence by reversing its letters.

Input

The input will be a string consisting of words separated by spaces. The sentence will have at most length 1000.

Output

The output should be a string consisting of the encrypted words separated by spaces.

Example

Input

"hello world"

Output

"olleh dlrow"
