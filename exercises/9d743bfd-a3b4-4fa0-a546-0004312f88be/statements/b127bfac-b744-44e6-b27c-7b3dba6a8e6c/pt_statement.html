Troca de Dois Números

Na programação, existem várias maneiras de trocar dois números sem usar uma terceira variável. Este pequeno exercício se concentra no entendimento das operações lógicas e matemáticas com variáveis

Task

Crie uma função que troque dois números sem usar uma terceira variável

Input

A função deve receber dois números inteiros como argumentos, separados por ponto e vírgula (;)

Output

A função deve retornar uma string com os valores trocados separados por vírgula (,).

Example

Input

"2;3"

Output

"3,2"
