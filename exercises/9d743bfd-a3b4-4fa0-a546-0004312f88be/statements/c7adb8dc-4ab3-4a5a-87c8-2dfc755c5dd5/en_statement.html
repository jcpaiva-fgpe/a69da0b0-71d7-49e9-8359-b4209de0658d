Swap Two Numbers

In programming, you have a variety of ways to swap two numbers without using a third variable. This small exercise focuses on understanding of logical and mathematical operations with variables

Task

Create a function that swaps two numbers without using a third variable.

Input

The function should take two integers as arguments separated by a semicolon (;)

Output

The function should return a string with the swapped values separated by a comma (,).

Example

Input

"2;3"

Output

"3,2"
