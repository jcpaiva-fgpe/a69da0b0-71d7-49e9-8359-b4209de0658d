Email Domain Extractor

You are building an email database for your company and need to extract the domain of each email from a list of email addresses. You need to create a function to extract the domain from a given email string.

Task

Create a function that takes an email address and returns only the domain of that email.

Input

The input will be a string that contains a valid email address.

Output

The output should be a string that contains only the domain of the email.

Example

Input

"john.doe@example.com"

Output

"example.com"
