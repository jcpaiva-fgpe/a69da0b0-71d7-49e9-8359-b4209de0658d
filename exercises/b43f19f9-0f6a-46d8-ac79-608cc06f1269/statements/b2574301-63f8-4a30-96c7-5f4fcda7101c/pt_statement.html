Extrator de Domínio de Email

Você está construindo um banco de dados de e-mails para a sua empresa e precisa extrair o domínio de cada e-mail em uma lista de endereços de e-mail. Você precisa criar uma função para extrair o domínio de uma string de e-mail fornecida.

Task

Crie uma função que receba um endereço de e-mail e retorne somente o domínio desse e-mail.

Input

A entrada será uma string que contém um endereço de e-mail válido.

Output

A saída deve ser uma string que contenha apenas o domínio do e-mail.

Example

Input

"john.doe@example.com"

Output

"example.com"
