Misturador de Cores

Você está desenvolvendo um software de mistura de tintas. O software recebe duas cores primárias e deve retornar a cor mista resultante. As cores primárias são vermelho, amarelo e azul. As cores misturadas resultantes são verde (azul + amarelo), laranja (vermelho + amarelo) e roxo (vermelho + azul).

Task

Dadas duas cores de entrada, retorne a cor mista resultante.

Input

Duas strings representando as cores primárias a serem misturadas, separadas por ponto e vírgula.

Output

Uma string representando a cor mista resultante.

Example

Input

"red;yellow"

Output

"orange"
