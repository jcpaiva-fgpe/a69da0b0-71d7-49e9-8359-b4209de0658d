Color Mixer

You are developing a paint-mixing software. The software receives two primary colors and must return the resulting mixed color. The primary colors are red, blue, and yellow. The resulting mixed colors are green (blue + yellow), orange (red + yellow), and purple (red + blue).

Task

Given two input colors, return the resulting mixed color.

Input

Two strings representing the primary colors to be mixed, separated by a semicolon.

Output

A string representing the resulting mixed color.

Example

Input

"red;yellow"

Output

"orange"
