Mieszanie Kolorów

Tworzysz oprogramowanie do mieszania farb. Oprogramowanie otrzymuje dwa kolory podstawowe i musi zwrócić wynikowy kolor mieszany. Kolory podstawowe to czerwony, niebieski i żółty. Wynikowe kolory mieszane to zielony (niebieski + żółty), pomarańczowy (czerwony + żółty) i fioletowy (czerwony + niebieski).

Task

Podając dwa kolory wejściowe, zwróć wynikowy kolor mieszany.

Input

Dwie ciągi znaków reprezentujące kolory podstawowe do wymieszania, oddzielone średnikiem.

Output

Ciag znaków reprezentujący wynikowy kolor mieszany.

Example

Input

"red;yellow"

Output

"orange"
