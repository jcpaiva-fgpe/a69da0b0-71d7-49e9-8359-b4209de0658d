Longest Palindromic Substring

A Palindromic Substring is a string that remains the same even after it has been reversed. For example, “racecar” is a Palindromic Substring because it reads the same backward as forwards. We have to find the longest Palindromic Substring in the given string.

Task

Create a function that receives a string and returns the longest Palindromic Substring.

Input

A string s consisting of lowercase characters, where 1 ≤ len(s) ≤ 1000

Output

The longest Palindromic Substring in string s

Example

Input

"babad"

Output

"bab"
