Maior Substring Palindrômica

Uma Substring Palindrômica é uma string que permanece a mesma mesmo quando invertida. Por exemplo, “ovo” é uma Substring Palindrômica porque é igual lida reversamente ou da forma normal. Temos que encontrar a maior Substring Palindrômica na string dada.

Task

Crie uma função que recebe uma string e retorna a maior Substring Palindrômica.

Input

Uma string s consistindo de caracteres minúsculos, onde 1 ≤ len(s) ≤ 1000

Output

A maior Substring Palindrômica na string s

Example

Input

"babad"

Output

"bab"
