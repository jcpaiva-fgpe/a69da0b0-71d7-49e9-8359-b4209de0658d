Fibonacci Sequence

The Fibonacci sequence is a mathematical sequence that starts with 0 and 1, and each subsequent number is the sum of the two previous numbers. Write a function that receives an integer n and returns the first n numbers of the Fibonacci sequence.

Task

Write a function that receives an integer n and returns the first n numbers in the Fibonacci sequence.

Input

An integer n (1 ≤ n ≤ 100)

Output

The first n numbers of the Fibonacci sequence, separated by spaces

Example

Input

"6"

Output

"0 1 1 2 3 5"
