Sequência de Fibonacci

A sequência de Fibonacci é uma sequência matemática que começa com 0 e 1, e cada número subsequente é a soma dos dois números anteriores. Escreva uma função que recebe um inteiro n e retorna os primeiros n números da sequência de Fibonacci.

Task

Escreva uma função que recebe um inteiro n e retorna os primeiros n números da sequência de Fibonacci.

Input

Um inteiro n (1 ≤ n ≤ 100)

Output

Os primeiros n números da sequência de Fibonacci, separados por espaços

Example

Input

"6"

Output

"0 1 1 2 3 5"
