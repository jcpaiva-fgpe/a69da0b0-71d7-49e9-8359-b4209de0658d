Ciag Fibonacciego

Ciag Fibonacciego to ciag matematyczny, który zaczyna sie od 0 i 1, a każda kolejna liczba jest suma dwóch poprzednich liczb. Napisz funkcję, która otrzymuje całkowitą wartość n i zwraca pierwsze n liczb ciagu Fibonacciego.

Task

Napisz funkcję, która otrzymuje całkowitą wartość n i zwraca pierwsze n liczb ciagu Fibonacciego.

Input

Calkowita liczba n (1 ≤ n ≤ 100)

Output

Pierwsze n liczb ciagu Fibonacciego, oddzielone od siebie spacja

Example

Input

"6"

Output

"0 1 1 2 3 5"
