Calculadora de Ano Bissexto

Um ano bissexto é um ano que é divisível exatamente por 4, exceto pelos anos do século. Os anos do século, como 1900 e 2000, devem ser divisíveis por 400 para serem considerados anos bissextos. Escreva uma função que calcula se um ano dado é bissexto ou não.

Task

Escreva uma função que receba um ano (inteiro) como entrada e o retorno informa se ele é um ano bissexto ou não. Se o ano for bissexto, retornar True. Caso contrário, retornar False.

Input

Um número inteiro representando o ano

Output

Um valor booleano indicando se o ano é bissexto ou não

Example

Input

"2012"

Output

"True"
