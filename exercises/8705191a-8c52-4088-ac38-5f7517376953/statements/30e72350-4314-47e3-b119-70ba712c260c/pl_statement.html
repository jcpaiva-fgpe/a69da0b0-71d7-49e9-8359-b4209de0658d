Kalkulator przestępnych lat

Rok przestępny to rok, który dzieli się dokładnie przez 4, z wyjątkiem lat stuletnich. Lata stuletnie (np. 1900, 2000) muszą być podzielne przez 400, aby były uważane za rok przestępny. Napisz funkcję, która obliczy, czy dany rok jest rokiem przestępnym czy nie.

Task

Napisz funkcję, która przyjmuje rok (liczba całkowita) jako wejście i zwraca, czy jest to rok przestępny czy nie. Jeśli rok jest przestępny, zwróć True. W przeciwnym razie, zwróć False.

Input

Liczba całkowita reprezentująca rok

Output

Wartość logiczna wskazująca, czy rok jest rokiem przestępnym czy nie

Example

Input

"2012"

Output

"True"
