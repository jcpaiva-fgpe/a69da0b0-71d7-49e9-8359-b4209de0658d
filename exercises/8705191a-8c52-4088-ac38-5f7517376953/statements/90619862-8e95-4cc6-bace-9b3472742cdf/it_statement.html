Calcolatore di Anni Bissextile

Un anno bisestile è un anno che è esattamente divisibile per 4 ad eccezione degli anni secolari. Gli anni secolari (ad esempio 1900, 2000) devono essere divisibili per 400 per essere considerati anni bisestili. Scrivi una funzione che calcoli se un dato anno è bisestile o meno.

Task

Scrivi una funzione che prenda un anno (intero) come input e restituisca se è un anno bisestile o meno. Se l'anno è bisestile, restituisci True. Altrimenti, restituisci False.

Input

Un numero intero che rappresenta l'anno

Output

Un valore booleano che indica se l'anno è bisestile o meno

Example

Input

"2012"

Output

"True"
