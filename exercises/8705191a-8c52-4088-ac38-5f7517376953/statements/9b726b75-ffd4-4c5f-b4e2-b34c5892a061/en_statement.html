Leap Year Calculator

A leap year is a year that is exactly divisible by 4 except for century years. Century years (e.g. 1900, 2000) must be divisible by 400 to be considered leap years. Write a function that calculates if a given year is a leap year or not.

Task

Write a function that takes a year (integer) as input and returns whether it is a leap year or not. If the year is a leap year, return True. Otherwise, return False.

Input

An integer representing the year

Output

A boolean value indicating whether the year is a leap year or not

Example

Input

"2012"

Output

"True"
