DNA Complementation

DNA, the carrier of genetic information, is composed of four nucleotide bases: adenine (A), thymine (T), cytosine (C), and guanine (G). The sequence of these bases forms genes, which in turn create proteins. Sometimes, it's needed to find the complement of a DNA strand, which means to find and return the base pairs for each nucleotide. The base pairs are A-T and C-G. For example, if the original strand is 'ACTG', its complement is 'TGAC'. You need to implement a function that will receive a DNA strand and return its complement.

Task

Create a function that receives a DNA strand (a string containing only the letters A, C, T or G) and returns its complement (a string containing the complementary letters for each nucleotide).

Input

The input will be a string containing only the letters A, C, T or G.

Output

The output should be the complement of the input string, where the pairs of nucleotide bases are A-T and C-G.

Example

Input

"ACTG"

Output

"TGAC"
