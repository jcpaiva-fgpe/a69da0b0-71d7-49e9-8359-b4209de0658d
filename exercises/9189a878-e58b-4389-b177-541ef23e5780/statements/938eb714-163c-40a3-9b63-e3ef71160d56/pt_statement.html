Complementação de DNA

O DNA, o portador de informações genéticas, é composto por quatro bases nucleotídicas: adenina (A), timina (T), citosina (C) e guanina (G). A sequência dessas bases forma genes, que por sua vez criam proteínas. Às vezes, é necessário encontrar o complemento de uma fita de DNA, o que significa encontrar e retornar os pares de bases para cada nucleotídeo. Os pares de bases são A-T e C-G. Por exemplo, se a fita original for 'ACTG', o complemento é 'TGAC'. Você precisa implementar uma função que receberá uma fita de DNA e retornará o seu complemento.

Task

Crie uma função que recebe uma fita de DNA (uma string contendo apenas as letras A, C, T ou G) e retorna o seu complemento (uma string contendo as letras complementares para cada nucleotídeo).

Input

A entrada será uma string contendo apenas as letras A, C, T ou G.

Output

A saída deve ser o complemento da string de entrada, onde os pares de bases de nucleotídeos são A-T e C-G.

Example

Input

"ACTG"

Output

"TGAC"
