Conversor de String para Inteiro

Você está desenvolvendo uma aplicação e precisa converter um string para um inteiro. Você decide escrever uma função para isso.

Task

Crie uma função que aceite uma string e a converta em um inteiro. A string sempre representará um inteiro válido.

Input

Uma string que contém um inteiro válido

Output

Um valor inteiro

Example

Input

"123"

Output

123
