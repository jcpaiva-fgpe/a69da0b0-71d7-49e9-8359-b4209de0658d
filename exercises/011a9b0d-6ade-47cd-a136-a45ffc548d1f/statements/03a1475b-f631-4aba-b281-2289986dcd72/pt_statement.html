Próximo Número Primo

Você recebe um número. Sua tarefa é encontrar o próximo número primo.

Task

Escreva uma função que receba um número inteiro e retorne o próximo número primo. Se o número de entrada for primo, retorne esse número.

Input

Um inteiro n (0 <= n <= 10^6)

Output

O próximo número primo maior que n

Example

Input

"7"

Output

"11"
