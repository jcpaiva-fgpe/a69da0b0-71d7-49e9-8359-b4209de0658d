Prossimo Numero Primo

Ti è dato un numero. Il tuo compito è trovare il prossimo numero primo.

Task

Scrivi una funzione che prenda un intero e restituisca il prossimo numero primo. Se il numero di input è primo, restituisci quel numero stesso.

Input

Un intero n (0 <= n <= 10^6)

Output

Il prossimo numero primo maggiore di n

Example

Input

"7"

Output

"11"
