Następna Liczba Pierwsza

Dostajesz liczbę. Twoim zadaniem jest znalezienie następnej liczby pierwszej.

Task

Napisz funkcję, która przyjmuje liczbę całkowitą i zwraca następną liczbę pierwszą. Jeśli liczba wejściowa jest pierwsza, zwróć tę liczbę.

Input

Liczba całkowita n (0 <= n <= 10^6)

Output

Następna liczba pierwsza większa niż n

Example

Input

"7"

Output

"11"
