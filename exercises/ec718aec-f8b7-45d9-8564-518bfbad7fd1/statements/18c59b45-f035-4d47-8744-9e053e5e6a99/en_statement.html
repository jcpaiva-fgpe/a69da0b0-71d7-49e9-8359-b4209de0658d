RGB to Hexadecimal Color Converter

In web development, we frequently use colors in different formats. One of the most commonly used formats is RGB (Red, Green, Blue). It is often required to convert these RGB colors to hexadecimal format while building web pages. Hexadecimal is a 6-digit representation of colors using numbers and letters.

Task

Write a function that takes in the RGB color values as input and converts it into its hexadecimal color code equivalent. The function should return a string consisting of the 6-digit hexadecimal color code.

Input

Input consists of three integers separated by semicolon (;) representing the values of Red, Green, Blue colors.

Output

The output should be a string consisting of a 6-digit hexadecimal code representing the equivalent color.

Example

Input

"255;165;0"

Output

"ffa500"
