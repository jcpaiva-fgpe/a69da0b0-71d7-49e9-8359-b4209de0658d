Conversor de Cores RGB para Hexadecimal

No desenvolvimento web, frequentemente usamos cores em diferentes formatos. Um dos formatos mais comumente usados é o RGB (vermelho, verde, azul). Frequentemente é necessário converter essas cores RGB para formato hexadecimal ao criar páginas da web. Hexadecimal é uma representação de 6 dígitos de cores usando números e letras.

Task

Escreva uma função que receba os valores de cores RGB como entrada e os converta em seu equivalente em código de cor hexadecimal. A função deve retornar uma string consistindo no código de cor hexadecimal de 6 dígitos.

Input

A entrada consiste em três inteiros separados por ponto e vírgula (;) representando os valores das cores Vermelho, Verde e Azul.

Output

A saída deve ser uma string que consiste em um código hexadecimal de 6 dígitos representando a cor equivalente.

Example

Input

"255;165;0"

Output

"ffa500"
