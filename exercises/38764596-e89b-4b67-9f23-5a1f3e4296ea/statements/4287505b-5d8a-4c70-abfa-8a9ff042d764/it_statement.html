Fattori del Numero Primo

Dato un intero N, trovare tutti i fattori primi di N. Un fattore primo è un fattore di N che è un numero primo. Ad esempio, i fattori primi di 15 sono 3 e 5.

Task

Scrivi una funzione che prende un intero N in input e restituisce una lista di tutti i suoi fattori primi in ordine crescente.

Input

L'input sarà un singolo intero positivo N.

Output

L'output sarà una lista di tutti i fattori primi di N in ordine crescente.

Example

Input

"15"

Output

"[3, 5]"
