Fatores Número Primo

Dado um número inteiro N, encontre todos os fatores primos de N. Um fator primo é um fator de N que é um número primo. Por exemplo, os fatores primos de 15 são 3 e 5.

Task

Escreva uma função que recebe um inteiro N como entrada e retorna uma lista de todos os seus fatores primos em ordem crescente.

Input

A entrada será um único inteiro positivo N.

Output

A saída será uma lista de todos os fatores primos de N em ordem crescente.

Example

Input

"15"

Output

"[3, 5]"
