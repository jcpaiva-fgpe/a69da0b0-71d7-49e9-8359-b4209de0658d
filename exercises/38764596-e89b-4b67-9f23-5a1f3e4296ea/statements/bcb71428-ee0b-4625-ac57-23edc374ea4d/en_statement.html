Prime Number Factors

Given an integer N, find all prime factors of N. A prime factor is a factor of N which is a prime number. For example, the prime factors of 15 are 3 and 5.

Task

Write a function that takes an integer N as input and returns a list of all its prime factors in increasing order.

Input

Input will be a single positive integer N.

Output

Output will be a list of all prime factors of N in increasing order.

Example

Input

"15"

Output

"[3, 5]"
