Maximizar Lucro

Você trabalha para uma empresa de comércio que lida com diferentes moedas. Seu trabalho é desenvolver uma função que determine o lucro máximo que pode ser obtido a partir de uma lista dada de taxas de câmbio de moeda. Você pode comprar e vender moedas quantas vezes quiser, mas precisa vender uma moeda antes de comprá-la novamente. Além disso, você pode manter apenas uma moeda de cada vez.

Task

Dada uma lista de taxas de câmbio de moeda, escreva uma função para determinar o lucro máximo que pode ser obtido. Você pode assumir que a lista de entrada não está vazia e contém pelo menos dois elementos.

Input

A entrada será uma lista de inteiros positivos representando as taxas de câmbio de moeda.

Output

A saída deve ser um único número representando o lucro máximo que pode ser obtido.

Example

Input

"[7,1,5,3,6,4]"

Output

"7"
