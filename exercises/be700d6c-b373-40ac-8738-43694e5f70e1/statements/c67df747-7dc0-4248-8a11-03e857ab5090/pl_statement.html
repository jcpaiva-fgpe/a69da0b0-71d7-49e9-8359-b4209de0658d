Maksymalizuj Zysk

Pracujesz dla firmy handlującej różnymi walutami. Twoim zadaniem jest opracowanie funkcji, która określi maksymalny zysk, jaki można osiągnąć na podstawie podanej listy kursów wymiany walut. Możesz kupować i sprzedawać waluty dowolną ilość razy, ale musisz sprzedać walutę przed jej ponownym kupieniem. Ponadto możesz trzymać tylko jedną walutę naraz.

Task

Na podstawie podanej listy kursów wymiany walut napisz funkcję, która określi maksymalny zysk, jaki można osiągnąć. Możesz założyć, że lista wejściowa nie jest pusta i zawiera co najmniej dwa elementy.

Input

Wejściem będzie lista dodatnich liczb całkowitych reprezentujących kursy wymiany walut.

Output

Wyjściem powinna być pojedyncza liczba reprezentująca maksymalny zysk, jaki można osiągnąć.

Example

Input

"[7,1,5,3,6,4]"

Output

"7"
