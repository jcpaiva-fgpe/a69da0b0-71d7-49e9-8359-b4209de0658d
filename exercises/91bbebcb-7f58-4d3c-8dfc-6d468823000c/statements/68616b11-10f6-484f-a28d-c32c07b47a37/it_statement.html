Verificatore di stringhe uniche

Devi scrivere una funzione che riceve una stringa e restituisce True se tutti i caratteri nella stringa sono unici e False altrimenti.

Task

Scrivi una funzione che riceve una stringa e restituisce True se tutti i caratteri nella stringa sono unici e False altrimenti.

Input

Una stringa di qualsiasi lunghezza.

Output

True se tutti i caratteri nella stringa sono unici e False altrimenti.

Example

Input

"hello"

Output

"False"
