Matrix Transposition

In linear algebra, the transpose of a matrix is an operator which flips a matrix over its diagonal. This operation is often used to simplify certain calculations, such as determining if a system of linear equations is solvable. Write a function that receives a matrix and returns its transpose, that is, a matrix where the rows and columns are swapped.

Task

Create a function that receives a matrix and returns its transpose.

Input

The function receives a matrix as a list of lists of integers.

Output

The function should return the transpose of the matrix, that is, a matrix where the rows and columns are swapped.

Example

Input

"[[1, 2, 3], [4, 5, 6], [7, 8, 9]]"

Output

"[[1, 4, 7], [2, 5, 8], [3, 6, 9]]"
