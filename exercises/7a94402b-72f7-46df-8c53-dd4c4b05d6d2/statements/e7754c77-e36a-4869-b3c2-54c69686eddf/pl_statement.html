Transpozycja macierzy

W matematyce liniowej transpozycja macierzy to operator, który odwraca macierz względem jej przekątnej. Ta operacja jest często stosowana do uproszczenia niektórych obliczeń, takich jak określenie, czy system równań liniowych jest rozwiązywalny. Napisz funkcję, która otrzymuje macierz i zwraca jej transpozycję, czyli macierz, w której wiersze i kolumny są zamienione miejscami.

Task

Utwórz funkcję, która otrzymuje macierz i zwraca jej transpozycję.

Input

Funkcja otrzymuje macierz jako listę list liczb całkowitych.

Output

Funkcja powinna zwrócić transpozycję macierzy, czyli macierz, w której wiersze i kolumny są zamienione miejscami.

Example

Input

"[[1, 2, 3], [4, 5, 6], [7, 8, 9]]"

Output

"[[1, 4, 7], [2, 5, 8], [3, 6, 9]]"
