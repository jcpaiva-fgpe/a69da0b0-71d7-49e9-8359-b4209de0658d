Transposição de Matriz

Na álgebra linear, a transposição de uma matriz é um operador que inverte uma matriz sobre sua diagonal. Essa operação é frequentemente usada para simplificar certos cálculos, como determinar se um sistema de equações lineares é solucionável. Escreva uma função que recebe uma matriz e devolve sua transposta, ou seja, uma matriz em que as linhas e colunas são trocadas.

Task

Crie uma função que recebe uma matriz e retorna sua transposta.

Input

A função recebe uma matriz como uma lista de listas de inteiros.

Output

A função deve retornar a transposta da matriz, ou seja, uma matriz em que as linhas e colunas são trocadas.

Example

Input

"[[1, 2, 3], [4, 5, 6], [7, 8, 9]]"

Output

"[[1, 4, 7], [2, 5, 8], [3, 6, 9]]"
