Inverter palavras em uma string

Você recebeu uma string. Você precisa inverter todas as palavras nessa string e retornar uma nova string sem alterar a ordem das palavras como um todo.

Task

Escreva uma função que receba uma string como parâmetro de entrada e retorne uma string em que todas as palavras sejam invertidas sem alterar sua ordem na frase.

Input

A entrada da função é uma string de n(1<=n<=10^3) número de caracteres/alfabetos. A string contém apenas um espaço entre duas palavras consecutivas.

Output

A saída da função é uma string em que cada palavra é invertida na string original, mas a ordem de todas as palavras ainda é a mesma.

Example

Input

"The quick brown fox jumps over the lazy dog"

Output

"ehT kciuq nworb xof spmuj revo eht yzal god"
