Reverse words in a string

You have been given a string. You have to reverse all the words in that string and return a new string without changing the order of words as a whole.

Task

Write a function that takes a string as an input parameter and returns a string such that all words are reversed without changing their order in the sentence.

Input

The input of the function is a string of n(1<=n<=10^3) number of characters/alphabets. The string contains only one space between two consecutive words.

Output

The output of the function is a string where each word is reversed in the original string but the order of all words is still the same.

Example

Input

"The quick brown fox jumps over the lazy dog"

Output

"ehT kciuq nworb xof spmuj revo eht yzal god"
