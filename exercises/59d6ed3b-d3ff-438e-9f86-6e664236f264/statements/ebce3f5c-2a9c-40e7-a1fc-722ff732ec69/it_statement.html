Invertire le parole in una stringa

Ti è stata data una stringa. Devi invertire tutte le parole in quella stringa e restituire una nuova stringa senza cambiare l'ordine delle parole nel loro complesso.

Task

Scrivi una funzione che prenda una stringa come parametro di ingresso e restituisca una stringa in cui tutte le parole sono invertite senza modificare il loro ordine nella frase.

Input

L'ingresso della funzione è una stringa di n(1<=n<=10^3) numero di caratteri/alfabeti. La stringa contiene solo uno spazio tra due parole consecutive.

Output

L'uscita della funzione è una stringa in cui ciascuna parola è invertita nella stringa originale, ma l'ordine di tutte le parole è sempre lo stesso.

Example

Input

"The quick brown fox jumps over the lazy dog"

Output

"ehT kciuq nworb xof spmuj revo eht yzal god"
