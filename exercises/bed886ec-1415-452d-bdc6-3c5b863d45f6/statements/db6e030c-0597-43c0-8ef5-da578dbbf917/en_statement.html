Array Intersection

You are given two arrays of integers, find their intersection.

Task

Write a function that takes in two arrays of integers and returns their intersection.

Input

The function should take in two arrays of integers.

Output

The function should return an array of integers, representing the intersection of the input arrays.

Example

Input

"[1,2,4,5,6];[2,3,6,9]"

Output

"[2, 6]"
