Intersezione degli Array

Ti vengono date due array di interi, trova la loro intersezione.

Task

Scrivi una funzione che prenda due array di interi e restituisca la loro intersezione.

Input

La funzione dovrebbe prendere due array di interi.

Output

La funzione dovrebbe restituire un array di interi, rappresentando l'intersezione degli array in input.

Example

Input

"[1,2,4,5,6];[2,3,6,9]"

Output

"[2, 6]"
