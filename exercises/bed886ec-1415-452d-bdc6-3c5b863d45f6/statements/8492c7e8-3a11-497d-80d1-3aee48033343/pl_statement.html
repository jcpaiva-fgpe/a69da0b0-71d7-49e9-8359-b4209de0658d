Przecięcie dwóch tablic

Masz dwie tablice z liczbami całkowitymi, znajdź ich przecięcie.

Task

Napisz funkcję, która bierze dwie tablice z liczbami całkowitymi i zwraca ich przecięcie.

Input

Funkcja powinna przyjmować dwie tablice z liczbami całkowitymi.

Output

Funkcja powinna zwrócić tablicę z liczbami całkowitymi, reprezentującą przecięcie wejściowych tablic.

Example

Input

"[1,2,4,5,6];[2,3,6,9]"

Output

"[2, 6]"
