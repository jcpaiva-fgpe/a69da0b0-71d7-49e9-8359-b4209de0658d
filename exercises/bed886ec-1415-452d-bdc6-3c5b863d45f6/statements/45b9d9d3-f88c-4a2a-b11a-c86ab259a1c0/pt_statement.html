Interseção de Arrays

Você recebe duas matrizes de inteiros, encontre a interseção delas.

Task

Escreva uma função que receba duas matrizes de inteiros e retorne sua interseção.

Input

A função deve receber duas matrizes de inteiros.

Output

A função deve retornar uma matriz de inteiros, representando a interseção das matrizes de entrada.

Example

Input

"[1,2,4,5,6];[2,3,6,9]"

Output

"[2, 6]"
