The Longest Word in a Sentence

You are writing a program that needs to find the longest word in a given sentence. The sentence may contain punctuation marks and special characters. Your program should return only the longest word itself, without any punctuation marks or spaces.

Task

Write a function that takes in a string representing a sentence, and returns the longest word in the sentence. The output should not contain any punctuation marks or spaces.

Input

The input will be a string that represents a sentence. The sentence may contain any character.

Output

The output should be a string representing the longest word in the sentence. The result should not contain any punctuation marks or spaces.

Example

Input

"The quick brown fox jumps over the lazy dog."

Output

"quick"
