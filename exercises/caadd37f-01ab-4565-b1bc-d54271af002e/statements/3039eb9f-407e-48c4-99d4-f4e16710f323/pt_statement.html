A Maior Palavra de uma Sentença

Você está escrevendo um programa que precisa encontrar a maior palavra em uma determinada sentença. A sentença pode conter sinais de pontuação e caracteres especiais. Seu programa deverá retornar apenas a maior palavra em si, sem quaisquer sinais de pontuação ou espaços.

Task

Escreva uma função que receba uma string representando uma sentença e retorne a maior palavra na sentença. A saída não deve conter quaisquer sinais de pontuação ou espaços.

Input

A entrada será uma string que representa uma sentença. A sentença pode conter qualquer caractere.

Output

A saída deve ser uma string representando a maior palavra na sentença. O resultado não deve conter quaisquer sinais de pontuação ou espaços.

Example

Input

"The quick brown fox jumps over the lazy dog."

Output

"quick"
