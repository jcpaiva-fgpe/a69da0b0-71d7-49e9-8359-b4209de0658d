Fibonacci Squared

The Fibonacci sequence is a famous sequence in mathematics, where each number is the sum of the two preceding ones, starting from 0 and 1. In this exercise, you should square each number of the sequence and save them in a list.

Task

Create a function that receives an integer n (n>=0) and returns a list containing the first n squared numbers of the Fibonacci sequence.

Input

An integer n (n>=0).

Output

A list containing n integers.

Example

Input

"6"

Output

"[0, 1, 1, 4, 9, 25]"
