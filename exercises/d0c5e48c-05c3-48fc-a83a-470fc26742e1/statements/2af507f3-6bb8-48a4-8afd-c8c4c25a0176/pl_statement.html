Kwadratowy ciąg Fibonacciego

Ciąg Fibonacciego to słynny ciąg matematyczny, w którym każda liczba jest sumą dwóch poprzednich, zaczynając od 0 i 1. W tym ćwiczeniu należy podnieść do kwadratu każdą liczbę ciągu i zapisać je na liście.

Task

Stwórz funkcję, która przyjmuje liczbę całkowitą n (n>=0) i zwraca listę zawierającą pierwsze n początkowych liczb kwadratowych ciągu Fibonacciego.

Input

Liczba całkowita n (n>=0).

Output

Lista zawierająca n liczb całkowitych.

Example

Input

"6"

Output

"[0, 1, 1, 4, 9, 25]"
