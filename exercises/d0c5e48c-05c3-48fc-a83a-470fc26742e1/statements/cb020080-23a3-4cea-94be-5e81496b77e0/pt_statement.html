Fibonacci ao quadrado

A sequência de Fibonacci é uma famosa sequência na matemática, em que cada número é a soma dos dois números precedentes, começando em 0 e 1. Neste exercício, você deve elevar cada número da sequência ao quadrado e guardá-los em uma lista.

Task

Crie uma função que recebe um inteiro n (n>=0) e retorna uma lista contendo os primeiros n números ao quadrado da sequência de Fibonacci.

Input

Um inteiro n (n>=0).

Output

Uma lista contendo n inteiros

Example

Input

"6"

Output

"[0, 1, 1, 4, 9, 25]"
