Verificatore di Parentesi Bilanciate

Ti è stata data una stringa che potrebbe contenere parentesi, parentesi quadre o graffe. Il tuo compito è scrivere una funzione che controlla se la stringa ha parentesi, parentesi quadre e graffe bilanciate.

Task

Scrivi una funzione che accetta una stringa in input e restituisce True se le parentesi, parentesi quadre e graffe nella stringa sono bilanciate, altrimenti restituisce False.

Input

L'input sarà una singola stringa contenente parentesi, parentesi quadre e/o graffe.

Output

L'output sarà un valore booleano True o False.

Example

Input

"([]{()})"

Output

"True"
