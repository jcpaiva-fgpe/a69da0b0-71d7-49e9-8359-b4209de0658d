Sprawdzacz równowagi nawiasów

Masz dane ciąg znaków, zawierający nawiasy, kwadratowe, klamry lub ich mieszanki. Twoim zadaniem jest napisanie funkcji, która sprawdza, czy w podanym ciągu nawiasy są zbalansowane.

Task

Napisz funkcję, która przyjmie ciąg znaków jako argument i zwróci True, jeśli nawiasy, kwadratowe i klamry w ciągu są zbalansowane, w przeciwnym razie zwróci False.

Input

Argumentem wejściowym jest pojedynczy ciąg znaków zawierający nawiasy, kwadratowe i/lub klamry.

Output

Zwracana wartość to wartość boolowska True lub False.

Example

Input

"([]{()})"

Output

"True"
