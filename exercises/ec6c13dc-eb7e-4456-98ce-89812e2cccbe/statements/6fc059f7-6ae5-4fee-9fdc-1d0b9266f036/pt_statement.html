Verificador de Parênteses Equilibrados

Você recebeu uma string que pode conter parênteses, colchetes ou chaves. Sua tarefa é escrever uma função que verifica se a string possui parênteses, colchetes e chaves equilibrados.

Task

Escreva uma função que receba uma string como entrada e retorne True se os parênteses, colchetes e chaves da string estiverem equilibrados, caso contrário, retorne False.

Input

A entrada será uma única string contendo parênteses, colchetes e/ou chaves.

Output

A saída será um valor booleano True ou False.

Example

Input

"([]{()})"

Output

"True"
