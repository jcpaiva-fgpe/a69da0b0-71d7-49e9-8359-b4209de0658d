Find the Median

In statistics, the median is the value separating the higher half from the lower half of a data sample. It is a measure of central tendency and is often used to establish the middle value of a data set. Your task is to find the median of a given set of numbers.

Task

Write a function that takes in a list of integers and returns the median value of the list. If the list has an even number of values, return the average of the two middle values.

Input

The function takes in a list of integers.

Output

The function returns a single integer which is the median value of the input list.

Example

Input

[5,3,1,2,4]

Output

3
