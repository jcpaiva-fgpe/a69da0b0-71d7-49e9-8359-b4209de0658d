Znajdź medianę

W statystyce mediana to wartość dzieląca połowę wyższą od połowy niższej próbki danych. Jest to miara tendencji centralnej i często służy do ustalenia środkowej wartości zbioru danych. Twoim zadaniem jest znalezienie mediany danego zbioru liczb.

Task

Napisz funkcję, która przyjmuje listę liczb całkowitych i zwraca wartość mediany listy. Jeśli lista ma parzystą liczbę wartości, zwróć średnią dwóch środkowych wartości.

Input

Funkcja przyjmuje listę liczb całkowitych.

Output

Funkcja zwraca pojedynczą liczbę całkowitą, która jest wartością mediany listy wejściowej.

Example

Input

[5,3,1,2,4]

Output

3
