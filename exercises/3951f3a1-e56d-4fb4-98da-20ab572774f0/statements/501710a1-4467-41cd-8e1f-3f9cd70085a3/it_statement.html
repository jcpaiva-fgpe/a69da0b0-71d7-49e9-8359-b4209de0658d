Trova la mediana

In statistica, la mediana è il valore che separa la metà superiore dalla metà inferiore di un campione di dati. È una misura di tendenza centrale ed è spesso utilizzata per stabilire il valore medio di un insieme di dati. Il tuo compito è trovare la mediana di un determinato insieme di numeri.

Task

Scrivi una funzione che prende in input una lista di interi e restituisce il valore mediano della lista. Se la lista ha un numero pari di valori, restituisci la media dei due valori centrali.

Input

La funzione riceve in input una lista di interi.

Output

La funzione restituisce un singolo intero che è il valore mediano della lista di input.

Example

Input

[5,3,1,2,4]

Output

3
