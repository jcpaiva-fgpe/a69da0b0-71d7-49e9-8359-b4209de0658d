Encontre a Mediana

Na estatística, a mediana é um valor que separa a metade maior e a metade menor de uma amostra de dados. É uma medida de tendência central e é frequentemente usada para encontrar o valor médio de um conjunto de dados. Sua tarefa é encontrar a mediana de um conjunto de números fornecido.

Task

Escreva uma função que receba uma lista de inteiros e retorne o valor da mediana da lista. Se a lista tiver um número par de valores, retorne a média dos dois valores do meio.

Input

A função recebe uma lista de inteiros.

Output

A função retorna um único inteiro que é o valor da mediana da lista de entrada.

Example

Input

[5,3,1,2,4]

Output

3
