Person Information

You have been given some information about a person and you need to store it and display it back to the user.

Task

Create a program that takes in a person's name, age, and address. Store this information and display it back to the user in a formatted string.

Input

The input will be three strings: name (30 characters maximum), address (100 characters maximum), and age (3 digits maximum)

Output

The output will be a formatted string containing the person's name, age, and address.

Example

Input

"John Doe;123 Main St;27"

Output

"Name: John Doe\nAge: 27\nAddress: 123 Main St"
