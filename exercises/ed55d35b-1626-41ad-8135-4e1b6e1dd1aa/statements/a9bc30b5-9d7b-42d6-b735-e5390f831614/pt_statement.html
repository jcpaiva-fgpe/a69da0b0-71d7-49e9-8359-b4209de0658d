Informação de Pessoa

Você recebeu algumas informações sobre uma pessoa e precisa armazená-las e exibi-las de volta ao usuário.

Task

Crie um programa que receba o nome, idade e endereço de uma pessoa. Armazene estas informações e exiba-as de volta ao usuário em uma string formatada.

Input

A entrada serão três strings: nome (máximo de 30 caracteres), endereço (máximo de 100 caracteres) e idade (máximo de 3 dígitos)

Output

A saída será uma string formatada contendo o nome, a idade e o endereço da pessoa.

Example

Input

"John Doe;123 Main St;27"

Output

"Name: John Doe\nAge: 27\nAddress: 123 Main St"
