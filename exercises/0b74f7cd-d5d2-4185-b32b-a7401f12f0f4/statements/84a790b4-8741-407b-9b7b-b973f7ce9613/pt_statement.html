Calculando o Índice de Massa Corporal

O Índice de Massa Corporal (IMC) é o cálculo do peso de uma pessoa dividido pela altura ao quadrado. Embora o IMC não seja uma medida perfeita, ela dá uma avaliação bastante precisa de quanto do corpo é composta por gordura.

Task

Escreva uma função para calcular o IMC de uma pessoa, dado o seu peso e altura em unidades métricas.

Input

A entrada consistirá em dois números de ponto flutuante positivos - peso (em kg) e altura (em metros).

Output

A saída será um único número de ponto flutuante - o IMC da pessoa.

Example

Input

"70;1.8"

Output

"21.6"
