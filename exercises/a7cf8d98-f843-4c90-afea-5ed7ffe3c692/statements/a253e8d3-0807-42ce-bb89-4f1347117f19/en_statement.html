Word Frequency Counter

You need to write a program that counts how many times each word appears in a given text.

Task

Given a text, count how many times each word appears and return the total count in descending order.

Input

A string consisting of words separated by spaces. The text contains only lowercase letters and spaces between words.

Output

A dictionary with each word as a key and its frequency count as the value, sorted in descending order by frequency.

Example

Input

"the quick brown fox jumps over the lazy dog"

Output

"{\"the\": 2, \"over\": 1, \"lazy\": 1, \"jumps\": 1, \"fox\": 1, \"quick\": 1, \"brown\": 1, \"dog\": 1}"
