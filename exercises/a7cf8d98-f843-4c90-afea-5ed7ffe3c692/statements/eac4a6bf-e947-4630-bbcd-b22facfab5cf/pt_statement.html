Contador de Frequência de Palavras

Você precisa escrever um programa que conte quantas vezes cada palavra aparece em um texto fornecido.

Task

Dado um texto, conte quantas vezes cada palavra aparece e retorne a contagem total em ordem decrescente.

Input

Uma string contendo palavras separadas por espaços. O texto contém apenas letras minúsculas e espaços entre as palavras.

Output

Um dicionário com cada palavra como chave e sua contagem de frequência como valor, ordenado em ordem decrescente por frequência.

Example

Input

"the quick brown fox jumps over the lazy dog"

Output

"{\"the\": 2, \"over\": 1, \"lazy\": 1, \"jumps\": 1, \"fox\": 1, \"quick\": 1, \"brown\": 1, \"dog\": 1}"
