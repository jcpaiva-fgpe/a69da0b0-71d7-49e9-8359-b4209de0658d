Palavra String Revertida por Palavra

Você recebe uma String composta de palavras separadas por espaços. Sua tarefa é inverter a String por palavra.

Task

Escreva uma função que receba uma entrada de String e retorne a String invertida por palavra.

Input

A String de entrada contém apenas caracteres alfabéticos e espaços.

Output

A String de saída deve ser a String de entrada invertida por palavra.

Example

Input

"Hello World"

Output

"World Hello"
