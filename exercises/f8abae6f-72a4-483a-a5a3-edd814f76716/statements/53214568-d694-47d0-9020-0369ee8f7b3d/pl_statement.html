Odwrócona Ciągła Słowo za Słowo

Masz dany ciąg ze słowami oddzielonymi spacjami. Twoim zadaniem jest odwrócenie ciągu słowo po słowie.

Task

Napisz funkcję, która przyjmuje ciąg wejściowy i zwraca odwrócony ciąg słowo za słowo.

Input

Wejściowy ciąg zawiera tylko znaki alfabetyczne i spacje.

Output

Wyjściowy łańcuch powinien być wejściowym ciągiem odwróconym słowo po słowie.

Example

Input

"Hello World"

Output

"World Hello"
