Perfect Number Checker II

A perfect number is a positive integer that is equal to the sum of its proper divisors (excluding itself). For instance, 6 is a perfect number because its proper divisors are 1, 2, and 3, and 1 + 2 + 3 = 6. In this exercise, you are asked to create a function that receives a positive integer and returns True if the given integer is a perfect number, or False if it is not.

Task

Create a function that receives a positive integer and returns True if the given integer is a perfect number; False otherwise.

Input

The function receives a positive integer (n) as input.

Output

The function returns True if the input integer is a perfect number, or False if it is not a perfect number.

Example

Input

"28"

Output

"True"
