Sprawdzacz Liczby Doskonałej II

Doskonała liczba to dodatnia liczba całkowita, która jest równa sumie swoich właściwych dzielników (z pominięciem siebie samej). Na przykład 6 to doskonała liczba, ponieważ jej właściwymi dzielnikami są 1, 2 i 3, a 1 + 2 + 3 = 6. W tym ćwiczeniu masz stworzyć funkcję, która otrzymuje dodatnią liczbę całkowitą i zwraca True, jeśli dana liczba jest liczbą doskonałą, lub False, jeśli nią nie jest.

Task

Stwórz funkcję, która otrzymuje dodatnią liczbę całkowitą i zwraca True, jeśli dana liczba jest liczbą doskonałą; False w przeciwnym razie.

Input

Funkcja otrzymuje jako wejście dodatnią liczbę całkowitą (n).

Output

Funkcja zwraca True, jeśli liczba całkowita wejściowa jest liczbą doskonałą; False, jeśli nie jest liczbą doskonałą.

Example

Input

"28"

Output

"True"
