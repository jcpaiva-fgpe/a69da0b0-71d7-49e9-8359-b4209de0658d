Verificatore di numeri perfetti II

Un numero perfetto è un numero intero positivo che è uguale alla somma dei suoi divisori propri (escluso se stesso). Ad esempio, 6 è un numero perfetto perché i suoi divisori appropriati sono 1, 2 e 3 e 1 + 2 + 3 = 6. In questo esercizio, ti viene chiesto di creare una funzione che riceve un numero intero positivo e restituisce True se il numero dato è un numero perfetto, o False se non lo è.

Task

Crea una funzione che riceve un numero intero positivo e restituisce True se il numero dato è un numero perfetto; False altrimenti.

Input

La funzione riceve come input un numero intero positivo (n).

Output

La funzione restituisce True se il numero intero in input è un numero perfetto, o False se non è un numero perfetto.

Example

Input

"28"

Output

"True"
