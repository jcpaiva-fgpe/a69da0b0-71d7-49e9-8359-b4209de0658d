Verificador de Números Perfeitos II

Um número perfeito é um número inteiro positivo que é igual a soma de seus divisores próprios (excluindo a si próprio). Por exemplo, 6 é um número perfeito porque seus divisores próprios são 1, 2 e 3, e 1 + 2 + 3 = 6. Neste exercício, você é solicitado a criar uma função que receba um inteiro positivo e retorne True se o número dado for um número perfeito, ou False se não for.

Task

Crie uma função que receba um inteiro positivo e retorne True se o número dado for um número perfeito; False caso contrário.

Input

A função recebe como entrada um inteiro positivo (n).

Output

A função retorna True se o inteiro de entrada for um número perfeito, ou False se não for um número perfeito.

Example

Input

"28"

Output

"True"
