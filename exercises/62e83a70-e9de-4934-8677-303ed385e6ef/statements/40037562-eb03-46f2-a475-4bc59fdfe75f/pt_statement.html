Interseção de Retângulos

Você está criando um programa de design auxiliado por computador (CAD) e precisa verificar se dois retângulos se intersectam.

Task

Escreva uma função que recebe dois retângulos como entrada e retorna verdadeiro se eles se intersectam, falso caso contrário.

Input

A entrada serão dois retângulos, cada um representado como uma lista de quatro inteiros correspondentes às coordenadas x e y de seus cantos superior esquerdo e inferior direito.

Output

A função deve retornar um valor booleano indicando se os dois retângulos se intersectam ou não.

Example

Input

"[0, 0, 2, 2];[1, 1, 3, 3]"

Output

"true"
