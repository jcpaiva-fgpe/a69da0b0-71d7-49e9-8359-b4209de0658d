Przecięcie Prostokątów

Tworzysz program do projektowania wspomaganego komputerowo (CAD) i musisz sprawdzić, czy dwa prostokąty przecinają się.

Task

Napisz funkcję, która jako wejście przyjmuje dwa prostokąty i zwraca true, jeśli się przecinają, false w przeciwnym przypadku.

Input

Wejście to dwa prostokąty, każdy reprezentowany jako lista czterech liczb całkowitych odpowiadających współrzędnym x i y ich lewego górnego i prawego dolnego rogu.

Output

Funkcja powinna zwrócić wartość logiczną wskazującą, czy dwa prostokąty przecinają się czy nie.

Example

Input

"[0, 0, 2, 2];[1, 1, 3, 3]"

Output

"true"
