Rectangle Intersection

You are creating a computer-aided design (CAD) program and need to check if two rectangles intersect.

Task

Write a function that takes two rectangles as input and returns true if they intersect, false otherwise.

Input

The input will be two rectangles, each represented as a list of four integers corresponding to the x and y coordinates of their top-left and bottom-right corners.

Output

The function should return a boolean value indicating whether the two rectangles intersect or not.

Example

Input

"[0, 0, 2, 2];[1, 1, 3, 3]"

Output

"true"
