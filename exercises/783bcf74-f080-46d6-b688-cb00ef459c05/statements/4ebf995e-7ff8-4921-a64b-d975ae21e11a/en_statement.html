Sum of Odd Integers

You have to find the sum of first n odd numbers. For example, if n=3, then the sum would be 1+3+5=9

Task

Write a function that takes in an integer n and returns the sum of the first n odd integers

Input

Input will be a positive integer n (1<=n<=1000)

Output

Output will be an integer representing the sum of the first n odd integers

Example

Input

"3"

Output

"9"
