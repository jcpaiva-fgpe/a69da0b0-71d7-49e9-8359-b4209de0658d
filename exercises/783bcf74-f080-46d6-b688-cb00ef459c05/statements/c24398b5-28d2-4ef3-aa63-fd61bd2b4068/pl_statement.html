Suma liczb nieparzystych

Musisz znaleźć sumę pierwszych n liczb nieparzystych. Na przykład, jeśli n=3, to suma wynosiłaby 1+3+5=9

Task

Napisz funkcję, która przyjmuje liczbę całkowitą n i zwraca sumę pierwszych n liczb nieparzystych

Input

Wejście będzie dodatnią liczbą całkowitą n (1<=n<=1000)

Output

Wyjście będzie liczbą całkowitą reprezentującą sumę pierwszych n liczb nieparzystych

Example

Input

"3"

Output

"9"
