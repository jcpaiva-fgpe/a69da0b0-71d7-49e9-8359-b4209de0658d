Somma di interi dispari

Devi trovare la somma dei primi n numeri dispari. Ad esempio, se n=3, la somma sarebbe 1+3+5=9

Task

Scrivi una funzione che prenda in input un intero n e restituisca la somma dei primi n numeri dispari

Input

L'input sarà un intero positivo n (1<=n<=1000)

Output

L'output sarà un intero che rappresenta la somma dei primi n numeri dispari

Example

Input

"3"

Output

"9"
