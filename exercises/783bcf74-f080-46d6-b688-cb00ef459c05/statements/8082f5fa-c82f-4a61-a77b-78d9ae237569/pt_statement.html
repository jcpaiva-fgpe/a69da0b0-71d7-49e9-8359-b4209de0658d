Soma de Inteiros Ímpares

Você tem que encontrar a soma dos primeiros n números ímpares. Por exemplo, se n=3, então a soma seria 1+3+5=9

Task

Escreva uma função que receba um inteiro n e retorne a soma dos primeiros n numeros ímpares

Input

A entrada será um número inteiro positivo n (1<=n<=1000)

Output

A saída será um número inteiro representando a soma dos primeiros n numeros ímpares

Example

Input

"3"

Output

"9"
