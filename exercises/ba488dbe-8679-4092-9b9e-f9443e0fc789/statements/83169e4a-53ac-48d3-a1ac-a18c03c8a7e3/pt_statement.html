Calculadora de Lucro de Venda

Você está trabalhando para uma startup chamada ABC tecnologias e foi solicitado que você desenvolva uma calculadora de lucro de vendas.

Task

Escreva uma função que receba o preço de custo e o preço de venda de um produto e retorne o lucro obtido com a venda desse produto.

Input

A entrada serão dois números inteiros 'cost_price' (0 <= cost_price <= 10000) e 'sell_price' (0 <= sell_price <= 10000) separados por ponto e vírgula.

Output

A saída será um número real arredondado para duas casas decimais.

Example

Input

"100;1500"

Output

"1400.00"
