Sales Profit Calculator

You are working for a startup named ABC technologies and you are asked to develop a sales profit calculator.

Task

Write a function that takes in the cost price and selling price of a product and returns the profit earned from selling that product.

Input

The input will be two integers 'cost_price' (0 <= cost_price <= 10000) and 'sell_price' (0 <= sell_price <= 10000) separated by semicolon.

Output

The output will be a float rounded to two decimal places.

Example

Input

"100;1500"

Output

"1400.00"
