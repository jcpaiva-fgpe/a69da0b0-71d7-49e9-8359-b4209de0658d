Kalkulator Zysku ze Sprzedaży

Pracujesz dla startupu o nazwie ABC technologies i zostałeś poproszony o opracowanie kalkulatora zysku ze sprzedaży.

Task

Napisz funkcję, która przyjmuje cenę zakupu i sprzedaży produktu i zwraca zysk uzyskany ze sprzedaży tego produktu.

Input

Wejściem będą dwie liczby całkowite 'cost_price' (0 <= cost_price <= 10000) i 'sell_price' (0 <= sell_price <= 10000) oddzielone średnikiem.

Output

Wyjściem będzie liczba zmiennoprzecinkowa zaokrąglona do dwóch miejsc po przecinku.

Example

Input

"100;1500"

Output

"1400.00"
