Matrix Rotation

You are given a 2D matrix of size MxN and you need to rotate it by 90 degrees clockwise. This is a common operation in image processing and computer graphics.

Task

Write a function that receives a 2D matrix and returns the same matrix rotated by 90 degrees clockwise.

Input

The function receives a 2D matrix as input, represented by a list of lists. Each inner list is a row and the values ​​represent the columns.

Output

The function should return a 2D matrix as output, represented by a list of lists, that is the input matrix rotated by 90 degrees clockwise.

Example

Input

"[[1,2,3],[4,5,6],[7,8,9]]"

Output

"[[7, 4, 1],[8, 5, 2],[9, 6, 3]]"
