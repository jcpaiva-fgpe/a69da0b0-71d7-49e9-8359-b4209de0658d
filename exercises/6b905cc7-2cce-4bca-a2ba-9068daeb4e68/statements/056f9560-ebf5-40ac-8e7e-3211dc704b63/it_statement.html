Rotazione di matrice

Ti viene data una matrice 2D di dimensione MxN e devi ruotarla di 90 gradi in senso orario. Questa è un'operazione comune nell'elaborazione delle immagini e nei grafici al computer.

Task

Scrivi una funzione che riceve una matrice 2D e restituisce la stessa matrice ruotata di 90 gradi in senso orario.

Input

La funzione riceve in input una matrice 2D, rappresentata da una lista di liste. Ogni lista interna è una riga e i valori rappresentano le colonne.

Output

La funzione dovrebbe restituire una matrice 2D come output, rappresentata da una lista di liste, che è la matrice di input ruotata di 90 gradi in senso orario.

Example

Input

"[[1,2,3],[4,5,6],[7,8,9]]"

Output

"[[7, 4, 1],[8, 5, 2],[9, 6, 3]]"
