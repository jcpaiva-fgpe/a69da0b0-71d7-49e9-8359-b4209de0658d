Rotação de matriz

Você recebe uma matriz 2D de tamanho MxN e precisa rotacioná-la 90 graus no sentido horário. Essa é uma operação comum em processamento de imagens e gráficos por computador.

Task

Escreva uma função que recebe uma matriz 2D e retorna a mesma matriz rotacionada em 90 graus no sentido horário.

Input

A função recebe uma matriz 2D como entrada, representada por uma lista de listas. Cada lista interna é uma linha e os valores ​​representam as colunas.

Output

A função deve retornar uma matriz 2D como saída, representada por uma lista de listas, que é a matriz de entrada rotacionada em 90 graus no sentido horário.

Example

Input

"[[1,2,3],[4,5,6],[7,8,9]]"

Output

"[[7, 4, 1],[8, 5, 2],[9, 6, 3]]"
