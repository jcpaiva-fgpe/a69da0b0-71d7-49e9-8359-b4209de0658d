Konwerter liczb na słowa

Czasami łatwiej jest zapisać liczbę słownie niż w formie cyfr. Program powinien zamienić liczbę całkowitą na jej reprezentację słowną. Na przykład: 245 powinno być zapisane jako 'dwieście czterdzieści pięć'.

Task

Napisz funkcję, która przyjmuje liczbę całkowitą i zwraca ciąg znaków z liczbą słownie.

Input

Funkcja powinna przyjmować liczbę całkowitą jako wejście.

Output

Funkcja powinna zwrócić ciąg znaków z liczbą słownie.

Example

Input

"0"

Output

"'zero'"
