Conversor de Números por Extenso

Às vezes, é mais conveniente escrever números com palavras do que em forma de digitos. Este programa deve converter um número inteiro em sua representação em palavras. Por exemplo: 245 seria 'duzentos e quarenta e cinco'.

Task

Escreva uma função que receba um número inteiro e retorne uma string com o número por extenso.

Input

A função deve receber um número inteiro como entrada.

Output

A função deve retornar uma string com o número por extenso.

Example

Input

"0"

Output

"'zero'"
