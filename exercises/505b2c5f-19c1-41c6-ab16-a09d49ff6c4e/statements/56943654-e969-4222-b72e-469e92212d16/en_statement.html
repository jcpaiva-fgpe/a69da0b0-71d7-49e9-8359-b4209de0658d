Number to Words Converter

Sometimes, it's more convenient to write numbers in words than as digits. This program should convert an integer number to its representation in words. For instance: 245 would be 'two hundred forty-five'.

Task

Write a function that takes an integer number and returns a string with the number in words.

Input

The function should receive an integer number as input.

Output

The function should return a string with the number in words.

Example

Input

"0"

Output

"'zero'"
