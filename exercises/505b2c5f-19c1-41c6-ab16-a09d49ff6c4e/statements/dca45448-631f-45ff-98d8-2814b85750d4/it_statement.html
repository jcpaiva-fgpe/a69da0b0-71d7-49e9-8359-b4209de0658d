Convertitore di Numeri in Parole

A volte è più comodo scrivere i numeri a parole invece che in forma di cifre. Questo programma dovrebbe convertire un numero intero nella sua rappresentazione in parole. Ad esempio: 245 sarebbe 'duecento quarantacinque'

Task

Scrivi una funzione che prenda un numero intero e restituisca una stringa con il numero in parole.

Input

La funzione dovrebbe ricevere come input un numero intero.

Output

La funzione dovrebbe restituire una stringa con il numero in parole.

Example

Input

"0"

Output

"'zero'"
