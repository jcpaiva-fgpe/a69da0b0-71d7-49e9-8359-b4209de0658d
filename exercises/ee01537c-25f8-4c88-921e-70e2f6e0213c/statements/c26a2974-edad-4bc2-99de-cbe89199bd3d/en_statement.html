Sorting Array by Parity

Given an array of non-negative integers, you need to sort them based on whether they are even or odd. The sorted array should have all the even numbers appear before the odd numbers.

Task

Create a function that receives an array of non-negative integers and returns the sorted array with even numbers appearing before odd numbers.

Input

The function should receive an array of non-negative integers. The length of the array should not exceed 5000.

Output

The function should return a sorted array with even numbers appearing before odd numbers.

Example

Input

"[3,1,2,4]"

Output

"[2,4,3,1]"
