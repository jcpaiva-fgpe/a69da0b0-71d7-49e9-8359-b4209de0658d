Ordinare un Array per Parità

Dato un array di interi non negativi, devi ordinarli in base al fatto che sono pari o dispari. L'array ordinato dovrebbe avere tutti i numeri pari che appaiono prima dei numeri dispari.

Task

Crea una funzione che riceva un array di interi non negativi e restituisca l'array ordinato con i numeri pari che appaiono prima dei numeri dispari.

Input

La funzione dovrebbe ricevere un array di interi non negativi. La lunghezza dell'array non deve superare i 5000.

Output

La funzione dovrebbe restituire un array ordinato con i numeri pari che appaiono prima dei numeri dispari.

Example

Input

"[3,1,2,4]"

Output

"[2,4,3,1]"
