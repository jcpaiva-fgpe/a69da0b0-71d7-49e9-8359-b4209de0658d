Sortowanie Tablicy pod względem Parzystości

Mając tablice nieujemnych liczb całkowitych, musisz je posortować zgodnie z tym, czy są parzyste czy nieparzyste. Posortowana tablica powinna mieć wszystkie liczby parzyste przed nieparzystymi.

Task

Utwórz funkcję, która otrzymuje tablicę nieujemnych liczb całkowitych i zwraca posortowaną tablicę z liczbami parzystymi przed nieparzystymi.

Input

Funkcja powinna przyjmować tablicę nieujemnych liczb całkowitych. Długość tablicy nie powinna przekraczać 5000.

Output

Funkcja powinna zwracać posortowaną tablicę z liczbami parzystymi przed nieparzystymi.

Example

Input

"[3,1,2,4]"

Output

"[2,4,3,1]"
