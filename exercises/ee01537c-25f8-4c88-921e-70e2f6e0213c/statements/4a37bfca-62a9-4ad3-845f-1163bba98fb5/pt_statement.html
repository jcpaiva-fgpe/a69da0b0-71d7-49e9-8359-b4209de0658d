Ordenando Array por Paridade

Dado um array de inteiros não negativos, você precisa ordená-los baseado se eles são pares ou impares. O array ordenado deve ter todos os números pares aparecendo antes dos números ímpares.

Task

Crie uma função que receba um array de inteiros não negativos e retorne o array ordenado com os números pares antes dos números ímpares.

Input

A função deve receber um array de inteiros não negativos. O tamanho do array não deve exceder 5000.

Output

A função deve retornar um array ordenado com os números pares aparecendo antes dos números ímpares.

Example

Input

"[3,1,2,4]"

Output

"[2,4,3,1]"
