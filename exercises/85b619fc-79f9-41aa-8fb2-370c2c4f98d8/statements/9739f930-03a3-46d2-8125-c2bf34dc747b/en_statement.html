Reverse Bits

You are given an integer 'n'. You have to return the binary representation of 'n' with reversed bits. For example, if 'n' is 13 (1101 in binary), the output should be 11 (1011 in binary).

Task

Write a function that takes an integer 'n' and returns its binary representation with reversed bits.

Input

An integer 'n'.

Output

Binary representation of 'n' with reversed bits.

Example

Input

"13"

Output

"11"
