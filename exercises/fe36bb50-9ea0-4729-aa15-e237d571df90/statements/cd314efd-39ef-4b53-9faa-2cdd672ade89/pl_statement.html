Suma liczb pierwszych

Masz daną liczbę n. Musisz znaleźć sumę wszystkich liczb pierwszych mniejszych lub równych n.

Task

Napisz funkcję, która przyjmuje liczbę całkowitą n jako dane wejściowe i zwraca sumę wszystkich liczb pierwszych mniejszych lub równych n.

Input

Dane wejściowe to pojedyncza liczba całkowita n (2 ≤ n ≤ 10^6).

Output

Wyjściem będzie pojedyncza liczba całkowita - suma wszystkich liczb pierwszych mniejszych lub równych n.

Example

Input

"10"

Output

"17"
