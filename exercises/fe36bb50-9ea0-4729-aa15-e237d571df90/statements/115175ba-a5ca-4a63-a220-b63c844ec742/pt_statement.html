Soma dos Números Primos

Dado um número n, você tem que encontrar a soma de todos os números primos menores ou iguais a n.

Task

Escreva uma função que receba um inteiro n como entrada e retorne a soma de todos os números primos menores ou iguais a n.

Input

A entrada será um único inteiro n (2 ≤ n ≤ 10^6).

Output

A saída será um único inteiro - a soma de todos os números primos menores ou iguais a n.

Example

Input

"10"

Output

"17"
