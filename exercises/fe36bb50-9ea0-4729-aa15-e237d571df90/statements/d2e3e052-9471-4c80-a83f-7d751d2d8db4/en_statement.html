Sum of Primes

You are given a number n. You have to find the sum of all the prime numbers less than or equal to n.

Task

Write a function that takes an integer n as input and returns the sum of all prime numbers less than or equal to n.

Input

The input will be a single integer n (2 ≤ n ≤ 10^6).

Output

The output will be a single integer - the sum of all prime numbers less than or equal to n.

Example

Input

"10"

Output

"17"
