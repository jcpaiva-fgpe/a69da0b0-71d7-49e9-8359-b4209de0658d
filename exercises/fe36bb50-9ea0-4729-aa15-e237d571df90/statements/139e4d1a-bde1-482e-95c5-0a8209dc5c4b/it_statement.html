Somma dei numeri primi

Ti è dato un numero n. Devi trovare la somma di tutti i numeri primi minori o uguali a n.

Task

Scrivi una funzione che prenda un intero n in input e restituisca la somma di tutti i numeri primi minori o uguali a n.

Input

L'input sarà un singolo intero n (2 ≤ n ≤ 10^6).

Output

L'output sarà un singolo intero - la somma di tutti i numeri primi minori o uguali a n.

Example

Input

"10"

Output

"17"
