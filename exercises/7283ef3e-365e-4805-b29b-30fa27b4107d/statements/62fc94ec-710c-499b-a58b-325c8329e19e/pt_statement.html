Validador de senhas

Você é um desenvolvedor de software e sua tarefa é criar um programa que valide se uma senha é segura ou não. Uma senha segura deve conter no mínimo 8 caracteres, pelo menos uma letra maiúscula, uma letra minúscula, um dígito e um caractere especial.

Task

Crie uma função que receba uma string como entrada e retorne True se for uma senha segura ou False se não for.

Input

A entrada é uma string representando uma senha.

Output

A saída é um valor booleano indicando se a senha é segura ou não.

Example

Input

"Abc123!@"

Output

true
