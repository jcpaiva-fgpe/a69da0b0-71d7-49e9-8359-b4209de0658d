Walidator hasła

Jesteś programistą i Twoim zadaniem jest stworzenie programu, który sprawdza, czy hasło jest bezpieczne. Bezpieczne hasło musi zawierać co najmniej 8 znaków, przynajmniej jedną wielką literę, jedną małą literę, jedną cyfrę i jeden znak specjalny.

Task

Stwórz funkcję, która jako wejście przyjmuje ciąg znaków i zwraca True, jeśli jest to bezpieczne hasło lub False, jeśli nie.

Input

Wejściem jest ciąg znaków reprezentujący hasło.

Output

Wyjściem jest wartość logiczna wskazująca, czy hasło jest bezpieczne czy nie.

Example

Input

"Abc123!@"

Output

true
