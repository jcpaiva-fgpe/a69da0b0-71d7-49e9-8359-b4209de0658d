Validatore di password

Sei uno sviluppatore di software e il tuo compito è creare un programma che valida se una password è sicura o meno. Una password sicura deve contenere almeno 8 caratteri, almeno una lettera maiuscola, una lettera minuscola, un numero e un carattere speciale.

Task

Crea una funzione che riceva una stringa in input e restituisca True se è una password sicura o False se non lo è.

Input

L'input è una stringa che rappresenta una password.

Output

L'output è un valore booleano che indica se la password è sicura o meno.

Example

Input

"Abc123!@"

Output

true
