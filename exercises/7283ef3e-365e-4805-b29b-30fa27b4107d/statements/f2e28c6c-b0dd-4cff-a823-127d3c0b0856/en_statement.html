Password Validator

You are a software developer and your task is to create a program that validates if a password is secure or not. A secure password must contain a minimum of 8 characters, at least one uppercase letter, one lowercase letter, one digit and one special character.

Task

Create a function that receives a string as an input and returns True if it's a secure password or False if it's not.

Input

The input is a string representing a password.

Output

The output is a boolean value indicating if the password is secure or not.

Example

Input

"Abc123!@"

Output

true
