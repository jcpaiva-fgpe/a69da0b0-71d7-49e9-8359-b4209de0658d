Validador de Email

Você está trabalhando em uma funcionalidade de registro de usuário para um site e precisa validar o e-mail do usuário.

Task

Escreva uma função que receba um endereço de e-mail como entrada e retorne 'Válido' se o endereço de e-mail for válido, caso contrário, retorne 'Inválido'. Um endereço de e-mail é considerado válido se atender aos seguintes critérios: A) Ele contém um único caractere @. B) Tem pelo menos um caractere antes do caractere @. C) O caractere @ é seguido por pelo menos um caractere '.' D) Existe pelo menos um caractere após o caractere '.'.

Input

Uma string que representa um endereço de e-mail.

Output

Uma string 'Válido' se o e-mail for válido ou 'Inválido' se o e-mail for inválido.

Example

Input

"test@example.com"

Output

"Valid"
