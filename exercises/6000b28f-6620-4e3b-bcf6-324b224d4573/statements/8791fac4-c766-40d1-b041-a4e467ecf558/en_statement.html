Email Validator

You are working on a user registration feature for a website and need to validate user's email.

Task

Write a function that takes an email address as an input and returns 'Valid' if the email address is valid, otherwise returns 'Invalid'. An email address is considered valid if it meets the following criteria: A) It contains a single @ character. B) It has at least one character before the @ character. C) The @ character is followed by at least one '.' character. D) There is at least one character after the '.' character.

Input

A string representing an email address.

Output

A string 'Valid' if the email is valid or 'Invalid' if the email is invalid.

Example

Input

"test@example.com"

Output

"Valid"
