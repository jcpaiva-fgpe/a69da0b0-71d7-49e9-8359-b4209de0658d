Verificador de pangramas II

Um pangrama é uma frase que contém todas as letras do alfabeto pelo menos uma vez. Neste exercício, você precisa criar uma função que recebe uma sentença e verifica se é um pangrama.

Task

Crie uma função que recebe uma sentença e verifica se ela é um pangrama. A função deve retornar True se for um pangrama, e False se não for.

Input

A função recebe uma string com a sentença a ser verificada.

Output

A função deve retornar True se a sentença de entrada for um pangrama e False se não for.

Example

Input

"The quick brown fox jumps over the lazy dog"

Output

"True"
