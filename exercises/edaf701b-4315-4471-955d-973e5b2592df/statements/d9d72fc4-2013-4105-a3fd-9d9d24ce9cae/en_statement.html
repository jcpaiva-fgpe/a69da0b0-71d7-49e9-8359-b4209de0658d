Number Sequence

You have to create a program that generates a sequence of numbers starting from 1 to n. That is, given an integer n, generate 1, 2, ..., n. This type of problem can be commonly found in mathematics and computer science.

Task

Create a function that receives an integer n and returns a list with a sequence of numbers from 1 to n.

Input

The function receives an integer n (1 <= n <= 100).

Output

The function has to return a list with a sequence of numbers from 1 to n.

Example

Input

"5"

Output

"[1, 2, 3, 4, 5]"
