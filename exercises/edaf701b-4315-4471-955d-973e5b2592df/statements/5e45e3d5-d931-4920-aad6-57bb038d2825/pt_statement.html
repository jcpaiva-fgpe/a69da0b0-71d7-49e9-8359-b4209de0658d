Sequência de Números

Você deve criar um programa que gera uma sequência de números começando do 1 até n. Ou seja, dado um número inteiro n, gere 1, 2, ..., n. Esse tipo de problema é comumente encontrado na matemática e ciência da computação.

Task

Crie uma função que recebe um inteiro n e retorna uma lista com uma sequência de números de 1 até n.

Input

A função recebe um inteiro n (1 <= n <= 100).

Output

A função tem que retornar uma lista com uma sequência de números de 1 até n.

Example

Input

"5"

Output

"[1, 2, 3, 4, 5]"
