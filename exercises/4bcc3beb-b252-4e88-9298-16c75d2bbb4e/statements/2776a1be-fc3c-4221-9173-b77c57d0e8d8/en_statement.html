Highest Scoring Word

Given a string of words, you need to find the highest scoring word. Each letter of a word scores points according to its position in the alphabet - a = 1, b = 2, c = 3, .... z = 26. You need to return the highest scoring word as a string.

Task

Write a function that takes a string of words and returns the highest scoring word. If two words score the same, return the word that appears first in the string.

Input

A string of words separated by spaces.

Output

A string with the highest scoring word.

Example

Input

"man i need a taxi up to ubud"

Output

"taxi"
