Parola con Punteggio Più Alto

Data una stringa di parole, devi trovare la parola con il punteggio più alto. Ogni lettera di una parola ottiene un punteggio in base alla sua posizione nell'alfabeto - a = 1, b = 2, c = 3, ... z = 26. Devi restituire la parola con il punteggio più alto come una stringa.

Task

Scrivi una funzione che prenda una stringa di parole e restituisca la parola con il punteggio più alto. Se due parole ottengono lo stesso punteggio, restituisci la parola che compare per prima nella stringa.

Input

Una stringa di parole separate da spazi.

Output

Una stringa con la parola che ha ottenuto il punteggio più alto.

Example

Input

"man i need a taxi up to ubud"

Output

"taxi"
