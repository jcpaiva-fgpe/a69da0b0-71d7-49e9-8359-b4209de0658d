Palavra de Pontuação Mais Alta

Dada uma string de palavras, você precisa encontrar a palavra com a maior pontuação. Cada letra de uma palavra pontua pontos de acordo com sua posição no alfabeto - a = 1, b = 2, c = 3, ... z = 26. Você precisa retornar a palavra de pontuação mais alta como uma string.

Task

Escreva uma função que receba uma string de palavras e retorne a palavra de pontuação mais alta. Se duas palavras pontuarem o mesmo valor, retorne a palavra que aparece primeiro na string.

Input

Uma string de palavras separadas por espaços.

Output

Uma string com a palavra com a pontuação mais alta.

Example

Input

"man i need a taxi up to ubud"

Output

"taxi"
