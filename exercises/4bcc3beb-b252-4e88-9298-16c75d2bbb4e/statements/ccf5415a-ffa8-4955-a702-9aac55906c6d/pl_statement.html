Najwyżej punktowane słowo

Mając ciąg słów, musisz znaleźć słowo z najwyższą punktacją. Każdej literze zostaje przypisana wartość punktowa, w zależności od jej pozycji w alfabecie - a = 1, b = 2, c = 3, ... z = 26. Musisz zwrócić słowo z najwyższą punktacją jako string.

Task

Napisz funkcję, która przyjmuje ciąg słów i zwraca słowo z najwyższą punktacją. Jeśli dwie lub więcej słów zdobędzie tyle samo punktów, należy zwrócić pierwsze z kolejnego ciągu słów.

Input

Ciąg słów oddzielonych spacjami.

Output

String z słowem o najwyższej punktacji.

Example

Input

"man i need a taxi up to ubud"

Output

"taxi"
