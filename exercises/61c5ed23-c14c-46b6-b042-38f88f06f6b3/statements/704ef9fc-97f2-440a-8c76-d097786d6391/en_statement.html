Pascal's Triangle

Pascal's triangle is a mathematical sequence that is named after Blaise Pascal, although he was not the first to study this triangle. It is a triangular array of binomial coefficients, which are the coefficients of the terms in the expansion of (a + b)^n, where n is a positive integer. In this triangle, the first and last numbers in each row are always 1, and each of the other numbers in the triangle is the sum of the two numbers directly above it.

Task

Write a function that takes in an integer n and returns the first n rows of Pascal's triangle. Each row should be returned as a list of integers.

Input

The input will be a single positive integer n (1 <= n <= 30), representing the number of rows to return.

Output

The function should return a list of lists of integers. Each inner list represents a row of Pascal's triangle, starting from the first row (which is always [1]).

Example

Input

"5"

Output

"[[1],[1,1],[1,2,1],[1,3,3,1],[1,4,6,4,1]]"
