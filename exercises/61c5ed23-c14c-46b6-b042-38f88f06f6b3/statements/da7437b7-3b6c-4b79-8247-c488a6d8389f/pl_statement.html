Trójkąt Pascala

Trójkąt Pascala to ciąg matematyczny nazwany imieniem Blaise'a Pascala, choć nie był on pierwszym, który badał ten trójkąt. Trójkąt ten jest trójkątem liczb dwumianowych, których współczynniki odpowiadają kolejnym wyrazom rozwinięcia wzoru (a+b)^n, gdzie n jest dowolną liczbą naturalną. W trójkącie tym pierwszą i ostatnią liczbą każdego wiersza jest zawsze 1, a każda kolejna liczba to suma dwóch liczb znajdujących się bezpośrednio nad nią.

Task

Napisz funkcję, która przyjmuje jeden parametr - liczbę całkowitą n - i zwraca n pierwszych wierszy trójkąta Pascala. Każdy zwrócony wiersz powinien być zwrócony jako lista liczb całkowitych.

Input

Wejście będzie pojedynczą dodatnią liczbą całkowitą n (1 <= n <= 30), reprezentującą liczba wierszy do zwrócenia.

Output

Funkcja powinna zwrócić listę list liczb całkowitych. Każda wewnętrzna lista reprezentuje jeden wiersz trójkąta Pascala, rozpoczynając od pierwszego wiersza (który zawsze jest [1]).

Example

Input

"5"

Output

"[[1],[1,1],[1,2,1],[1,3,3,1],[1,4,6,4,1]]"
