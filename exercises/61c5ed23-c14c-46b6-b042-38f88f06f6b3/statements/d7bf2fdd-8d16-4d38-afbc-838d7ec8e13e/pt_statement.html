Triângulo de Pascal

O triângulo de Pascal é uma sequência matemática que recebe o nome de Blaise Pascal, embora ele não tenha sido o primeiro a estudar esse triângulo. É uma matriz triangular de coeficientes binomiais, que são os coeficientes dos termos na expansão de (a + b)^n, onde n é um número inteiro positivo. Neste triângulo, os primeiros e últimos números de cada linha são sempre 1, e cada um dos outros números no triângulo é a soma dos dois números diretamente acima dele.

Task

Escreva uma função que receba um inteiro n e retorne as primeiras n linhas do triângulo de Pascal. Cada linha deve ser retornada como uma lista de inteiros.

Input

A entrada será um único inteiro positivo n (1 <= n <= 30), representando o número de linhas a serem retornadas.

Output

A função deve retornar uma lista de listas de inteiros. Cada lista interna representa uma linha do triângulo de Pascal, começando pela primeira linha (que sempre é [1]).

Example

Input

"5"

Output

"[[1],[1,1],[1,2,1],[1,3,3,1],[1,4,6,4,1]]"
