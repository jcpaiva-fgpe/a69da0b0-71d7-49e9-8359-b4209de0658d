Triangolo di Pascal

Il triangolo di Pascal è una sequenza matematica che prende il nome da Blaise Pascal, anche se non è stato il primo a studiare questo triangolo. Si tratta di un array triangolare di coefficienti binomiali, che sono i coefficienti dei termini nell'espansione di (a + b)^n, dove n è un numero intero positivo. In questo triangolo, il primo e l'ultimo numero di ogni riga sono sempre 1, e ciascuno degli altri numeri nel triangolo è la somma dei due numeri direttamente sopra di esso.

Task

Scrivi una funzione che prende in input un intero n e restituisce le prime n righe del triangolo di Pascal. Ogni riga deve essere restituita come una lista di interi.

Input

L'input sarà un singolo numero intero positivo n (1 <= n <= 30), che rappresenta il numero di righe da restituire.

Output

La funzione dovrebbe restituire una lista di liste di interi. Ogni lista interna rappresenta una riga del triangolo di Pascal, a partire dalla prima riga (che è sempre [1]).

Example

Input

"5"

Output

"[[1],[1,1],[1,2,1],[1,3,3,1],[1,4,6,4,1]]"
