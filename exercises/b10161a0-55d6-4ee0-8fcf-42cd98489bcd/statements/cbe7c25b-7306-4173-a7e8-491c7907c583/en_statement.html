Perfect Number Checker

A perfect number is a positive integer that is equal to the sum of its proper positive divisors, excluding the number itself. For example, 6 is a perfect number because its proper divisors are 1, 2 and 3, and 1 + 2 + 3 = 6. Write a function that checks if a given number is a perfect number or not.

Task

Create a function that takes an integer and returns 'True' if it is a perfect number, 'False' otherwise.

Input

A single positive integer.

Output

'True' if the input is a perfect number, 'False' otherwise

Example

Input

"28"

Output

"True"
