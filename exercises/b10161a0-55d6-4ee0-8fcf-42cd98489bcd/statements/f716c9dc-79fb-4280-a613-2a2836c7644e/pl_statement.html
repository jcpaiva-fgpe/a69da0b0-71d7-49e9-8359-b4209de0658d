Sprawdzacz Liczby Doskonałej

Liczba doskonała to dodatnia liczba całkowita, którą można przedstawić jako sumę swoich dzielników właściwych, czyli takich, które są mniejsze od niej samej. Na przykład, 6 jest liczbą doskonałą, ponieważ jej dzielniki właściwe to 1, 2 i 3, a 1 + 2 + 3 = 6. Napisz funkcję, która sprawdzi, czy podana liczba jest liczbą doskonałą czy nie.

Task

Stwórz funkcję, która przyjmuje liczbę całkowitą i zwraca wartość 'True', jeśli jest to liczba doskonała, 'False' w przeciwnym razie.

Input

Jedna dodatnia liczba całkowita.

Output

'True', jeśli wejście jest liczbą doskonałą, 'False' w przeciwnym razie

Example

Input

"28"

Output

"True"
