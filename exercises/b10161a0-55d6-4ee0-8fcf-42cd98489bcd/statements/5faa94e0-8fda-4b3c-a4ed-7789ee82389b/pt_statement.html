Verificador de Número Perfeito

Um número perfeito é um inteiro positivo que é igual à soma de seus divisores positivos adequados, excluindo o próprio número. Por exemplo, 6 é um número perfeito porque seus divisores adequados são 1, 2 e 3, e 1 + 2 + 3 = 6. Escreva uma função que verifique se um número dado é um número perfeito ou não.

Task

Crie uma função que receba um inteiro e retorne 'True' caso ele seja um número perfeito, 'False' caso contrário.

Input

Um único inteiro positivo.

Output

'True' se a entrada é um número perfeito, 'False' caso contrário

Example

Input

"28"

Output

"True"
