Verificatore di Numero Perfetto

Un numero perfetto è un numero intero positivo che è uguale alla somma dei suoi divisori positivi adeguati, escluso il numero stesso. Ad esempio, 6 è un numero perfetto perché i suoi divisori adeguati sono 1, 2 e 3, e 1 + 2 + 3 = 6. Scrivi una funzione che controlla se un dato numero è un numero perfetto o meno.

Task

Crea una funzione che prende un intero e restituisce 'True' se è un numero perfetto, 'False' altrimenti.

Input

Un singolo numero intero positivo.

Output

'True' se l'input è un numero perfetto, 'False' altrimenti

Example

Input

"28"

Output

"True"
