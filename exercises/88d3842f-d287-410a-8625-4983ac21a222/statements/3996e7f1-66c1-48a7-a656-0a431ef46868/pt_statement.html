Média de dois números

Você foi encarregado de escrever um programa que receba dois números e retorne a média deles.

Task

Escreva uma função que receba dois números e retorne a média deles.

Input

Dois inteiros separados por um ponto e vírgula (;).

Output

Um número decimal representando a média dos dois números de entrada.

Example

Input

"4;6"

Output

"5.0"
