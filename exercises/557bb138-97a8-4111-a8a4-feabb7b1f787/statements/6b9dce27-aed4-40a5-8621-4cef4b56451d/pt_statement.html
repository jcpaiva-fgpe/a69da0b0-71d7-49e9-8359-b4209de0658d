Conversor Decimal-Binário

O sistema binário é um sistema numérico utilizado na computação para representar números utilizando apenas dois dígitos: 0 e 1. Neste exercício, você deve escrever uma função que converte números decimais (base 10) para a representação binária (base 2).

Task

Escreva uma função que recebe um número inteiro e retorna sua representação binária em forma de string.

Input

A entrada será um único número inteiro n (0 <= n <= 10^6).

Output

A saída deve ser uma string que representa a representação binária do número de entrada.

Example

Input

"13"

Output

"1101"
