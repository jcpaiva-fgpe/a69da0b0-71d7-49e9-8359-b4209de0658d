Konwerter z Liczb na Binarny

System binarny to system numeryczny używany w informatyce do reprezentacji liczb za pomocą tylko dwóch cyfr: 0 i 1. W tym zadaniu napiszesz funkcję, która zamienia liczby dziesiętne (podstawa 10) na reprezentacje binarnych (podstawa 2).

Task

Napisz funkcję, która przyjmuje liczbę całkowitą i zwraca jej reprezentację binarną jako łańcuch znaków.

Input

Wejściem będzie pojedyncza liczba całkowita n (0 <= n <= 10^6).

Output

Wyjściem powinien być łańcuch znaków reprezentujący reprezentację binarną danej liczby całkowitej.

Example

Input

"13"

Output

"1101"
