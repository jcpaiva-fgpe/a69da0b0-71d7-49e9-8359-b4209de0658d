Number to Binary Converter

Binary is a numeric system used in computer science to represent numbers using only two digits: 0 and 1. In this exercise, you will write a function that converts decimal (base 10) numbers to binary (base 2) representation.

Task

Write a function that receives an integer number and returns its binary representation as a string.

Input

The input will be a single integer n (0 <= n <= 10^6).

Output

The output should be a string representing the binary representation of the input integer.

Example

Input

"13"

Output

"1101"
