Sprawdzacz liczb consecutive

Pracujesz nad projektem, w którym musisz sprawdzić, czy dany zbiór liczb jest ciągły. Musisz stworzyć program, który może sprawdzić, czy lista liczb całkowitych zawiera sekwencję ciągłą. Twój program zostanie przetestowany na dużej liczbie liczb, więc musi być wydajny.

Task

Utwórz funkcję, która przyjmuje listę liczb całkowitych jako argument i zwraca wartość True, jeśli lista zawiera kolejne liczby, False w przeciwnym razie.

Input

Lista liczb całkowitych

Output

Wartość logiczna True lub False

Example

Input

"[1,2,3,4,5]"

Output

"True"
