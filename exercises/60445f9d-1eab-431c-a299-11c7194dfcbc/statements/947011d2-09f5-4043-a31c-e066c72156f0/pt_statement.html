Verificador de números consecutivos

Você trabalha em um projeto em que precisa verificar se um determinado conjunto de números é consecutivo. Você precisa criar um programa que possa verificar se uma lista de inteiros contém uma sequência consecutiva. Seu programa será testado em um grande conjunto de números, portanto, precisa ser eficiente.

Task

Crie uma função que receba uma lista de inteiros como argumento e retorne True se a lista contém inteiros consecutivos, False caso contrário.

Input

Uma lista de inteiros

Output

O valor booleano True ou False

Example

Input

"[1,2,3,4,5]"

Output

"True"
