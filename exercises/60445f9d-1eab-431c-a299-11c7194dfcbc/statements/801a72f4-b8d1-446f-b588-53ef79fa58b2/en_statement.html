Consecutive Numbers Checker

You are working on a project where you need to verify if a given set of numbers are consecutive. You need to create a program that can verify if a list of integers contains a consecutive sequence. Your program will be tested on a large set of numbers, so it needs to be efficient.

Task

Create a function that takes a list of integers as an argument and returns True if the list contains consecutive integers, False otherwise.

Input

A list of integers

Output

A boolean value of True or False

Example

Input

"[1,2,3,4,5]"

Output

"True"
