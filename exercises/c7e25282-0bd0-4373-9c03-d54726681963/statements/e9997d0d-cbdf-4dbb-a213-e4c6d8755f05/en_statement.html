Sorting Coins

You are given a jar full of coins, but they are not sorted. The coins have different values, and you want to sort them in ascending order so that you can easily count how much money you have in the jar. Write a function that receives a list of coins and returns the list sorted in ascending order.

Task

Create a function that receives a list of coins and returns the same list sorted in ascending order.

Input

The input is a list of integers representing the value of each coin in the jar.

Output

The output is a list of integers representing the value of each coin in ascending order.

Example

Input

"[25,10,5,50,1,100]"

Output

"[1, 5, 10, 25, 50, 100]"
