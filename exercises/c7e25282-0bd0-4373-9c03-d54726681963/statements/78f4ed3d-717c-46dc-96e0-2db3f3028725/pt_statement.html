Classificando Moedas

Você recebeu um pote cheio de moedas, mas elas não estão organizadas. As moedas têm valores diferentes, e você deseja organizá-las em ordem crescente para que possa contar facilmente quanto dinheiro tem no pote. Escreva uma função que receba uma lista de moedas e retorne a lista ordenada em ordem crescente.

Task

Crie uma função que receba uma lista de moedas e retorne a mesma lista ordenada em ordem crescente.

Input

A entrada é uma lista de inteiros representando o valor de cada moeda no pote.

Output

A saída é uma lista de inteiros representando o valor de cada moeda em ordem crescente.

Example

Input

"[25,10,5,50,1,100]"

Output

"[1, 5, 10, 25, 50, 100]"
