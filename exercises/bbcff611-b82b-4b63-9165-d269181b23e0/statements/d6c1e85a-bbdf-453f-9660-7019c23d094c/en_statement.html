Celsius to Fahrenheit Converter

As an intern at a weather forecasting company, you are tasked with creating a program that can quickly convert Celsius temperatures to Fahrenheit for international clients.

Task

Create a function that converts Celsius temperatures to Fahrenheit

Input

A floating point number representing a temperature in Celsius

Output

A floating point number representing a temperature in Fahrenheit

Example

Input

"0"

Output

"32.0"
