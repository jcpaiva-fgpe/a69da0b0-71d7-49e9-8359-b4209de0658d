Conversor de Celsius para Fahrenheit

Como estagiário em uma empresa de previsão do tempo, você tem a tarefa de criar um programa que possa rapidamente converter temperaturas Celsius em Fahrenheit para clientes internacionais.

Task

Crie uma função que converta temperaturas Celsius em Fahrenheit

Input

Um número de ponto flutuante representando uma temperatura em Celsius

Output

Um número de ponto flutuante representando uma temperatura em Fahrenheit

Example

Input

"0"

Output

"32.0"
