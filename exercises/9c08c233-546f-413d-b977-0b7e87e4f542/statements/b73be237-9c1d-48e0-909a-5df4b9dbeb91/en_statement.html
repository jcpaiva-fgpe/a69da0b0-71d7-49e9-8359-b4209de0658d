String Compression

You are implementing a chat application and you realize that message transfer rate is slow. You suspect that this is due to the large size of the messages. You decide to create a function that takes in a message and compresses it to reduce its size.

Task

Write a function that takes in a string and compresses it by replacing consecutive duplicate characters with the character and the count of times it is repeated. For example, the string aaabbbbbcccccddd would become a3b5c5d3.

Input

A string to be compressed.

Output

A compressed string with consecutive duplicates replaced with the character and the count of times it is repeated.

Example

Input

"aaabbbbbcccccddd"

Output

"a3b5c5d3"
