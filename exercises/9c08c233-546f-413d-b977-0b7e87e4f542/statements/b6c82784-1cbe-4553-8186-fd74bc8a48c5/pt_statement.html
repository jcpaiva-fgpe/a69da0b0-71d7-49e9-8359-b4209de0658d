Compressão de String

Você está implementando um aplicativo de bate-papo e percebe que a taxa de transferência de mensagens é lenta. Você suspeita que isso se deve ao grande tamanho das mensagens. Você decide criar uma função que recebe uma mensagem e a comprime para reduzir seu tamanho.

Task

Escreva uma função que receba uma string e a comprima, substituindo caracteres duplicados consecutivos pelo caractere e a contagem de vezes que ele é repetido. Por exemplo, a string aaabbbbbcccccddd se tornaria a3b5c5d3.

Input

Uma string para ser comprimida.

Output

Uma string comprimida com caracteres duplicados consecutivos substituídos pelo caractere e a contagem de vezes que ele é repetido.

Example

Input

"aaabbbbbcccccddd"

Output

"a3b5c5d3"
