Encontre o segundo menor número em uma lista.

Você recebe uma lista de números inteiros e sua tarefa é encontrar e retornar o segundo número mais baixo da lista.

Task

Escreva uma função que receba uma lista de números inteiros como argumento e retorne o segundo inteiro mais baixo da lista.

Input

A entrada é uma lista de números inteiros.

Output

A saída é o segundo número mais baixo da lista.

Example

Input

"[2, 5, 3, 8, 9]"

Output

"3"
