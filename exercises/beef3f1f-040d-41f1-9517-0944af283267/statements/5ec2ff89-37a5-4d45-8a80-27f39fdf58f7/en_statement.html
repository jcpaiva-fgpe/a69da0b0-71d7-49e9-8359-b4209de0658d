Find the Second Smallest Number in a List

You are given a list of integers and your task is to find and return the second smallest number on the list.

Task

Write a function that takes a list of integers as an argument and returns the second smallest integer in the list.

Input

The input is a list of integers.

Output

The output is the second smallest integer in the list.

Example

Input

"[2, 5, 3, 8, 9]"

Output

"3"
