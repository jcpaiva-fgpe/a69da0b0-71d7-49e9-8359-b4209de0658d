Pangram Checker II

A pangram is a sentence that contains every letter of the alphabet at least once. In this exercise, you need to create a function that receives a sentence and checks if it's a pangram.

Task

Create a function that receives a sentence and checks if it's a pangram. The function should return True if it's a pangram, and False if it's not.

Input

The function receives a string with the sentence to be checked.

Output

The function should return True if the input sentence is a pangram and False if it is not.

Example

Input

"The quick brown fox jumps over the lazy dog"

Output

"True"
