Verificatore di pangrammi II

Un pangramma è una frase che contiene almeno una volta ogni lettera dell'alfabeto. In questo esercizio, devi creare una funzione che riceve una frase e controlla se è un pangramma.

Task

Crea una funzione che riceve una frase e controlla se è un pangramma. La funzione dovrebbe restituire True se è un pangramma e False se non lo è.

Input

La funzione riceve una stringa con la frase da controllare.

Output

La funzione dovrebbe restituire True se la frase in input è un pangramma e False se non lo è.

Example

Input

"The quick brown fox jumps over the lazy dog"

Output

"True"
