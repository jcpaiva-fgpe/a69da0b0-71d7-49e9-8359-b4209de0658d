Elemento Più Frequente

Data una lista di interi, trovare l'elemento(i) più frequente(i).

Task

Scrivere una funzione che prende in input una lista di interi e restituisce l'elemento (o gli elementi) più frequente(i). Se ci sono più elementi con la stessa frequenza, restituiscili tutti. Gli elementi restituiti devono essere in ordine crescente. Se la lista è vuota, restituisci una lista vuota.

Input

Una lista di interi.

Output

Una lista di interi che rappresenta l'elemento (o gli elementi) più frequente(i) in ordine crescente.

Example

Input

"[1, 2, 3, 2, 2, 4, 5, 5, 5]"

Output

"[2, 5]"
