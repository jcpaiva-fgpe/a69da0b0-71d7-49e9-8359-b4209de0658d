Removedor de Duplicatas Consecutivas

Você tem uma string com caracteres duplicados consecutivos, precisa removê-los

Task

Crie uma função que receba uma string e remova os caracteres duplicados consecutivos

Input

Uma string com caracteres duplicados consecutivos

Output

Uma string com os caracteres duplicados consecutivos removidos

Example

Input

"aaabbbccc"

Output

"abc"
