Usuwanie Sąsiednich Powtórzeń

Masz ciąg znaków z duplikującymi się znakami, musisz je usunąć

Task

Utwórz funkcję, która pobierze ciąg znaków i usunie sąsiednie powtórzenia znaków

Input

Ciąg znaków z duplikującymi się znakami

Output

Ciąg znaków bez sąsiednich powtórzeń znaków

Example

Input

"aaabbbccc"

Output

"abc"
