Conversor de Números

Você é solicitado a criar uma função que converte um determinado número em sua representação de string correspondente. Você precisa implementar funções separadas para converter um número decimal em representação binária, octal e hexadecimal.

Task

Escreva uma função que recebe dois parâmetros, um número e uma base, e retorne a representação de string desse número na base fornecida.

Input

A entrada deve conter dois inteiros separados por um ponto e vírgula. O primeiro inteiro é o número a ser convertido e o segundo inteiro é a base desejada.

Output

A função deve retornar uma string representando o número convertido na base desejada.

Example

Input

"12;2"

Output

"1100"
