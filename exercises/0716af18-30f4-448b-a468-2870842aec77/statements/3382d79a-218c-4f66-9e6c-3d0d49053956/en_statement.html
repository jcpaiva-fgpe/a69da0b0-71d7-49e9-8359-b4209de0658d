Number Converter

You are asked to create a function that converts a given number into its corresponding string representation. You need to implement separate functions to convert a decimal number to binary, octal and hexadecimal representation.

Task

Write a function that takes two parameters, a number and a base, and returns the string representation of that number in the given base.

Input

The input should contain two integers separated by a semicolon. The first integer is the number to be converted and the second integer is the desired base.

Output

The function should return a string representing the converted number in the desired base.

Example

Input

"12;2"

Output

"1100"
