Converting Decimal to Binary

Your friend is trying to understand the binary system and he asked for your help. He wants to convert decimal numbers to binary, but he doesn't know how to do it. You decided to create a function that receives an integer decimal number and returns its binary representation.

Task

Create a function that receives an integer decimal number and returns its binary representation.

Input

An integer decimal number

Output

A string representing the binary representation of the input number

Example

Input

"13"

Output

"1101"
