Conversão Decimal para Binário

Seu amigo está tentando entender o sistema binário e pediu sua ajuda. Ele deseja converter números decimais para binário, mas não sabe como fazê-lo. Você decidiu criar uma função que recebe um número decimal inteiro e retorna sua representação binária.

Task

Crie uma função que recebe um número decimal inteiro e retorna sua representação binária.

Input

Um número decimal inteiro

Output

Uma string representando a representação binária do número de entrada

Example

Input

"13"

Output

"1101"
