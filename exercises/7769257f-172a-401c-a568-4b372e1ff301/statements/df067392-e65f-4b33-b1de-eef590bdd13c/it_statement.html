Conversione di tempo

Data un'ora nel formato AM / PM a 12 ore, convertila in formato militare (24 ore).

Task

Scrivi una funzione che, dato un formato stringa di un'ora a 12 ore (cioè hh:mm:ss AM o hh:mm:ss PM), restituisce una stringa che rappresenta quella stessa ora nel formato a 24 ore (cioè hh:mm:ss), dove 00 <= hh <= 23

Input

Una stringa che rappresenta un'ora nel formato a 12 ore

Output

Una stringa che rappresenta quella stessa ora nel formato a 24 ore

Example

Input

"07:05:45PM"

Output

"19:05:45"
