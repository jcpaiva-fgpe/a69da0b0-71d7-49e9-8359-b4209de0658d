Conversão de Tempo

Dado um tempo no formato de 12 horas AM / PM, converta-o para o formato militar (24 horas).

Task

Escreva uma função que, dada uma string representando um tempo no formato de 12 horas (ou seja, hh:mm:ss AM ou hh:mm:ss PM), retorne uma string representando esse mesmo tempo no formato de 24 horas (ou seja, hh:mm:ss), onde 00 <= hh <= 23

Input

Uma string que representa um tempo no formato de 12 horas

Output

Uma string que representa esse mesmo tempo no formato de 24 horas

Example

Input

"07:05:45PM"

Output

"19:05:45"
