Time Conversion

Given a time in 12-hour AM/PM format, convert it to military (24-hour) time.

Task

Write a function that given a string representing a time in 12-hour format (i.e. hh:mm:ssAM or hh:mm:ssPM), returns a string representing that same time in 24-hour format (i.e. hh:mm:ss) where 00 <= hh <= 23

Input

A string representing a time in 12-hour format

Output

A string representing the same time in 24-hour format

Example

Input

"07:05:45PM"

Output

"19:05:45"
