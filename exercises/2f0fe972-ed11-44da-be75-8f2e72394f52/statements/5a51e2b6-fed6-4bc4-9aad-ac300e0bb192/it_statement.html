Verificatore di Numeri Primi

Un numero primo è un numero naturale maggiore di 1 che non è il prodotto di due numeri naturali più piccoli.

Task

Scrivi una funzione per verificare se un numero dato è un numero primo.

Input

Un numero intero positivo maggiore di 1.

Output

Un valore booleano che indica se il numero dato è primo o meno.

Example

Input

"7"

Output

"True"
