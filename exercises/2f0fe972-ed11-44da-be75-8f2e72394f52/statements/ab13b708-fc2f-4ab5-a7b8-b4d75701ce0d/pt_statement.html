Verificador de Número Primo

Um número primo é um número natural maior que 1 que não é produto de dois números naturais menores.

Task

Escreva uma função para verificar se um número dado é um número primo.

Input

Um número inteiro positivo maior que 1.

Output

Um valor booleano indicando se o número dado é primo ou não.

Example

Input

"7"

Output

"True"
