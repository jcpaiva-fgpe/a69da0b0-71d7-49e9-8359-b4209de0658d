Substring Palindrômica

Task

Escreva uma função que receba uma string e retorne a substring palindrômica mais longa contida nela. Um palíndromo é uma palavra, frase, número ou outra sequência de caracteres que pode ser lida da mesma forma da esquerda para a direita e da direita para a esquerda.

Input

A entrada será uma única string não vazia s composta por letras minúsculas do alfabeto inglês

Output

Retorne uma string que seja a substring palindrômica

Example

Input

"babad"

Output

"bab"
