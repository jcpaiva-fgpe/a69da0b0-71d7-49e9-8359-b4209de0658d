Podciąg Palindromowy

Task

Napisz funkcję, która przyjmuje ciąg znaków i zwraca najdłuższy palindromowy podciąg w tym ciągu. Palindromem może być słowo, fraza, liczba lub inna sekwencja znaków, która czytana jest tak samo od przodu jak i od tyłu.

Input

Wejściem będzie pojedynczy niepusty ciąg znaków s składający się z małych liter alfabetu angielskiego

Output

Zwróć ciąg znaków, który jest podciągiem palindromowym

Example

Input

"babad"

Output

"bab"
