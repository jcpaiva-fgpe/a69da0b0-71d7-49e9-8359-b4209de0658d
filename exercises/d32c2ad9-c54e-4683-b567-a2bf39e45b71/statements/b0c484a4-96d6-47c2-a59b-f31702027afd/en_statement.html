Palindromic Substring

Task

Write a function that takes a string and returns the longest palindromic substring in that string. A palindrome is defined as a word, phrase, number, or other sequence of characters which reads the same backwards and forwards.

Input

The input will be a single non-empty string s consisting of lowercase English letters

Output

Return a string which is the palindrome substring

Example

Input

"babad"

Output

"bab"
