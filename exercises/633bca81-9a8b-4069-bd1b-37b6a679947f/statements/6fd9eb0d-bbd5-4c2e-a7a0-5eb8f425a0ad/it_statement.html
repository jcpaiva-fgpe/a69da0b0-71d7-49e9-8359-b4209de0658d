Massimizza il Profitto

Lavori per una società di trading che gestisce diverse valute. Il tuo compito è sviluppare una funzione che determinerà il massimo profitto che può essere fatto da un elenco dato di tassi di cambio di valuta. Puoi comprare e vendere valute quante volte vuoi, ma devi vendere una valuta prima di ricomprarla. Inoltre, puoi tenere solo una valuta alla volta.

Task

Data una lista di tassi di cambio di valuta, scrivere una funzione per determinare il massimo profitto che può essere fatto. Puoi assumere che l'elenco di input non sia vuoto e contenga almeno due elementi.

Input

L'input sarà una lista di numeri interi positivi che rappresentano i tassi di cambio di valuta.

Output

L'output dovrebbe essere un singolo numero che rappresenti il massimo profitto che può essere fatto.

Example

Input

"[7,1,5,3,6,4]"

Output

"7"
