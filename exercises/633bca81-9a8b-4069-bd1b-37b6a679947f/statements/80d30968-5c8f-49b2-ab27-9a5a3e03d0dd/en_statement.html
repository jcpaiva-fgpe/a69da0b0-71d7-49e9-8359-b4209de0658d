Maximize Profit

You are working for a trading firm that deals with different currencies. Your job is to develop a function that will determine the maximum profit that can be made from a given list of currency exchange rates. You can buy and sell currencies as many times as you want, but you have to sell a currency before buying it again. Also, you can only hold one currency at a time.

Task

Given a list of currency exchange rates, write a function to determine the maximum profit that can be made. You can assume that the input list is non-empty and contains at least two elements.

Input

The input will be a list of positive integers representing currency exchange rates.

Output

The output should be a single number representing the maximum profit that can be made.

Example

Input

"[7,1,5,3,6,4]"

Output

"7"
