Sum of Squares

Given an integer N, calculate the sum of the squares of the first N natural numbers.

Task

Write a function that receives an integer N and returns the sum of the squares of the first N natural numbers.

Input

The input parameter is an integer N (1 <= N <= 1000).

Output

The function must return an integer representing the sum of the squares of the first N natural numbers.

Example

Input

"4"

Output

"30"
