Soma dos Quadrados

Dado um número inteiro N, calcule a soma dos quadrados dos N primeiros números naturais.

Task

Escreva uma função que receba um inteiro N e retorne a soma dos quadrados dos N primeiros números naturais.

Input

O parâmetro de entrada é um inteiro N (1 <= N <= 1000).

Output

A função deve retornar um inteiro representando a soma dos quadrados dos N primeiros números naturais.

Example

Input

"4"

Output

"30"
