Suma Kwadratów

Dla danego całkowitego N, oblicz sumę kwadratów pierwszych N liczb naturalnych.

Task

Napisz funkcję, która przyjmuje całkowite N i zwraca sumę kwadratów pierwszych N liczb naturalnych.

Input

Parametrem wejściowym jest całkowite N (1 <= N <= 1000).

Output

Funkcja musi zwrócić całkowitą wartość będącą sumą kwadratów pierwszych N liczb naturalnych.

Example

Input

"4"

Output

"30"
