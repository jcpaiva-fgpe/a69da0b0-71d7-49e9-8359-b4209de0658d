Sprawdzarka hasła

Tworzysz system logowania dla witryny internetowej i chcesz upewnić się, że hasła użytkowników spełniają pewne kryteria. Postanawiasz napisać program, który sprawdzi, czy hasło jest prawidłowe czy nie.

Task

Napisz funkcję, która przyjmuje ciąg znaków jako wejście i zwraca 'Valid', jeśli hasło spełnia następujące kryteria, a 'Invalid' w przeciwnym razie. Kryteria są następujące: - Długość co najmniej 8 znaków - Zawiera co najmniej jedną wielką literę, jedną małą literę i jedną cyfrę.

Input

Ciąg znaków reprezentujący hasło.

Output

Ciąg znaków reprezentujący, czy hasło jest prawidłowe czy nie.

Example

Input

"Abcdef1!"

Output

"Valid"
