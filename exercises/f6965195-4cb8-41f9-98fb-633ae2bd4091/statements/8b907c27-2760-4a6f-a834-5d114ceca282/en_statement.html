Password Checker

You are developing a login system for a website and want to make sure that users' passwords meet certain criteria. You decide to write a program to check if a password is valid or not.

Task

Write a function that takes a string as input and returns 'Valid' if the password meets the following criteria and 'Invalid' otherwise. The criteria are: - At least 8 characters long - Contains at least one uppercase letter, one lowercase letter and one digit.

Input

A string representing the password.

Output

A string representing whether the password is valid or not.

Example

Input

"Abcdef1!"

Output

"Valid"
