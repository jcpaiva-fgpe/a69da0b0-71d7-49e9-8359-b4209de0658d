Verificatore di Password

Stai sviluppando un sistema di accesso per un sito web e vuoi assicurarti che le password degli utenti soddisfino determinati criteri. Decidi di scrivere un programma per verificare se una password è valida o meno.

Task

Scrivi una funzione che prende in input una stringa e restituisce 'Valid' se la password soddisfa i seguenti criteri e 'Invalid' altrimenti. I criteri sono: - Lunga almeno 8 caratteri - Contiene almeno una lettera maiuscola, una lettera minuscola e un numero.

Input

Una stringa che rappresenta la password.

Output

Una stringa che rappresenta se la password è valida o meno.

Example

Input

"Abcdef1!"

Output

"Valid"
