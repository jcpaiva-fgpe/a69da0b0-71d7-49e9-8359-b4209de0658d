Verificador de Senha

Você está desenvolvendo um sistema de login para um site e quer ter certeza de que as senhas dos usuários atendem a certos critérios. Você decide escrever um programa para verificar se uma senha é válida ou não.

Task

Escreva uma função que receba uma string como entrada e retorne 'Válido' se a senha cumprir os seguintes critérios e 'Inválido' caso contrário. Os critérios são: - Pelo menos 8 caracteres de comprimento - Contém pelo menos uma letra maiúscula, uma letra minúscula e um dígito.

Input

Uma string que representa a senha.

Output

Uma string que representa se a senha é válida ou não.

Example

Input

"Abcdef1!"

Output

"Valid"
