Licznik Częstotliwości Tekstu

Pracujesz dla firmy, która chce analizować opinie użytkowników na temat swojego produktu. Chcą wiedzieć, które słowa pojawiają się najczęściej w opinii. Zostałeś poproszony o stworzenie programu, który może zliczać częstotliwość słów w danym bloku tekstowym.

Task

Napisz funkcję, która pobiera ciąg tekstu i zwraca słownik zawierający częstotliwość każdego słowa w tekście. Funkcja powinna ignorować interpunkcję i nie zależeć od wielkości liter.

Input

Ciąg tekstu.

Output

Słownik zawierający częstotliwość każdego słowa.

Example

Input

"This is a test. This is only a test."

Output

"{\"this\":2,\"is\":2,\"a\":2,\"test\":2,\"only\":1}"
