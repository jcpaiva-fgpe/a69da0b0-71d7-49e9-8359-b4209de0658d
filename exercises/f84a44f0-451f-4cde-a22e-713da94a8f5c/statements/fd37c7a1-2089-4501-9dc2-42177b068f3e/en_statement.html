Text Frequency Counter

You work for a company that wants to analyze user feedback to their product. They want to know which words show up the most in the feedback. You have been tasked with creating a program that can count the frequency of words in a given block of text.

Task

Write a function that takes in a string of text and returns a dictionary containing the frequency of each word in the text. The function should ignore punctuation and be case-insensitive.

Input

A string of text.

Output

A dictionary containing the frequency of each word.

Example

Input

"This is a test. This is only a test."

Output

"{\"this\":2,\"is\":2,\"a\":2,\"test\":2,\"only\":1}"
