Contador de Frequências de Texto

Você trabalha para uma empresa que deseja analisar o feedback dos usuários sobre seus produtos. Eles querem saber quais palavras aparecem com mais frequência no feedback. Você foi encarregado de criar um programa que possa contar a frequência das palavras em um determinado bloco de texto.

Task

Escreva uma função que receba uma string de texto e retorne um dicionário contendo a frequência de cada palavra no texto. A função deve ignorar a pontuação e não diferenciar maiúsculas e minúsculas.

Input

Uma string de texto.

Output

Um dicionário contendo a frequência de cada palavra.

Example

Input

"This is a test. This is only a test."

Output

"{\"this\":2,\"is\":2,\"a\":2,\"test\":2,\"only\":1}"
