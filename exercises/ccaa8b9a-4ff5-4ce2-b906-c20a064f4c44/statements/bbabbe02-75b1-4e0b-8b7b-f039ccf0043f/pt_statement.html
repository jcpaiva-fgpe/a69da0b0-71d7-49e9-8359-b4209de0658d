Verificador de Permutação de Palíndromos

Um palíndromo é uma palavra que é a mesma quando lida de trás para frente ou da frente para trás. Uma permutação é uma reorganização das letras. Por exemplo, a palavra 'ovo' é um palíndromo. Uma permutação de 'ovo' é 'voo'. Neste exercício, você receberá uma string e terá que determinar se é uma permutação de um palíndromo.

Task

Escreva uma função que recebe uma string e retorna True se for uma permutação de um palíndromo, e False caso contrário.

Input

Uma string contendo apenas letras minúsculas sem espaços em branco

Output

Um valor booleano indicando se a string de entrada é uma permutação de um palíndromo

Example

Input

"tactcoa"

Output

"True"
