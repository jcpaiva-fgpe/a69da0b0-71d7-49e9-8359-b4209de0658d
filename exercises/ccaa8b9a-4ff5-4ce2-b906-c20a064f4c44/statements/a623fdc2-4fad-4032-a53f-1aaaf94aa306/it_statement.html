Verificatore di permutazioni di palindromi

Un palindromo è una parola che è la stessa quando la leggi dall'inizio alla fine o dalla fine all'inizio. Una permutazione è una riorganizzazione delle lettere. Ad esempio, la parola 'radar' è un palindromo. Una permutazione di 'radar' è 'rarad'. In questo esercizio, ti verrà data una stringa e dovrai determinare se è una permutazione di un palindromo.

Task

Scrivi una funzione che prende in input una stringa e restituisce True se è una permutazione di un palindromo, False altrimenti.

Input

Una stringa composta solo da lettere minuscole senza spazi

Output

Un valore booleano che indica se la stringa di input è una permutazione di un palindromo

Example

Input

"tactcoa"

Output

"True"
