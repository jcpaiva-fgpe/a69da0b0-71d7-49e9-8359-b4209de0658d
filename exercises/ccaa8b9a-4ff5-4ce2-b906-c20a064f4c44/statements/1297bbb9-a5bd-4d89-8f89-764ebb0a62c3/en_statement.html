Palindrome Permutation Checker

A palindrome is a word that is the same when you read it from start to finish or from finish to start. A permutation is a rearrangement of the letters. For example, the word 'racecar' is a palindrome. A permutation of 'racecar' is 'carrace'. In this exercise, you will be given a string and you have to determine whether it is a permutation of a palindrome.

Task

Write a function that takes in a string and returns True if it is a permutation of a palindrome, and False otherwise.

Input

A string consisting of only lowercase letters without any spaces

Output

A boolean value indicating whether the input string is a permutation of a palindrome

Example

Input

"tactcoa"

Output

"True"
