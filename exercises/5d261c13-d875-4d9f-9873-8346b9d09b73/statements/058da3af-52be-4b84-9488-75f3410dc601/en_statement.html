Palindrome Checker

Palindrome is a word, phrase or number that reads the same backward as forward. This can be useful in different areas like cryptography, literature or simply as a fun challenge. In this exercise, you should create a function that receives a string as input and checks if it is a palindrome or not.

Task

Create a function that receives a string and checks if it is a palindrome. The function should return True if the string is a palindrome and False otherwise.

Input

The input is a string of alphanumeric characters (capitalized and lowercase letters are considered the same) and may contain spaces and special characters.

Output

The output should be a boolean value: True if the input string is a palindrome and False otherwise.

Example

Input

"A man, a plan, a canal, Panama!"

Output

true
