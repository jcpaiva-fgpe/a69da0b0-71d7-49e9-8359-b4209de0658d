Inverter Palavras em uma Frase

Você recebe uma frase e precisa inverter a ordem das palavras na frase

Task

Escreva uma função que aceite uma frase como parâmetro e retorne a frase com as palavras em ordem reversa. As palavras na frase invertida devem ser separadas por um único espaço.

Input

A entrada será uma string representando uma frase

Output

A saída deve ser uma string representando a frase de entrada com as palavras em ordem reversa

Example

Input

"The quick brown fox jumps over the lazy dog"

Output

"dog lazy the over jumps fox brown quick The"
