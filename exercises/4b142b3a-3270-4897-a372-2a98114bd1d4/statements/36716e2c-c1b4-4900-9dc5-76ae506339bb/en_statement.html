Reverse Words in a Sentence

You're given a sentence and you have to reverse the order of the words in the sentence

Task

Write a function that accepts a sentence as a parameter and returns the sentence with the words in reverse order. The words in the reversed sentence should be separated by a single space.

Input

The input will be a string representing a sentence

Output

The output should be a string representing the input sentence with the words in reverse order

Example

Input

"The quick brown fox jumps over the lazy dog"

Output

"dog lazy the over jumps fox brown quick The"
