Shopping List

You are going grocery shopping and want to keep track of the items you need to buy. Instead of making a physical list, you want to create an app to help you organize your shopping list digitally.

Task

Create a function that takes in a list of strings representing items and returns a formatted shopping list in alphabetical order.

Input

The function takes in a list of strings.

Output

The function outputs a formatted shopping list in alphabetical order.

Example

Input

["eggs","milk","bread","cheese"]

Output

"Your Shopping List:\n\n1. Bread\n2. Cheese\n3. Eggs\n4. Milk"
