Lista de Compras

Você irá ao supermercado e gostaria de manter controle dos itens que precisa comprar. Em vez de fazer uma lista física, você deseja criar um aplicativo para ajudá-lo a organizar sua lista de compras digitalmente.

Task

Crie uma função que receba uma lista de strings representando itens e retorne uma lista de compras formatada em ordem alfabética.

Input

A função recebe uma lista de strings.

Output

A função retorna uma lista de compras formatada em ordem alfabética.

Example

Input

["eggs","milk","bread","cheese"]

Output

"Your Shopping List:\n\n1. Bread\n2. Cheese\n3. Eggs\n4. Milk"
