Contador de Vogais

Você foi convidado a criar um programa que lê um texto de entrada e retorna a quantidade de vogais presente nele.

Task

Crie uma função que receba uma string e retorne a quantidade de vogais presentes.

Input

Uma string de caracteres

Output

Um número inteiro com a contagem de vogais presentes na string de entrada

Example

Input

"hello world"

Output

3
