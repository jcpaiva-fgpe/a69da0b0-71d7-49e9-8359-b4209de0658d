Valid Anagram

Given two strings s and t, return true if t is an anagram of s, and false otherwise.

Task

Implement a function that receives two strings and returns a boolean indicating if they are valid anagrams.

Input

The input consists of two string s and t separated by a semicolon (;)

Output

The output is a boolean value indicating whether or not the two strings are valid anagrams.

Example

Input

"listen;silent"

Output

"true"
