Anagrama Válido

Dadas duas strings s e t, retorne true se t for um anagrama de s e false caso contrário.

Task

Implemente uma função que receba duas strings e retorne um boolean indicando se elas são anagramas válidos.

Input

O input consiste em duas strings s e t separadas por um ponto e vírgula (;)

Output

O output é um valor booleano indicando se as duas strings são anagramas válidos ou não.

Example

Input

"listen;silent"

Output

"true"
