Poprawny Anagram

Mając dwie ciągi znaków s i t, zwróć true, jeśli t jest anagramem s, a false w przeciwnym razie.

Task

Zaimplementuj funkcję, która przyjmuje dwa ciągi znaków i zwraca boolowską wartość wskazującą, czy są one poprawnymi anagramami.

Input

Wejście składa się z dwóch ciągów znaków s i t oddzielonych średnikiem (;)

Output

Wyjście to wartość boolowska wskazująca, czy dwa ciągi znaków są poprawnymi anagramami czy nie.

Example

Input

"listen;silent"

Output

"true"
