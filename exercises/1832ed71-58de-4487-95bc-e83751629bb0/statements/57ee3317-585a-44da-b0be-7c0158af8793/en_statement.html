Largest Word in a Sentence

Your task is to write a function that should receive a sentence as a parameter and should evaluate what is the largest word in it.

Task

Create a function that receives a sentence (string) and returns the largest word in it. If two words have the same length, return the first one.

Input

A string representing the sentence to analyze.

Output

The largest word in the sentence.

Example

Input

"The quick brown fox jumped over the lazy dog"

Output

"jumped"
