Maior palavra em uma frase

Sua tarefa é escrever uma função que deve receber uma frase como parâmetro e avaliar qual é a palavra mais longa nela.

Task

Crie uma função que receba uma frase (string) e retorne a maior palavra que está na mesma. Se duas palavras possuem o mesmo tamanho, retorne a primeira.

Input

Uma string representando a frase a ser analisada.

Output

A maior palavra da frase.

Example

Input

"The quick brown fox jumped over the lazy dog"

Output

"jumped"
