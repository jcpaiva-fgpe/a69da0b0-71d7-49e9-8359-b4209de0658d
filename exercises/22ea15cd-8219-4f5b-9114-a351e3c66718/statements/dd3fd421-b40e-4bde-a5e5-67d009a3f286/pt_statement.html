Conversor de Numerais Romanos

Os numerais romanos foram usados antes do nosso sistema numérico atual para representar valores. Nos tempos modernos, os numerais romanos ainda são usados para alguns fins, como numerar capítulos de livros, sequências de filmes e até mesmo em relógios. Este exercício envolve a criação de um programa para converter numerais romanos para o nosso sistema numérico atual.

Task

Crie uma função que receba uma entrada string no formato de um numeral romano e retorne o número equivalente no nosso sistema atual.

Input

A entrada consistirá em uma string representando um numeral romano. O numeral terá no máximo comprimento 15.

Output

A saída deve ser um único número inteiro representando o valor do numeral romano de entrada.

Example

Input

"LXVIII"

Output

68
