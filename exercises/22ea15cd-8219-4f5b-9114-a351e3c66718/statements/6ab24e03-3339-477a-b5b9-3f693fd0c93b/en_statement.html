Roman Numeral Converter

Roman numerals were used before our current number system to represent values. In modern times, Roman numerals are still used for some purposes such as numbering book chapters, movie sequels, and even on clocks. This exercise involves creating a program to convert Roman numerals to our current number system.

Task

Create a function that takes a string input in the form of a Roman numeral, and returns the equivalent number in our current system.

Input

The input will consist of a string representing a Roman numeral. The numeral will be at most length 15.

Output

The output should be a single integer representing the value of the input Roman numeral.

Example

Input

"LXVIII"

Output

68
