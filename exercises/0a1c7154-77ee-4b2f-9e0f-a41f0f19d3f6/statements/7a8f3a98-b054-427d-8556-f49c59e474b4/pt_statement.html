Verificador de strings únicas

Você precisa escrever uma função que receba uma string e retorne True se todos os caracteres da string forem únicos e False caso contrário.

Task

Escreva uma função que receba uma string e retorne True se todos os caracteres da string forem únicos e False caso contrário.

Input

Uma string de qualquer tamanho.

Output

True se todos os caracteres da string forem únicos e False caso contrário.

Example

Input

"hello"

Output

"False"
