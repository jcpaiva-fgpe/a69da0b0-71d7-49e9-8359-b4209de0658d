Sprawdzanie unikalnych łańcuchów znaków

Musisz napisać funkcję, która otrzymuje łańcuch znaków i zwraca True, jeśli wszystkie znaki w ciągu są unikalne, a False w przeciwnym razie.

Task

Napisz funkcję, która otrzymuje łańcuch znaków i zwraca True, jeśli wszystkie znaki w ciągu są unikalne, a False w przeciwnym razie.

Input

Łańcuch znaków o dowolnej długości.

Output

True, jeśli wszystkie znaki w ciągu są unikalne, a False w przeciwnym razie.

Example

Input

"hello"

Output

"False"
