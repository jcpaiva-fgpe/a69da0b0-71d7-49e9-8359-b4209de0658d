Maximum Profit Calculator

You have a list of stock prices for a single stock. You want to maximize your profit by buying low and selling high. You can only make one transaction (buying and selling counts as one transaction) and you cannot short-sell (buying after selling).

Task

Create a function that receives a list of integers representing the stock price for each day and returns the maximum profit that can be obtained by buying one time and selling one time.

Input

A list of n integers (1 <= n <= 10^5) representing the price of the stock for each day (1 <= stock_price <= 10^4)

Output

An integer representing the maximum profit that can be obtained from buying and selling the stock once. If no profit can be obtained, return 0.

Example

Input

"[7, 1, 5, 3, 6, 4]"

Output

"5"
