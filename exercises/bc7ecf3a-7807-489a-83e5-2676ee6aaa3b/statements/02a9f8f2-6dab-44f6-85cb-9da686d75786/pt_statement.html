Calculadora de lucro máximo

Você tem uma lista de preços de ações de uma única ação. Você deseja maximizar seu lucro comprando barato e vendendo caro. Você pode fazer apenas uma transação (comprar e vender contam como uma transação) e não pode vendê-la sem tê-la comprado antes.

Task

Crie uma função que receba uma lista de inteiros representando o preço das ações para cada dia e retorne o lucro máximo que pode ser obtido comprando uma vez e vendendo uma vez.

Input

Uma lista de n inteiros (1 <= n <= 10^5) representando o preço da ação para cada dia (1 <= stock_price <= 10^4)

Output

Um inteiro representando o lucro máximo que pode ser obtido comprando e vendendo a ação uma vez. Se nenhum lucro puder ser obtido, retorne 0.

Example

Input

"[7, 1, 5, 3, 6, 4]"

Output

"5"
