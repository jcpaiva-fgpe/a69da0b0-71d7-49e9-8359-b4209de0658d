Kalkulator maksymalnego zysku

Masz listę cen akcji dla jednej akcji. Chcesz zwiększyć swój zysk, kupując tanio i sprzedając drogo. Możesz dokonać tylko jednej transakcji (kupno i sprzedaż stanowi jedną transakcję) i nie możesz krótko sprzedawać (kupować po sprzedaży).

Task

Utwórz funkcję, która przyjmuje listę liczb całkowitych reprezentujących cenę akcji dla każdego dnia i zwraca maksymalny zysk, jaki można uzyskać, kupując raz i sprzedając raz.

Input

Lista n liczb całkowitych (1 <= n <= 10^5) reprezentująca cenę akcji dla każdego dnia (1 <= stock_price <= 10^4).

Output

Liczba całkowita reprezentująca maksymalny zysk, który można uzyskać, kupując i sprzedając akcje raz. Jeśli nie można uzyskać żadnego zysku, zwróć 0.

Example

Input

"[7, 1, 5, 3, 6, 4]"

Output

"5"
