Calcolatrice del massimo profitto

Hai una lista di prezzi di azioni per una singola azione. Vuoi massimizzare il tuo profitto comprando a basso prezzo e vendendo a prezzo alto. Puoi effettuare una sola transazione (comprare e vendere conta come una transazione) e non puoi fare short-selling (comprare dopo aver venduto).

Task

Creare una funzione che riceva una lista di interi che rappresentano il prezzo delle azioni per ogni giorno e restituisca il massimo profitto che può essere ottenuto comprando una volta e vendendo una volta.

Input

Una lista di n interi (1 <= n <= 10^5) che rappresentano il prezzo dell'azione per ogni giorno (1 <= stock_price <= 10^4)

Output

Un intero che rappresenta il massimo profitto che può essere ottenuto acquistando e vendendo l'azione una volta. Se non è possibile ottenere alcun profitto, restituire 0.

Example

Input

"[7, 1, 5, 3, 6, 4]"

Output

"5"
