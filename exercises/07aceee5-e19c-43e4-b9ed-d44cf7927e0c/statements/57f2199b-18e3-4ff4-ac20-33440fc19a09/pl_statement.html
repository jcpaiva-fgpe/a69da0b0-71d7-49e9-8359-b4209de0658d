Weryfikator adresu IP

Tworzysz oprogramowanie do komunikacji sieciowej i chcesz weryfikować adresy IP wprowadzane przez użytkowników. Weryfikacja adresu IP zapewnia, że wprowadzony adres IP jest poprawny i zmniejsza ryzyko wystąpienia błędów w oprogramowaniu.

Task

Napisz funkcję, która przyjmuje ciąg znaków jako wejście i zwraca True, jeśli wejście ma poprawny format adresu IP, w przeciwnym razie False. Poprawny adres IP składa się z czterech liczb całkowitych oddzielonych kropkami. Każda liczba całkowita musi mieścić się w zakresie od 0 do 255, włącznie z końcową liczbą.

Input

Ciąg znaków zawierający potencjalny adres IP.

Output

Wartość logiczna wskazująca, czy wejście jest poprawnym adresem IP, czy nie.

Example

Input

"192.168.1.1"

Output

true
