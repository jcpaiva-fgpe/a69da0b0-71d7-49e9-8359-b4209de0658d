Validador de Endereço IP

Você está desenvolvendo um software de comunicação em rede e quer validar endereços IP que são inseridos pelos usuários. Validar o IP garante que o IP inserido é um endereço válido e reduz o risco de bugs no software.

Task

Escreva uma função que recebe uma string como entrada e retorna True se a entrada seguir o formato de um endereço IP válido, caso contrário retorna False. Um endereço IP válido tem quatro valores inteiros separados por pontos. Cada valor inteiro deve estar entre 0 e 255, inclusive.

Input

Uma string contendo um endereço IP potencial.

Output

Um valor booleano indicando se a entrada é um endereço IP válido ou não.

Example

Input

"192.168.1.1"

Output

true
