Validatore di Indirizzo IP

Stai costruendo un software di comunicazione di rete e vuoi convalidare gli indirizzi IP che vengono inseriti dagli utenti. La convalida dell'IP garantisce che l'IP inserito sia un indirizzo valido e riduce il rischio di bug nel software.

Task

Scrivi una funzione che prende una stringa in input e restituisce True se l'input è un formato di indirizzo IP valido, altrimenti False. Un indirizzo IP valido ha quattro valori interi separati da punti. Ogni valore intero deve essere compreso tra 0 e 255, inclusi.

Input

Una stringa che contiene un potenziale indirizzo IP.

Output

Un valore booleano che indica se l'input è un indirizzo IP valido o meno.

Example

Input

"192.168.1.1"

Output

true
