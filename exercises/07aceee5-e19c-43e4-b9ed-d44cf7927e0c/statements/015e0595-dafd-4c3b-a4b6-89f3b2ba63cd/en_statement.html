IP Address Validator

You are building a network communication software and want to validate IP addresses that are entered by users. Validating the IP ensures that the entered IP is a valid address and reduces the risk of bugs in the software.

Task

Write a function that takes a string as input and returns True if the input is a valid IP address format, otherwise False. A valid IP address has four integer values separated by dots. Each integer value must be between 0 and 255, inclusive.

Input

A string containing a potential IP address.

Output

A boolean value indicating whether the input is a valid IP address or not.

Example

Input

"192.168.1.1"

Output

true
