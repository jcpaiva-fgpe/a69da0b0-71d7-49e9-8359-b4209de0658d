Salti minimi

Ti viene data un'array di interi in cui ogni elemento rappresenta il massimo numero di passi che possono essere fatti in avanti da quell'elemento. Scrivi una funzione per restituire il numero minimo di salti che devi fare per arrivare dall'inizio alla fine dell'array.

Task

Scrivi una funzione che riceva un'array di interi e restituisca il numero minimo di salti necessari per raggiungere la fine dell'array. Se non è possibile raggiungere la fine dell'array, restituisci -1.

Input

La funzione riceve un'array di interi.

Output

La funzione dovrebbe restituire un intero, che rappresenta il numero minimo di salti necessari per raggiungere la fine dell'array. Se non è possibile raggiungere la fine dell'array, restituisci -1.

Example

Input

"[2, 3, 1, 1, 4]"

Output

2
