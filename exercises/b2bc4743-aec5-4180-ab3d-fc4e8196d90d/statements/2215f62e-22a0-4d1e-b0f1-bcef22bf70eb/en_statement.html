Minimum Jumps

You are given an array of integers where each element represents the maximum number of steps that can be jumped going forward from that element. Write a function to return the minimum number of jumps you must take to get from the start to the end of the array.

Task

Write a function that takes in an array of integers and returns the minimum number of jumps needed to reach the end of the array. If it's not possible to reach the end of the array, return -1.

Input

The function takes in an array of integers.

Output

The function should return an integer, which represents the minimum number of jumps needed to reach the end of the array. If it's not possible to reach the end of the array, return -1.

Example

Input

"[2, 3, 1, 1, 4]"

Output

2
