Saltos mínimos

Você recebe uma matriz de inteiros onde cada elemento representa o número máximo de etapas que podem ser puladas indo para a frente a partir desse elemento. Escreva uma função para retornar o número mínimo de saltos que você deve fazer para chegar do início ao final da matriz.

Task

Escreva uma função que receba uma matriz de inteiros e retorne o número mínimo de saltos necessários para chegar ao final da matriz. Se não for possível atingir o final da matriz, retorne -1.

Input

A função recebe uma matriz de inteiros.

Output

A função deve retornar um inteiro, que representa o número mínimo de saltos necessários para chegar ao final da matriz. Se não for possível atingir o final da matriz, retorne -1.

Example

Input

"[2, 3, 1, 1, 4]"

Output

2
