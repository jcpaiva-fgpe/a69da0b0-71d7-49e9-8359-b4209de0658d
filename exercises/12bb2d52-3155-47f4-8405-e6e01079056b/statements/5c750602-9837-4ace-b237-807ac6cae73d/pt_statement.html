Ordenando Palavras

Você recebeu uma lista de palavras e foi pedido para as ordenar em ordem alfabética. Você precisa escrever uma função para te ajudar nesta tarefa.

Task

Escreva uma função que recebe uma lista de palavras como entrada e retorna a mesma lista ordenada em ordem alfabética.

Input

Uma lista de n palavras, onde 1 <= n <= 1000. Cada palavra será composta apenas por letras minúsculas e maiúsculas.

Output

Uma lista de n palavras ordenada em ordem alfabética.

Example

Input

["banana","apple","orange"]

Output

["apple","banana","orange"]
