Ordinamento Parole

Ti è stata data una lista di parole e ti è stato chiesto di ordinarle in ordine alfabetico. Devi scrivere una funzione per aiutarti a svolgere questo compito.

Task

Scrivi una funzione che prenda una lista di parole in input e restituisca la stessa lista ordinata in ordine alfabetico.

Input

Una lista di n parole, dove 1 <= n <= 1000. Ogni parola sarà composta solo da lettere minuscole e maiuscole.

Output

Una lista di n parole ordinata in ordine alfabetico.

Example

Input

["banana","apple","orange"]

Output

["apple","banana","orange"]
