Sortowanie Słów

Otrzymałeś / otrzymałaś listę słów i poproszono cię, aby posortować je w porządku alfabetycznym. Musisz napisać funkcję, która pomoże ci w wykonaniu tego zadania.

Task

Napisz funkcję, która przyjmuje listę słów jako dane wejściowe i zwraca tę samą listę posortowaną w porządku alfabetycznym.

Input

Lista n słów, gdzie 1 <= n <= 1000. Każde słowo będzie składało się tylko z małych i wielkich liter.

Output

Lista n słów posortowana w porządku alfabetycznym.

Example

Input

["banana","apple","orange"]

Output

["apple","banana","orange"]
