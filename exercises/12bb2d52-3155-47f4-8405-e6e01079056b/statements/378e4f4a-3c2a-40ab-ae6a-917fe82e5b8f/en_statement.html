Sorting Words

You have been given a list of words and have been asked to sort them in alphabetical order. You need to write a function to help you perform this task.

Task

Write a function that takes a list of words as input and returns the same list sorted in alphabetical order.

Input

A list of n words, where 1 <= n <= 1000. Each word will be composed of lowercase and uppercase letters only.

Output

A list of n words sorted in alphabetical order.

Example

Input

["banana","apple","orange"]

Output

["apple","banana","orange"]
