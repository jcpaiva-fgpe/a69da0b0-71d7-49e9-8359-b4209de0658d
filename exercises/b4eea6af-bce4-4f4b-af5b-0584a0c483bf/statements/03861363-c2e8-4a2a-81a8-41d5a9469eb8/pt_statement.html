Calculadora Simples

Você trabalha para um site de tutoria de matemática e você tem que criar uma calculadora simples para os alunos usarem.

Task

Crie uma função que receba três entradas: um número, um operador (+, -, *, /) e um segundo número, e execute a operação correspondente.

Input

Três entradas: primeiro número, operador, segundo número. Todos eles devem ser separados por ponto e vírgula.

Output

Um único número, resultado da operação.

Example

Input

"5;*;6"

Output

"30"
