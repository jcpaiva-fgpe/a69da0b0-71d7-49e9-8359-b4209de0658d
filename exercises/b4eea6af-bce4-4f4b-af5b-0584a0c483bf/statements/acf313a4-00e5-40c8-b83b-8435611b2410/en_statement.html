Simple Calculator

You are working for a math tutoring website and you have to create a simple calculator for the students to use.

Task

Create a function that takes three inputs: a number, an operator (+, -, *, /), and a second number, and performs the corresponding operation.

Input

Three inputs: first number, operator, second number. All of them should be separated with semicolon.

Output

A single number, result of the operation.

Example

Input

"5;*;6"

Output

"30"
