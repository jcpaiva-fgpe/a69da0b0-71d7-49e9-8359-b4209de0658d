Prosty kalkulator

Pracujesz dla strony internetowej zajmującej się nauką matematyki i musisz stworzyć prosty kalkulator, który uczniowie będą mogli używać.

Task

Stwórz funkcję, która przyjmuje trzy argumenty: liczba, operator (+, -, *, /) oraz druga liczba i wykonuje odpowiednią operację.

Input

Trzy argumenty: pierwsza liczba, operator, druga liczba. Wszyscy powinni być oddzieleni średnikiem.

Output

Jedna liczba, wynik operacji.

Example

Input

"5;*;6"

Output

"30"
