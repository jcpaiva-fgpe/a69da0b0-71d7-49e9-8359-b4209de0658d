Dígitos Únicos

Você recebe um número e precisa determinar se todos os dígitos nele são únicos.

Task

Escreva uma função que receba um inteiro como entrada e retorne 'Verdadeiro' se todos os seus dígitos são únicos e 'Falso' caso contrário.

Input

A função recebe um único inteiro como entrada.

Output

A função deve retornar um booleano indicando se todos os dígitos são únicos.

Example

Input

"3978"

Output

"True"
