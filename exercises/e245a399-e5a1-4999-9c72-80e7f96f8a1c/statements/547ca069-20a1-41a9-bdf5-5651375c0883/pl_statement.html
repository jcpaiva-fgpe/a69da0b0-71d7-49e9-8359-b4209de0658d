Unikalne Cyfry

Dostajesz liczbę i musisz określić, czy wszystkie cyfry w niej są unikalne.

Task

Napisz funkcję, która przyjmuje jako wejście liczbę całkowitą i zwraca 'Prawda', jeśli wszystkie jej cyfry są unikalne, a 'Fałsz' w przeciwnym razie.

Input

Funkcja przyjmuje pojedynczą liczbę całkowitą jako dane wejściowe.

Output

Funkcja powinna zwrócić wartość boolowską wskazującą, czy wszystkie cyfry są unikalne.

Example

Input

"3978"

Output

"True"
