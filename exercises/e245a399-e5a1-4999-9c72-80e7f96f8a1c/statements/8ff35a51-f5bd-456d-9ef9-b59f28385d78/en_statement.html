Unique Digits

You are given a number, and you need to determine if all the digits in it are unique.

Task

Write a function that takes an integer as input and returns 'True' if all its digits are unique, and 'False' otherwise.

Input

The function takes a single integer as input.

Output

The function should return a boolean indicating if all the digits are unique.

Example

Input

"3978"

Output

"True"
