Aproximação da Raiz Quadrada

É importante para os programas de computador calcular valores rapidamente - às vezes mesmo aproximações - para otimizar seu desempenho. Um exemplo clássico de um algoritmo de aproximação é aquele que calcula a raiz quadrada de um número.

Task

Dado um número inteiro positivo n, crie uma função que retorna uma aproximação da raiz quadrada de n sem utilizar funções de biblioteca interna.

Input

A entrada será um número inteiro positivo n (1 <= n <= 10 ^ 9)

Output

A saída deve ser um número de ponto flutuante com 2 casas decimais representando uma aproximação da raiz quadrada de n.

Example

Input

"25"

Output

"5.00"
