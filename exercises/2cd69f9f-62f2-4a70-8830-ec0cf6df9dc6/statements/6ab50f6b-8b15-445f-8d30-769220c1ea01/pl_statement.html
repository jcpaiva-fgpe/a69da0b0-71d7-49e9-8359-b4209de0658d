Aproksymacja Pierwiastka Kwadratowego

Dla programów komputerowych ważne jest szybkie obliczanie wartości - czasami nawet przybliżonych - w celu zoptymalizowania ich wydajności. Klasycznym przykładem algorytmu przybliżającego jest ten, który oblicza pierwiastek kwadratowy z liczby.

Task

Dla dodatniej liczby całkowitej n stwórz funkcję, która zwraca przybliżenie pierwiastka kwadratowego z n bez korzystania z wbudowanych funkcji bibliotecznych.

Input

Wejściem będzie dodatnia liczba całkowita n (1 <= n <= 10^9)

Output

Wyjście powinno być liczbą zmiennoprzecinkową z 2 miejscami po przecinku reprezentującą przybliżenie pierwiastka kwadratowego z n.

Example

Input

"25"

Output

"5.00"
