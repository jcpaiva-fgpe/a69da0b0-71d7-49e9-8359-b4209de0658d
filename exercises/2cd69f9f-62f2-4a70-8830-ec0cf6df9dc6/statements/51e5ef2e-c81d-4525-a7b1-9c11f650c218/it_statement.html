Approssimazione della Radice Quadrata

È importante per i programmi informatici calcolare rapidamente i valori - talvolta anche approssimativi - per ottimizzare le loro prestazioni. Un classico esempio di algoritmo di approssimazione è quello che calcola la radice quadrata di un numero.

Task

Dato un numero intero positivo n, creare una funzione che restituisce un'approssimazione della radice quadrata di n senza utilizzare funzioni di libreria incorporate.

Input

L'input sarà un numero intero positivo n (1 <= n <= 10^9)

Output

L'output dovrebbe essere un numero in virgola mobile con 2 cifre decimali che rappresenta un'approssimazione della radice quadrata di n.

Example

Input

"25"

Output

"5.00"
