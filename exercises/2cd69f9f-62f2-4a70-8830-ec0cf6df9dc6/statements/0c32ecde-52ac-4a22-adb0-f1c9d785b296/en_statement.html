Square Root Approximation

It is important for computer programs to quickly calculate values - sometimes even approximate ones - to optimize their performance. One classic example of an approximation algorithm is the one that calculates the square root of a number.

Task

Given a positive integer n, create a function that returns an approximation of the square root of n without using built-in library functions.

Input

Input will be a positive integer n (1 <= n <= 10^9)

Output

Output should be a floating-point number with 2 decimal places representing an approximation of the square root of n.

Example

Input

"25"

Output

"5.00"
