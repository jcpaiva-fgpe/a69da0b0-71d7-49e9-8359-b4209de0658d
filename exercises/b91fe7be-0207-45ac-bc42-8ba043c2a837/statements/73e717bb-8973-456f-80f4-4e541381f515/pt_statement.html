Encontrar Substring

Você tem que encontrar substrings em uma string dada.

Task

Escreva uma função que receba dois strings como entrada, a string principal e a substring que você deseja encontrar e retorne o número de vezes que a substring ocorre na string principal.

Input

A entrada consistirá em duas strings - a string principal (S) e a substring (s).

Output

A saída será um único número inteiro - o número de vezes que a substring ocorre na string principal.

Example

Input

"hello world;lo"

Output

"1"
