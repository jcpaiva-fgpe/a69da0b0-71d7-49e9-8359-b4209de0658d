Wyszukiwanie Podciągu

Musisz znaleźć podciągi w podanym ciągu znaków.

Task

Napisz funkcję, która przyjmuje dwa ciągi znaków jako wejście - ciąg główny oraz podciąg, który chcesz znaleźć - i zwraca liczbę wystąpień podciągu w ciągu głównym.

Input

Wejście składać się będzie z dwóch ciągów znaków: ciągu głównego (S) oraz podciągu (s).

Output

Wyjście będzie jedną liczbą całkowitą - liczbą wystąpień podciągu w ciągu głównym.

Example

Input

"hello world;lo"

Output

"1"
