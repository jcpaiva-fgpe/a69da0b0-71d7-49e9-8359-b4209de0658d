Substring Finder

You have to find subtrings in a given string.

Task

Write a function that takes two strings as input, the main string and the substring you want to find and returns the number of times the substring occurs in the main string.

Input

The input will consist of two strings - the main string (S) and the substring (s).

Output

The output will be a single integer - the number of times the substring occurs in the main string.

Example

Input

"hello world;lo"

Output

"1"
