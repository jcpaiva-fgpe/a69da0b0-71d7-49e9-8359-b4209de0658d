Sprawdzacz Wieków

Twoim zadaniem jest napisanie funkcji przyjmującej liczbe całkowitą, która reprezentuje rok urodzenia danej osoby. Funkcja ma zwrócić string mówiący, czy dana osoba osiągnęła pełnoletność (wiek 18) czy nie.

Task

Stwórz funkcję, która przyjmuje liczbę całkowitą, która reprezentuje rok urodzenia danej osoby i zwraca string mówiący, czy dana osoba osiągnęła pełnoletność czy nie.

Input

Funkcja przyjmuje liczbę całkowitą reprezentującą rok urodzenia danej osoby.

Output

Funkcja powinna zwrócić string mówiący, czy dana osoba osiągnęła pełnoletność czy nie.

Example

Input

"2005"

Output

"Not of legal age"
