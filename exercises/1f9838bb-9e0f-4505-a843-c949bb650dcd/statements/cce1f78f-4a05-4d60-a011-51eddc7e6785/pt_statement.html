Verificador de Idade

Seu trabalho é construir uma função que irá receber um número inteiro representando o ano de nascimento de alguém e retornar uma string dizendo se a pessoa atingiu a maioridade (idade 18) ou não.

Task

Crie uma função que receba um número representando o ano de nascimento de alguém e retorne uma string dizendo se a pessoa atingiu a maioridade ou não.

Input

A função receberá um inteiro representando o ano de nascimento de alguém.

Output

A função deve retornar uma string afirmando se a pessoa atingiu a maioridade ou não.

Example

Input

"2005"

Output

"Not of legal age"
