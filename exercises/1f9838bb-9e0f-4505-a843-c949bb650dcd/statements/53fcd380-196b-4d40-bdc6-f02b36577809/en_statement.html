Age Checker

Your task here building a function which will receive an integer representing someone's birth year and return a string saying if the person reached the age of majority (age 18) or not.

Task

Create a function that receives a number representing someone's birth year and returns a string saying if the person reached the age of majority or not.

Input

The function will receive an integer representing a birth year.

Output

The function should return a string stating if the person reached the age of majority or not.

Example

Input

"2005"

Output

"Not of legal age"
