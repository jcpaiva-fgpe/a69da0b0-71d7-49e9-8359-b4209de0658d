The Pythagorean Theorem

The Pythagorean Theorem is one of the fundamental concepts in mathematics. It describes the relationship between the sides of a right triangle. The theorem states that the square of the length of the hypotenuse of a right triangle is equal to the sum of the squares of the other two sides. In this exercise, you will be asked to implement a function that calculates the length of the hypotenuse of a right triangle given the length of the other two sides.

Task

Write a function that takes two positive integer values representing the lengths of the non-hypotenuse sides of a right triangle and returns the length of the hypotenuse. If the input is invalid (i.e. negative numbers, non-integers, non-triangle values), return None or raise an exception.

Input

Two positive integers separated by a semicolon (;) representing the lengths of the non-hypotenuse sides of a triangle.

Output

A positive float representing the length of the hypotenuse if the input is valid. None or an exception if the input is invalid.

Example

Input

"3;4"

Output

"5.0"
