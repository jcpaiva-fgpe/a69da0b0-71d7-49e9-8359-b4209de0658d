Twierdzenie Pitagorasa

Twierdzenie Pitagorasa jest jednym z podstawowych pojęć matematyki. Opisuje ono zależność między bokami trójkąta prostokątnego. Twierdzenie mówi, że kwadrat długości przeciwprostokątnej trójkąta prostokątnego jest równy sumie kwadratów długości pozostałych dwóch boków. W tym zadaniu będziesz musiał zaimplementować funkcję obliczającą długość przeciwprostokątnej trójkąta prostokątnego, mając daną długość pozostałych dwóch boków.

Task

Napisz funkcję, która przyjmuje dwie dodatnie wartości całkowite reprezentujące długości nie przeciwprostokątnej stron trójkąta prostokątnego i zwraca długość przeciwprostokątnej. Jeśli dane wejściowe są nieprawidłowe (np. ujemne liczby, liczby niecałkowite, wartości, które nie tworzą trójkąta), zwróć wartość None lub zgłoś wyjątek.

Input

Dwie dodatnie liczby całkowite oddzielone średnikiem (;) reprezentujące długości nie przeciwprostokątnej boków trójkąta.

Output

Dodatnia wartość zmiennoprzecinkowa reprezentująca długość przeciwprostokątnej, jeśli dane wejściowe są poprawne. None lub wyjątek, jeśli dane wejściowe są nieprawidłowe.

Example

Input

"3;4"

Output

"5.0"
