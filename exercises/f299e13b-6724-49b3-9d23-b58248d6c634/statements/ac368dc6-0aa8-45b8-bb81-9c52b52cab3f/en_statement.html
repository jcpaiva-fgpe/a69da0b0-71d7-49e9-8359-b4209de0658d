Binary Search

You want to create a function that implements the binary search algorithm to find the position of a given value within a sorted list.

Task

Create a function that implements the binary search algorithm to find the position of a given value within a sorted list.

Input

The function should receive two parameters: a sorted list and the value to be searched for.

Output

The function should return the index of the value if it exists in the list, otherwise it should return -1.

Example

Input

"[1,2,3,4,6,7];6"

Output

"4"
