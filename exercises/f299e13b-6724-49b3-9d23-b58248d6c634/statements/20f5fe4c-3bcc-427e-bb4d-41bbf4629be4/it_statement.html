Ricerca Binaria

Si desidera creare una funzione che implementi l'algoritmo di ricerca binaria per trovare la posizione di un determinato valore all'interno di un elenco ordinato.

Task

Crea una funzione che implementi l'algoritmo di ricerca binaria per trovare la posizione di un determinato valore all'interno di un elenco ordinato.

Input

La funzione dovrebbe ricevere due parametri: un elenco ordinato e il valore da cercare.

Output

La funzione dovrebbe restituire l'indice del valore se esiste nell'elenco, altrimenti dovrebbe restituire -1.

Example

Input

"[1,2,3,4,6,7];6"

Output

"4"
