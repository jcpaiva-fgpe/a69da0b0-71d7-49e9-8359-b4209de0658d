Busca Binária

Você deseja criar uma função que implemente o algoritmo de busca binária para encontrar a posição de um determinado valor dentro de uma lista ordenada.

Task

Crie uma função que implemente o algoritmo de busca binária para encontrar a posição de um determinado valor dentro de uma lista ordenada.

Input

A função deve receber dois parâmetros: uma lista ordenada e o valor a ser procurado.

Output

A função deve retornar o índice do valor se ele existir na lista, caso contrário deve retornar -1.

Example

Input

"[1,2,3,4,6,7];6"

Output

"4"
