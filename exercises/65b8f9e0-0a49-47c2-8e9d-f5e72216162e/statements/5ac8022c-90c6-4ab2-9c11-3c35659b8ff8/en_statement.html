The Missing Number

You have a list of n-1 integers and these integers are in the range of 1 to n. There are no duplicates in the list. One of the integers is missing in the list. Your task is to find the missing integer.

Task

Write a function that takes in a list and returns the missing integer.

Input

A list of n-1 integers in the range of 1 to n where n >= 2

Output

The missing integer in the given list

Example

Input

"[1, 2, 4, 5]"

Output

"3"
