Task

Scrivi una funzione che prende in input una lista e restituisce il numero mancante.

Input

Una lista di n-1 numeri interi nell'intervallo da 1 a n dove n >= 2

Output

Il numero mancante nell'elenco dato

Example

Input

Output

"3"
