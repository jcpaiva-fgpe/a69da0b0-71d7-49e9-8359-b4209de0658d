Kolejność Liczb

Musisz stworzyć program, który generuje sekwencję liczb zaczynając od 1 do n. Mówiąc inaczej, dla danego całkowitego n wygeneruj 1, 2, ..., n. Ten rodzaj problemu jest powszechnie spotykany w matematyce i informatyce.

Task

Stwórz funkcję, która przyjmuje liczbę całkowitą n i zwraca listę sekwencji liczb od 1 do n.

Input

Funkcja przyjmuje liczbę całkowitą n (1 <= n <= 100).

Output

Funkcja musi zwrócić listę sekwencji liczb od 1 do n.

Example

Input

"5"

Output

"[1, 2, 3, 4, 5]"
