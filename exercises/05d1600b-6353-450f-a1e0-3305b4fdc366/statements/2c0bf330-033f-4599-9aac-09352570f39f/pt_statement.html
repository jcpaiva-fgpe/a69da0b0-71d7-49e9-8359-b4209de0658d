Contador de Palavras

Você foi encarregado de criar um programa que conta o número de palavras em uma string fornecida.

Task

Escreva uma função que recebe uma string como entrada e retorna o número de palavras na string.

Input

Uma única string.

Output

Um inteiro representando o número de palavras na string de entrada.

Example

Input

"The quick brown fox jumps over the lazy dog"

Output

"9"
