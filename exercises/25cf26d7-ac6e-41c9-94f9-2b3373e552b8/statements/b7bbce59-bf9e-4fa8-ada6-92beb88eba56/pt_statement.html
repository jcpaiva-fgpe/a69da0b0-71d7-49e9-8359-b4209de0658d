Verificador de Fizz Buzz

Fizz Buzz é um jogo jogado entre muitos programadores para testar suas habilidades de codificação. O jogo é jogado com um grupo de pessoas dando voltas para dizer números a partir de 1. No entanto, quando o número é um múltiplo de 3, o jogador deve dizer 'Fizz' em vez disso. Se o número for um múltiplo de 5, o jogador deve dizer 'Buzz'. Se o número for um múltiplo de ambos 3 e 5, o jogador deve dizer 'Fizz Buzz'. Este exercício envolve a criação de uma função que verifica se um determinado número adere às regras do Fizz Buzz.

Task

Crie uma função que receba um número como entrada e retorne 'Fizz' se o número for um múltiplo de 3, 'Buzz' se o número for um múltiplo de 5, 'Fizz Buzz' se o número for um múltiplo de ambos 3 e 5, ou o número em si se nenhuma das condições for atendida.

Input

Um número inteiro n onde 1 <= n <= 10^9

Output

Produza uma string representando 'Fizz', 'Buzz', 'Fizz Buzz' ou o número de entrada como uma string.

Example

Input

"15"

Output

"'Fizz Buzz'"
