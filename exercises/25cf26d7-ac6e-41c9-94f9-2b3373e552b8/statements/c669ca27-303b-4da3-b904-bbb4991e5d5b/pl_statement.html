Sprawdzacz Fizz Buzz

Fizz Buzz to gra rozgrywana przez wielu programistów w celu przetestowania ich umiejętności programowania. Gra jest rozgrywana przez grupę ludzi, którzy na zmianę mówią liczby zaczynając od 1. Jednak gdy liczba jest wielokrotnością liczby 3, gracz powinien powiedzieć „Fizz”. Jeśli liczba jest wielokrotnością liczby 5, gracz powinien powiedzieć „Buzz”. Jeśli liczba jest wielokrotnością zarówno 3, jak i 5, gracz powinien powiedzieć „Fizz Buzz”. Ćwiczenie to polega na stworzeniu funkcji, która sprawdza, czy podana liczba spełnia zasady Fizz Buzz.

Task

Stwórz funkcję, która przyjmuje liczbę jako danych wejściowych i zwraca 'Fizz' jeśli liczba jest wielokrotnością liczby 3, 'Buzz' jeśli liczba jest wielokrotnością liczby 5, 'Fizz Buzz' jeśli liczba jest wielokrotnością zarówno 3, jak i 5, lub samą liczbę, jeśli żadne z warunków nie zostanie spełnione.

Input

Liczbę całkowitą n, gdzie 1 <= n <= 10^9

Output

Wyjściem jest ciąg znaków reprezentujący 'Fizz', 'Buzz', 'Fizz Buzz' lub liczbę wejściową jako ciąg znaków.

Example

Input

"15"

Output

"'Fizz Buzz'"
