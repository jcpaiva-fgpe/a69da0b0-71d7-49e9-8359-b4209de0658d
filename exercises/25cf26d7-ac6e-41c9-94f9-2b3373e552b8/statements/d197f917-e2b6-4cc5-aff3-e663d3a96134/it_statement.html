Controllatore Fizz Buzz

Fizz Buzz è un gioco giocato tra molti programmatori per testare le loro abilità di codifica. Il gioco si gioca con un gruppo di persone che si alternano per dire i numeri a partire da 1. Tuttavia, quando il numero è un multiplo di 3, il giocatore dovrebbe dire 'Fizz' invece. Se il numero è un multiplo di 5, il giocatore dovrebbe dire 'Buzz'. Se il numero è un multiplo sia di 3 che di 5, il giocatore dovrebbe dire 'Fizz Buzz'. Questo esercizio prevede la creazione di una funzione che verifica se un dato numero aderisce alle regole di Fizz Buzz.

Task

Crea una funzione che prende un numero in input e restituisce 'Fizz' se il numero è un multiplo di 3, 'Buzz' se il numero è un multiplo di 5, 'Fizz Buzz' se il numero è un multiplo di entrambi 3 e 5, o il numero stesso se nessuna delle condizioni è soddisfatta.

Input

Un intero n dove 1 <= n <= 10^9

Output

Produci una stringa che rappresenta 'Fizz', 'Buzz', 'Fizz Buzz' o il numero di input come stringa.

Example

Input

"15"

Output

"'Fizz Buzz'"
