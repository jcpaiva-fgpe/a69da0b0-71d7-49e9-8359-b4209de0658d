Różnica Parzystych i Nieparzystych

Dostajesz tablicę liczb całkowitych, Twoim zadaniem jest znaleźć różnicę między sumą liczb parzystych i nieparzystych.

Task

Napisz funkcję, która przyjmuje tablicę liczb całkowitych i zwraca dodatnią różnicę między sumą liczb parzystych i nieparzystych.

Input

Funkcja przyjmuje tablicę liczb całkowitych.

Output

Funkcja powinna zwracać liczbę całkowitą.

Example

Input

"[1,2,3,4,5,6]"

Output

"3"
