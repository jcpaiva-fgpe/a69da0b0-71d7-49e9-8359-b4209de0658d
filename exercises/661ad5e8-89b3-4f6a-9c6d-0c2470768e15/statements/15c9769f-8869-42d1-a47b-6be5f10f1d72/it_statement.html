Differenza tra Pari e Dispari

Ti è dato un array di interi, il tuo compito è trovare la differenza tra la somma dei numeri pari e dei numeri dispari.

Task

Scrivi una funzione che prenda in input un array di interi e restituisca la differenza positiva tra la somma dei numeri pari e dei numeri dispari.

Input

La funzione prende in input un array di interi.

Output

La funzione dovrebbe restituire un intero.

Example

Input

"[1,2,3,4,5,6]"

Output

"3"
