Diferença Par-Impar

Você recebe um array de inteiros, sua tarefa é encontrar a diferença entre a soma dos números pares e dos números ímpares.

Task

Escreva uma função que receba um array de inteiros e retorne a diferença positiva entre a soma dos números pares e ímpares.

Input

A função recebe um array de inteiros.

Output

A função deve retornar um inteiro.

Example

Input

"[1,2,3,4,5,6]"

Output

"3"
