Even-Odd Difference

You are given an array of integers, your task is to find the difference between the sum of even numbers and odd numbers.

Task

Write a function that takes in an array of integers and returns the positive difference between the sum of even numbers and odd numbers.

Input

The function takes in an array of integers.

Output

The function should return an integer.

Example

Input

"[1,2,3,4,5,6]"

Output

"3"
