Password Strength Checker

In today's world, there are many online services available that require users to create logins and passwords to access them.

Task

You need to write a function that takes a string as an input which represents a password and returns a string indicating the strength of the password. You can define a strong password as one that contains at least one uppercase letter, one lowercase letter, one digit and one special character (!@#$%^&*).

Input

A string representing a password.

Output

A string indicating the strength of the password ('Weak', 'Moderate', 'Strong', 'Very Strong').

Example

Input

"P@ssw0rd!"

Output

"Strong"
