Verifica della forza della password

Nel mondo di oggi, ci sono molti servizi online disponibili che richiedono agli utenti di creare login e password per accedervi.

Task

Devi scrivere una funzione che prenda una stringa come input che rappresenta una password e restituisce una stringa che indica la forza della password. Puoi definire una password forte come una che contiene almeno una lettera maiuscola, una lettera minuscola, una cifra e un carattere speciale (! @ # $% ^ & *).

Input

Una stringa che rappresenta una password.

Output

Una stringa che indica la forza della password ('Debole', 'Moderata', 'Forte', 'Molto Forte').

Example

Input

"P@ssw0rd!"

Output

"Strong"
