Verificador de Força de Senha

No mundo de hoje, há muitos serviços online disponíveis que exigem que os usuários criem logins e senhas para acessá-los.

Task

Você precisa escrever uma função que recebe uma string como entrada, que representa uma senha e retorna uma string indicando a força da senha. Você pode definir uma senha forte como aquela que contém pelo menos uma letra maiúscula, uma letra minúscula, um dígito e um caractere especial (! @ # $% ^ & *).

Input

Uma string que representa uma senha.

Output

Uma string indicando a força da senha ('Fraco', 'Moderado', 'Forte', 'Muito Forte').

Example

Input

"P@ssw0rd!"

Output

"Strong"
