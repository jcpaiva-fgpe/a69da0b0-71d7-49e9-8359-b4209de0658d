Anagram Checker

An anagram is a word formed by rearranging the letters of another. For example, the words 'listen' and 'silent' are anagrams of each other. Write a function that takes in two strings and returns True if they are anagrams of each other and False if they are not.

Task

Create a function that takes two strings and returns True if they are anagrams of each other and False if they are not.

Input

The input consists of two strings.

Output

The output is a boolean value indicating whether the two strings are anagrams of each other.

Example

Input

"listen;silent"

Output

"True"
