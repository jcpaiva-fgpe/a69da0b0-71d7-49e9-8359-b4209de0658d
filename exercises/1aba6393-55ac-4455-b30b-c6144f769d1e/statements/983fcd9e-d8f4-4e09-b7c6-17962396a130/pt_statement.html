Verificador de Anagrama

Um anagrama é uma palavra formada pela rearranjo das letras de outra. Por exemplo, as palavras 'ouvir' e 'viro' são anagramas uma da outra. Escreva uma função que recebe duas strings e retorna Verdadeiro se elas são anagramas uma da outra e Falso caso contrário.

Task

Crie uma função que recebe duas strings e retorna Verdadeiro se elas são anagramas uma da outra e Falso caso contrário.

Input

A entrada consiste de duas strings.

Output

A saída é um valor booleano indicando se as duas strings são anagramas uma da outra.

Example

Input

"listen;silent"

Output

"True"
