Sito Eratostenesa

Sito Eratostenesa to starożytny algorytm służący do znajdowania wszystkich liczb pierwszych do określonego limitu. Algorytm działa poprzez iteracyjne oznaczanie jako złożone (tj. nie pierwsze) wielokrotności każdej liczby pierwszej, zaczynając od wielokrotności 2.

Task

Napisz funkcję, która jako wejście będzie przyjmować liczbę całkowitą i zwróci tablicę zawierającą wszystkie liczby pierwsze do i włącznie z tą liczbą, wykorzystując Sito Eratostenesa. Funkcja powinna działać dla dowolnej liczby całkowitej większej lub równej 2.

Input

Wejściem będzie pojedyncza dodatnia liczba całkowita większa lub równa 2.

Output

Wyjściem powinna być tablica wszystkich liczb pierwszych do i włącznie z liczbą wejściową.

Example

Input

"10"

Output

"[2, 3, 5, 7]"
