Crivo de Eratóstenes

O Crivo de Eratóstenes é um antigo algoritmo para encontrar todos os números primos até um limite dado. O algoritmo funciona marcando iterativamente como compostos (ou seja, não primos) os múltiplos de cada número primo, começando com os múltiplos de 2.

Task

Escreva uma função que receba um número inteiro como entrada e retorne um array contendo todos os números primos até e incluindo esse número usando o Crivo de Eratóstenes. A função deve funcionar para qualquer número inteiro maior ou igual a 2.

Input

A entrada será um único número inteiro positivo maior ou igual a 2.

Output

A saída deve ser um array de todos os números primos até e incluindo o inteiro de entrada.

Example

Input

"10"

Output

"[2, 3, 5, 7]"
