Cribro di Eratostene

Il Cribro di Eratostene è un antico algoritmo per trovare tutti i numeri primi fino ad un dato limite. L'algoritmo funziona iterativamente contrassegnando come composti (cioè non primi) i multipli di ciascun numero primo, a partire dai multipli di 2.

Task

Scrivi una funzione che prenda un intero in input e restituisca un array contenente tutti i numeri primi fino a e inclusa quella intera utilizzando il Cribro di Eratostene. La funzione dovrebbe funzionare per qualsiasi intero maggiore o uguale a 2.

Input

L'input sarà un singolo numero intero positivo maggiore o uguale a 2.

Output

L'output dovrebbe essere un array di tutti i numeri primi fino a e inclusa la stessa intera.

Example

Input

"10"

Output

"[2, 3, 5, 7]"
