Sieve of Eratosthenes

The Sieve of Eratosthenes is an ancient algorithm for finding all prime numbers up to a given limit. The algorithm works by iteratively marking as composite (i.e., not prime) the multiples of each prime, starting with the multiples of 2.

Task

Write a function that takes an integer as input and returns an array containing all prime numbers up to and including that integer using the Sieve of Eratosthenes. The function should work for any integer greater than or equal to 2.

Input

The input will be a single positive integer greater than or equal to 2.

Output

The output should be an array of all prime numbers up to and including the input integer.

Example

Input

"10"

Output

"[2, 3, 5, 7]"
