Trovare l'Elemento più Piccolo in un Array

Ti viene dato un array di interi e devi scrivere una funzione che trova e restituisce il numero più piccolo in quell'array.

Task

Scrivi una funzione che prende in input un array di interi e restituisce il numero più piccolo in quell'array.

Input

L'input sarà un array di interi con almeno un elemento e al massimo 10^4 elementi. Ogni intero nell'array sarà compreso tra -10^6 e 10^6.

Output

La funzione dovrebbe restituire un singolo intero, che è il numero più piccolo nell'array di input.

Example

Input

"[4, 2, 8, 1, 5]"

Output

"1"
