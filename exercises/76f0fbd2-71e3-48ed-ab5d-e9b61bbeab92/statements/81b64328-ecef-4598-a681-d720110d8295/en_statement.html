Finding the Smallest Element in an Array

You are given an array of integers and you need to write a function that finds and returns the smallest number in that array.

Task

Write a function that takes in an array of integers and returns the smallest number in that array.

Input

The input will be an array of integers with at least one element and at most 10^4 elements. Each integer in the array will be between -10^6 and 10^6.

Output

The function should return a single integer, which is the smallest number in the input array.

Example

Input

"[4, 2, 8, 1, 5]"

Output

"1"
