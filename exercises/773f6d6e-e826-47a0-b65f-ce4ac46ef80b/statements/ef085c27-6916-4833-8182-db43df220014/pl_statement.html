Konwersja czasu

Podany czas w formacie 12-godzinnym AM / PM, przekonwertuj go na format wojskowy (24-godzinny).

Task

Napisz funkcję, która dla podanego łańcucha reprezentującego czas w formacie 12-godzinnym (tj. hh:mm:ss AM lub hh:mm:ss PM) zwraca łańcuch reprezentujący ten sam czas w formacie 24-godzinnym (czyli hh:mm:ss), gdzie 00 <= hh <= 23

Input

Łańcuch reprezentujący czas w formacie 12-godzinnym

Output

Łańcuch reprezentujący ten sam czas w formacie 24-godzinnym

Example

Input

"07:05:45PM"

Output

"19:05:45"
