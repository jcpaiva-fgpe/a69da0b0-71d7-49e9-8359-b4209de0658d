Teste de Psicologia

Um teste de psicologia está sendo desenvolvido para avaliar os traços de personalidade dos estudantes. Uma das tarefas envolve a identificação da palavra mais comum usada pelo indivíduo, pois isso pode fornecer uma ideia do seu processo de pensamento. Sua tarefa é desenvolver uma função que receba uma sentença e retorne a palavra mais usada na sentença.

Task

Desenvolva uma função que receba uma sentença e retorne a palavra mais usada na sentença.

Input

Uma string representando uma sentença

Output

Uma string representando a palavra mais usada na sentença

Example

Input

"The quick brown fox jumped over the lazy dog"

Output

"the"
