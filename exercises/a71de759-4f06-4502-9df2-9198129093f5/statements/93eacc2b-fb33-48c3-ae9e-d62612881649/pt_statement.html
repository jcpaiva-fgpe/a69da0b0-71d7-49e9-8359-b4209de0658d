Caracteres Únicos

Você foi contratado por uma empresa para desenvolver um sistema de senhas que requer que todas as senhas tenham apenas caracteres únicos. Eles querem que você escreva um programa que verifique se uma determinada string tem apenas caracteres únicos ou não.

Task

Escreva uma função que receba uma string como entrada e retorne True se todos os caracteres forem únicos e False caso contrário.

Input

Uma string com até 10^5 caracteres.

Output

Um valor booleano indicando se a string tem apenas caracteres únicos ou não.

Example

Input

"hello"

Output

"False"
