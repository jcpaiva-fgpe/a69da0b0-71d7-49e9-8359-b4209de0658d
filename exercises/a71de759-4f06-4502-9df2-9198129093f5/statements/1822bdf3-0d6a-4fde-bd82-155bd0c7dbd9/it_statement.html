Caratteri Unici

Sei stato assunto da un'azienda per sviluppare un sistema di password che richiede che tutte le password abbiano solo caratteri unici. Vogliono che tu scriva un programma che controlla se una determinata stringa ha solo caratteri unici o meno.

Task

Scrivi una funzione che prenda una stringa come input e restituisca True se tutti i caratteri sono unici e False altrimenti.

Input

Una stringa di massimo 10^5 caratteri.

Output

Un valore booleano che indica se la stringa ha solo caratteri unici o meno.

Example

Input

"hello"

Output

"False"
