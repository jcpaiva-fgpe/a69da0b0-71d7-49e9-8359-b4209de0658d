Unique Characters

You have been hired by a company to develop a password system that requires all passwords to have only unique characters. They want you to write a program that checks if a given string has only unique characters or not.

Task

Write a function that takes a string as an input and returns True if all characters are unique, and False otherwise.

Input

A string of up to 10^5 characters.

Output

A boolean value indicating if the string has only unique characters or not.

Example

Input

"hello"

Output

"False"
