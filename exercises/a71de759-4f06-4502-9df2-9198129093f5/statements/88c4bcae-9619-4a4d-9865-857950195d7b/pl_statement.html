Unikalne znaki

Zostałeś zatrudniony przez firmę do opracowania systemu haseł, który wymaga, aby wszystkie hasła miały tylko unikalne znaki. Chcą, abyś napisał program, który sprawdza, czy podany ciąg zawiera tylko unikalne znaki czy nie.

Task

Napisz funkcję, która przyjmuje ciąg znaków jako dane wejściowe i zwraca wartość True, jeśli wszystkie znaki są unikalne, a False w przeciwnym przypadku.

Input

Ciąg znaków o maksymalnej długości 10^5 znaków.

Output

Wartość logiczna wskazująca, czy ciąg znaków zawiera tylko unikalne znaki, czy nie.

Example

Input

"hello"

Output

"False"
