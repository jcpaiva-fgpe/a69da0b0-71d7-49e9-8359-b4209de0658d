Temperature Converter

In daily life, we encounter the need to convert between the two most common temperature units â€“ degrees Celsius and degrees Fahrenheit. When you read the weather forecast on your mobile phone or computer, it may appear in units you do not prefer. This exercise is about building a function that will do our calculations, whenever you want to convert from Celsius to Fahrenheit and vice versa, This function will be simple but handy whenever we need it.

Task

Create a function that converts temperature from Celsius to Fahrenheit and vice versa.

Input

The input will be a string indicating whether the input temperature value is in Celsius or Fahrenheit.

Output

The output will be a float value representing the converted temperature.

Example

Input

"celsius;100"

Output

"212.0"
