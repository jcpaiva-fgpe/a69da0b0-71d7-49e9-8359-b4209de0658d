Conversor de Temperatura

Na vida diária, encontramos a necessidade de converter entre as duas unidades de temperatura mais comuns - graus Celsius e graus Fahrenheit. Ao ler a previsão do tempo no seu telefone celular ou computador, ela pode aparecer em unidades que você não prefere. Este exercício trata de construir uma função que fará nossos cálculos, sempre que você quiser converter de Celsius para Fahrenheit e vice-versa. Esta função será simples, mas útil sempre que precisarmos dela.

Task

Crie uma função que converta temperatura de Celsius para Fahrenheit e vice-versa

Input

A entrada será uma string indicando se o valor de temperatura de entrada está em Celsius ou Fahrenheit.

Output

A saída será um valor float que representa a temperatura convertida.

Example

Input

"celsius;100"

Output

"212.0"
