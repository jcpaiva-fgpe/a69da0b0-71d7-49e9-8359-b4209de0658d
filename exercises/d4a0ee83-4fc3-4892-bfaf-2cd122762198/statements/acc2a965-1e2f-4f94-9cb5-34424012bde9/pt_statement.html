Conversor

Você está trabalhando em uma empresa que usa medidas em polegadas. No entanto, uma outra empresa que você está trabalhando com utiliza medidas em centímetros. Para facilitar o seu trabalho, você decidiu criar um programa que possa converter polegadas para centímetros ou vice-versa.

Task

Crie uma função que recebe um número e uma string como entrada: se a string for 'polegada', ela deve retornar o valor em centímetros, se a string for 'cm', ela deve retornar o valor em polegadas.

Input

A função recebe um número e uma string como entrada. O número é o valor da medida e a string só pode ser 'polegada' ou 'cm'.

Output

A função deve retornar um float com o valor convertido.

Example

Input

"5;inch"

Output

"12.7"
