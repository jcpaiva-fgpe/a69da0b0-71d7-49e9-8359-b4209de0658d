Converter

You are working at a company that uses numbers in inches. However, another company you are working with uses centimeters. To make your work easier, you decided to create a program that can convert inches to centimeters or vice versa.

Task

Create a function that takes a number and a string as input: if the string is 'inch' it should return the value in centimeters, if the string is 'cm' it should return the value in inches.

Input

The function takes a number and a string as input. The number is the measurement value and the string can only be 'inch' or 'cm'.

Output

The function should return a float with the converted value.

Example

Input

"5;inch"

Output

"12.7"
