Atrasos de Entrega

Uma loja online está tendo dificuldades em controlar os horários de entrega. Eles precisam de uma função que possa calcular o atraso da entrega, dada a data esperada de chegada e a data real de chegada. O atraso deve estar em dias.

Task

Crie uma função que recebe duas strings. A primeira é a data de entrega esperada (formato: 'AAAA-MM-DD') e a segunda é a data de entrega real (formato: 'AAAA-MM-DD'). A função deve retornar o atraso em dias (inteiro). Se a entrega foi feita na data ou antes da data esperada, a função deve retornar 0.

Input

Duas strings com a data de entrega esperada e a data de entrega real.

Output

Um número inteiro representando o atraso em dias. Se a entrega foi feita na data ou antes da data esperada, a função deve retornar 0.

Example

Input

"2022-11-25;2022-11-30"

Output

5
