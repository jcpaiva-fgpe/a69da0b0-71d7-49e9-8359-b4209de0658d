Order Delays

An online store is struggling with keeping track of shipping schedules. They need a function that can calculate the delay of delivery, given the expected arrival date and the actual arrival date. The delay should be in days.

Task

Create a function that receives two strings. The first one is the expected delivery date (format: 'YYYY-MM-DD') and the second one is the actual delivery date (format: 'YYYY-MM-DD'). The function should return the delay in days (integer). If the delivery was made on time or before the expected date, the function should return 0.

Input

Two strings with the expected delivery date and the actual delivery date.

Output

An integer representing the delay in days. If the delivery was made on time or before the expected date, the function should return 0.

Example

Input

"2022-11-25;2022-11-30"

Output

5
