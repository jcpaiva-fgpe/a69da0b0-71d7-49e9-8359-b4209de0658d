Largest Consecutive Sum

Given an array of integers, find the largest sum of consecutive integers within the array.

Task

Write a function that takes in an array of integers and returns the largest sum of consecutive integers within the array.

Input

The function takes in an array of integers.

Output

The function should return a single integer, which is the largest sum of consecutive integers within the array.

Example

Input

"[-2, 1, -3, 4, -1, 2, 1, -5, 4]"

Output

"6"
