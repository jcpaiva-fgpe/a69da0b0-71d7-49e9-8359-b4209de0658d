Maior Soma Consecutiva

Dado um array de inteiros, encontre a maior soma de inteiros consecutivos dentro do array.

Task

Escreva uma função que receba um array de inteiros e retorne a maior soma de inteiros consecutivos dentro do array.

Input

A função recebe um array de inteiros.

Output

A função deve retornar um único inteiro, que é a maior soma de inteiros consecutivos dentro do array.

Example

Input

"[-2, 1, -3, 4, -1, 2, 1, -5, 4]"

Output

"6"
