Somma Consecutiva Più Grande

Dato un array di interi, trovare la somma più grande di interi consecutivi all'interno dell'array.

Task

Scrivi una funzione che prenda in ingresso un array di interi e restituisca la somma più grande di interi consecutivi all'interno dell'array.

Input

La funzione prende in ingresso un array di interi.

Output

La funzione dovrebbe restituire un singolo intero, che è la somma più grande di interi consecutivi all'interno dell'array.

Example

Input

"[-2, 1, -3, 4, -1, 2, 1, -5, 4]"

Output

"6"
