Największa suma kolejnych elementów

Dany jest tablica liczb całkowitych, znajdź największą sumę kolejnych liczb całkowitych w tablicy.

Task

Napisz funkcję, która przyjmuje tablicę liczb całkowitych i zwraca największą sumę kolejnych liczb całkowitych w tablicy.

Input

Funkcja przyjmuje tablicę liczb całkowitych.

Output

Funkcja powinna zwrócić pojedynczą liczbę całkowitą, która jest największą sumą kolejnych liczb całkowitych w tablicy.

Example

Input

"[-2, 1, -3, 4, -1, 2, 1, -5, 4]"

Output

"6"
