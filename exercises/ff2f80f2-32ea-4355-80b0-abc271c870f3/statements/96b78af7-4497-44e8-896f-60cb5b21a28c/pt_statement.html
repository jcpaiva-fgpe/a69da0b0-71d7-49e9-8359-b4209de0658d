Parênteses Correspondentes

Ao escrever um programa, é importante que seu código tenha a estrutura adequada, o que inclui o uso de chaves {} para definir blocos de código. Neste exercício, você escreverá uma função que verifica se os parênteses correspondentes de determinada string estão bem formatados.

Task

Escreva uma função que verifique se os parênteses correspondentes de determinada string estão bem formatados. Retorne Verdadeiro se estiverem e Falso caso não estejam.

Input

Uma string contendo parênteses {}.

Output

Verdadeiro se os parênteses estão corretamente formatados, Falso caso contrário.

Example

Input

"({[]})"

Output

"True"
