Dopasowywanie nawiasów

Podczas pisania programu ważne jest, aby jego struktura była odpowiednio zdefiniowana, co obejmuje użycie nawiasów klamrowych {} do definiowania bloków kodu. W tym ćwiczeniu napiszesz funkcję, która sprawdza, czy nawiasy w danym ciągu są poprawnie dopasowane.

Task

Napisz funkcję, która sprawdza, czy nawiasy w danym ciągu są poprawnie dopasowane. Zwróć wartość True, jeśli są dopasowane albo False, jeśli nie.

Input

Ciąg znaków zawierający nawiasy {}.

Output

True, jeśli nawiasy są poprawnie dopasowane, False w przeciwnym razie.

Example

Input

"({[]})"

Output

"True"
