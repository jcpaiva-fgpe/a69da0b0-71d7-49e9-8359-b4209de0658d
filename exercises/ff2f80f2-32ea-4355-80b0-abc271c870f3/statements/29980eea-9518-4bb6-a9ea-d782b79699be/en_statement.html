Matching Brackets

When writing a program, it's important that your code is properly structured, which includes using brackets {} to define code blocks. In this exercise, you'll write a function that checks if brackets in a given string are properly matched.

Task

Write a function that checks if the brackets in a given string are properly matched. Return True if they are and False if they are not.

Input

A string containing brackets {}.

Output

True if the brackets are properly matched, False otherwise.

Example

Input

"({[]})"

Output

"True"
