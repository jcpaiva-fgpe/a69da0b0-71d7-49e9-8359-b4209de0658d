Somma del sottoarray più grande

Dato un array di numeri interi, trova il sottoarray contiguo con la somma più grande e restituisci la somma.

Task

Scrivi una funzione che prenda un array di numeri interi e restituisca la somma del sottoarray contiguo con la somma più grande.

Input

L'input sarà un array di numeri interi.

Output

L'output sarà un singolo numero intero che rappresenta la somma del sottoarray contiguo più grande.

Example

Input

"[-2,1,-3,4,-1,2,1,-5,4]"

Output

"6"
