Largest Subarray Sum

Given an array of integers, find the contiguous subarray with the largest sum and return the sum.

Task

Write a function that takes an array of integers and returns the sum of the contiguous subarray with the largest sum.

Input

The input will be an array of integers.

Output

The output will be a single integer representing the sum of the largest sum contiguous subarray.

Example

Input

"[-2,1,-3,4,-1,2,1,-5,4]"

Output

"6"
