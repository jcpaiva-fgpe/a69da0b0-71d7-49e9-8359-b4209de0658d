Największa suma podtablicy

Dla danego tablicy liczb całkowitych, znajdź podtablice, w której suma elementów jest największa i zwróć tę sumę.

Task

Napisz funkcję, która przyjmuje tablicę liczb całkowitych i zwraca sumę elementów podtablicy o największej sumie.

Input

Wejściem będzie tablica liczb całkowitych.

Output

Wyjściem będzie jedna liczba całkowita reprezentująca sumę podtablicy o największej sumie.

Example

Input

"[-2,1,-3,4,-1,2,1,-5,4]"

Output

"6"
