Maior Soma de Subarray

Dado um array de inteiros, encontre o subarray contíguo com a maior soma e retorne a soma.

Task

Escreva uma função que receba um array de inteiros e retorne a soma do subarray contíguo com a maior soma.

Input

A entrada será um array de inteiros.

Output

A saída será um único inteiro representando a soma do maior subarray contíguo.

Example

Input

"[-2,1,-3,4,-1,2,1,-5,4]"

Output

"6"
