Powierzchnia trójkąta

Trójkąt definiujemy jako figurę o trzech bokach, a jego powierzchnię możemy obliczyć znając długość podstawy oraz wysokość. Napisz funkcję, która przyjmuje długość podstawy oraz wysokość trójkąta i zwraca jego powierzchnię.

Task

Stwórz funkcję, która jako argumenty przyjmuje długość podstawy oraz wysokość trójkąta i zwraca jego powierzchnię.

Input

Funkcja powinna przyjmować jako argumenty długość podstawy oraz wysokość jako liczby zmiennoprzecinkowe.

Output

Funkcja powinna zwracać jako wynik pole trójkąta jako liczba zmiennoprzecinkowa.

Example

Input

"3.0;4.0"

Output

"6.0"
