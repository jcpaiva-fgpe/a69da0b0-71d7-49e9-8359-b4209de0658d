Area of a Triangle

A triangle is defined as having three sides and its area can be calculated if you know the base length and the height. Write a function that receives the base length and the height of a triangle and returns its area.

Task

Create a function that receives the base length and the height of a triangle and returns its area.

Input

The function should receive the base length and the height as floats.

Output

The function should return the area of the triangle as a float.

Example

Input

"3.0;4.0"

Output

"6.0"
