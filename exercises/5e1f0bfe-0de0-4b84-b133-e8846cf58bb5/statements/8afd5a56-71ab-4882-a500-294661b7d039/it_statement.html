Parola in Numero

Ti viene data una frase contenente parole che si riferiscono alle proprie cifre numeriche.

Task

Scrivi una funzione che prenda in input una frase e restituisca le cifre trovate nella frase, come una stringa.

Input

Una stringa contenente parole e cifre

Output

Una stringa che contiene le cifre trovate nella frase di input

Example

Input

"The number seven is amazing, and eight is too"

Output

"78"
