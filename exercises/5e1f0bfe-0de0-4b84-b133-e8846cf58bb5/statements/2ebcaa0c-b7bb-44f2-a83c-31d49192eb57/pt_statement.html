Palavra para Dígito

Você recebe uma frase contendo palavras que se referem aos seus dígitos numéricos.

Task

Escreva uma função que receba uma frase como entrada e retorne os dígitos encontrados na frase, como uma string.

Input

Uma string contendo palavras e dígitos

Output

Uma string que contenha os dígitos encontrados na frase de entrada

Example

Input

"The number seven is amazing, and eight is too"

Output

"78"
