The Phone Number Validator

You are creating an application for a telecommunications company. One of the features of the application requires you to validate phone numbers for a specific country. The phone number should match the specific format, otherwise it will not be accepted.

Task

Create a function that takes a string as an argument. The function should return 'Valid' if the given string is a valid phone number for the specific country. The phone number should have the following format: [country code]-[area code]-[phone number]. The country code should have a length of 3 digits, the area code should have a length of 3 digits and the phone number should have a length of 7 digits. The only allowed characters are digits 0-9 and '-'

Input

A string that represents the phone number to be validated

Output

A string that says 'Valid' if the given phone number is valid for the specific country or 'Invalid' otherwise

Example

Input

"123-456-7890123"

Output

"Invalid"
