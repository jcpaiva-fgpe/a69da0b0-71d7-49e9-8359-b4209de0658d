Walidator numerów telefonów

Tworzysz aplikację dla firmy telekomunikacyjnej. Jedną z funkcji aplikacji jest konieczność walidacji numerów telefonów dla określonego kraju. Numer telefonu musi odpowiadać określonemu formatowi, w przeciwnym razie nie zostanie zaakceptowany.

Task

Stwórz funkcję, która przyjmuje ciąg znaków jako argument. Funkcja powinna zwrócić 'Valid', jeśli podany ciąg znaków jest poprawnym numerem telefonu dla określonego kraju. Numer telefonu powinien mieć następujący format: [kod kraju]-[kod obszaru]-[numer telefonu]. Kod kraju powinien mieć 3 cyfry, kod obszaru powinien mieć 3 cyfry, a numer telefonu powinien mieć 7 cyfr. Dozwolone są tylko cyfry od 0 do 9 i '-'.

Input

Ciąg znaków reprezentujący numer telefonu do walidacji

Output

Ciąg znaków 'Valid', jeśli podany numer telefonu jest poprawny dla określonego kraju lub 'Invalid' w przeciwnym razie

Example

Input

"123-456-7890123"

Output

"Invalid"
