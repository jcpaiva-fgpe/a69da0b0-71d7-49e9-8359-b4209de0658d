Verificatore di Pangrammi III

Un pangramma è una frase che contiene almeno una volta tutte le lettere dell'alfabeto. Il tuo compito è creare una funzione che riceve una stringa e restituisce se è o non è un pangramma.

Task

Crea una funzione che riceve una stringa e restituisce se è o non è un pangramma.

Input

Una stringa contenente solo lettere minuscole e spazi.

Output

Un valore booleano che indica se la stringa di input è o non è un pangramma.

Example

Input

"the quick brown fox jumps over the lazy dog"

Output

true
