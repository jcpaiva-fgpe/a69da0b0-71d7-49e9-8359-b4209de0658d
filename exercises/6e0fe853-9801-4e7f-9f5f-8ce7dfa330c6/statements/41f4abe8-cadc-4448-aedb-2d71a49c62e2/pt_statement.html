Verificador de Pangrama III

Um pangrama é uma frase que contém todas as letras do alfabeto pelo menos uma vez. Sua tarefa é criar uma função que recebe uma string e retorna se ela é ou não um pangrama.

Task

Crie uma função que recebe uma string e retorna se ela é ou não um pangrama.

Input

Uma string contendo apenas letras minúsculas e espaços.

Output

Um valor booleano indicando se a string de entrada é ou não um pangrama.

Example

Input

"the quick brown fox jumps over the lazy dog"

Output

true
