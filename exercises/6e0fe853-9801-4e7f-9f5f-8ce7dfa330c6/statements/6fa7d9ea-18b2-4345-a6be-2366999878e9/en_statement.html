Pangram Checker III

A pangram is a sentence that contains all the letters of the alphabet at least once. Your task is to create a function that receives a string and returns whether it is a pangram or not.

Task

Create a function that receives a string and returns whether it is a pangram or not.

Input

A string containing only lowercase letters and spaces.

Output

A boolean value indicating whether or not the input string is a pangram.

Example

Input

"the quick brown fox jumps over the lazy dog"

Output

true
