Fatoração de Números Primos

A fatoração de números primos é um conceito importante na matemática. Ela envolve a decomposição de um número composto em seus fatores primos.

Task

Escreva uma função que recebe um número inteiro e retorna seus fatores primos como um array.

Input

A função recebe um número inteiro como entrada.

Output

A função deve retornar um array de fatores primos.

Example

Input

"24"

Output

"[2, 2, 2, 3]"
