Faktoryzacja liczb pierwszych

Faktoryzacja liczb pierwszych to ważne pojęcie matematyczne. Polega ona na rozkładzie liczby złożonej na czynniki pierwsze.

Task

Napisz funkcję, która przyjmuje liczbę całkowitą i zwraca jej czynniki pierwsze w postaci tablicy.

Input

Funkcja przyjmuje liczbę całkowitą jako wejście.

Output

Funkcja powinna zwrócić tablicę czynników pierwszych.

Example

Input

"24"

Output

"[2, 2, 2, 3]"
