Licznik samogłosek

Zadaniem jest stworzenie programu, który odczytuje tekst i zwraca liczbę samogłosek występujących w nim.

Task

Stwórz funkcję, która przyjmuje ciąg znaków i zwraca liczbę samogłosek w nim zawartych.

Input

Łańcuch znaków

Output

Liczba całkowita zawierająca liczbę samogłosek w podanym łańcuchu znaków

Example

Input

"hello world"

Output

3
