Konwersja dziesiętna na binarną

Twoi przyjaciele próbuje zrozumieć system binarny i poprosił o Twoją pomoc. Należy przekonwertować liczby dziesiętne na binarne, ale nie wie jak to zrobić. Zdecydowałeś stworzyć funkcję, która będzie przyjmować liczbę dziesiętną całkowitą i zwracać jej reprezentację binarną.

Task

Utwórz funkcję, która będzie przyjmować liczbę dziesiętną całkowitą i zwracać jej reprezentację binarną.

Input

Liczba dziesiętna całkowita

Output

Ciąg znaków reprezentujący reprezentację binarną podanej liczby

Example

Input

"13"

Output

"1101"
