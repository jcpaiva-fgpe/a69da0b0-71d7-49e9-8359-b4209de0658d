Pangram Checker

A pangram is a sentence that contains every letter of the English alphabet at least once. Write a program to check if a given sentence is a pangram or not.

Task

Create a function that accepts a string as input and returns 'True' if the string is a pangram and 'False' otherwise.

Input

A string of any length.

Output

'True' if the input string is a pangram, 'False' otherwise.

Example

Input

"The quick brown fox jumps over the lazy dog"

Output

"True"
