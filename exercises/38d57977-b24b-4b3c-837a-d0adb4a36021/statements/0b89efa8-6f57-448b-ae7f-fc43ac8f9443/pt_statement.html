Verificador de Pangramas

Um pangrama é uma frase que contém cada letra do alfabeto inglês pelo menos uma vez. Escreva um programa para verificar se uma determinada frase é um pangrama ou não.

Task

Crie uma função que aceita uma string como entrada e retorna 'Verdadeiro' se a string é um pangrama e 'Falso' caso contrário.

Input

Uma string de qualquer tamanho.

Output

'Verdadeiro' se a string de entrada é um pangrama, 'Falso' caso contrário.

Example

Input

"The quick brown fox jumps over the lazy dog"

Output

"True"
