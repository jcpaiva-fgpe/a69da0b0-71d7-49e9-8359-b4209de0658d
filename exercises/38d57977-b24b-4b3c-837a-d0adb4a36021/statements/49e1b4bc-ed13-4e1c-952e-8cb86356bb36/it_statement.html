Verificatore di pangrammi

Un pangramma è una frase che contiene tutte le lettere dell'alfabeto inglese almeno una volta. Scrivi un programma per verificare se una data frase è un pangramma o no.

Task

Crea una funzione che accetti una stringa in ingresso e restituisca 'True' se la stringa è un pangramma e 'False' altrimenti.

Input

Una stringa di qualsiasi lunghezza.

Output

'True' se la stringa in input è un pangramma, 'False' altrimenti.

Example

Input

"The quick brown fox jumps over the lazy dog"

Output

"True"
