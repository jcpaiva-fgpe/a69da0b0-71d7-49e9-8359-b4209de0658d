Verificatore di Parentesi Bilanciate

In programmazione, spesso utilizziamo le parentesi per definire un blocco di codice, come quando scriviamo funzioni o cicli. È importante assicurarsi che le parentesi siano bilanciate, ovvero che ogni parentesi di apertura abbia la corrispondente parentesi di chiusura nella giusta posizione. Questo esercizio ti aiuterà a praticare la verifica se una stringa di parentesi è bilanciata.

Task

Scrivi una funzione che prenda come input una stringa di parentesi e restituisca True se le parentesi sono bilanciate, e False se non lo sono.

Input

L'input della funzione sarà una stringa di parentesi, contenente solo i caratteri '(', ')', '[', ']', '{' e '}'.

Output

La funzione restituirà un valore booleano: True se la stringa di parentesi in input è bilanciata, e False se non è bilanciata.

Example

Input

"()[]{}"

Output

true
