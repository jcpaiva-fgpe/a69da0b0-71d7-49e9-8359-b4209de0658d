Verificador de Parênteses Balanceados

Na programação, utilizamos frequentemente parênteses para definir um bloco de código, como quando escrevemos funções ou loops. É importante verificar se os parênteses estão balanceados, ou seja, cada parêntese de abertura tem um parêntese de fechamento correspondente na ordem correta. Este exercício ajudará você a praticar a verificação se uma sequência de parênteses está balanceada.

Task

Escreva uma função que receba uma string de parênteses como entrada e retorne True se os parênteses estiverem balanceados e False se não estiverem.

Input

A entrada da função será uma string de parênteses, contendo apenas os caracteres '(', ')', '[', ']', '{' e '}'.

Output

A função retornará um valor booleano: True se a string de parênteses de entrada estiver balanceada e False se não estiver balanceada.

Example

Input

"()[]{}"

Output

true
