Balanced Bracket Checker

In programming, we often use brackets to define a block of code, such as when writing functions or loops. It is important to make sure that the brackets are balanced, meaning that each opening bracket has a corresponding closing bracket in the correct order. This exercise will help you practice checking if a string of brackets is balanced.

Task

Write a function that takes a string of brackets as input and returns True if the brackets are balanced, and False if they are not.

Input

The input to the function will be a string of brackets, containing only the characters '(', ')', '[', ']', '{', and '}'.

Output

The function will return a boolean value: True if the input string of brackets is balanced, and False if it is not balanced.

Example

Input

"()[]{}"

Output

true
