Sprawdzacz Zbalansowanych Nawiasów

W programowaniu często używamy nawiasów, aby zdefiniować blok kodu, na przykład podczas pisania funkcji lub pętli. Ważne jest, aby upewnić się, że nawiasy są zbilansowane, co oznacza, że każdy otwierający nawias ma odpowiadający mu nawias zamykający w prawidłowej kolejności. To ćwiczenie pomoże Ci w praktyce sprawdzać, czy ciąg nawiasów jest zbilansowany.

Task

Napisz funkcję, która przyjmuje ciąg nawiasów jako dane wejściowe i zwraca wartość True, jeśli nawiasy są zbilansowane, a False, jeśli nie są.

Input

Dane wejściowe funkcji to ciąg nawiasów zawierający tylko znaki '(', ')', '[', ']', '{' i '}'.

Output

Funkcja zwróci wartość logiczną: True, jeśli ciąg nawiasów wejściowych jest zbilansowany, a False, jeśli nie jest zbilansowany.

Example

Input

"()[]{}"

Output

true
