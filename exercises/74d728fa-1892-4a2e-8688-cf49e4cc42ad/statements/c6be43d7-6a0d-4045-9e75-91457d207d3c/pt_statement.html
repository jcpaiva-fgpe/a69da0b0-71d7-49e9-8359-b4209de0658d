Verificador de Palíndromo

Palíndromo é uma palavra, frase ou número que pode ser lido tanto de trás para frente quanto de frente para trás. Isso pode ser útil em diferentes áreas como criptografia, literatura ou simplesmente como um desafio divertido. Neste exercício, você deve criar uma função que recebe uma string como entrada e verifica se é um palíndromo ou não.

Task

Crie uma função que recebe uma string e verifica se é um palíndromo. A função deve retornar True se a string é palíndromo e False caso contrário.

Input

A entrada é uma string de caracteres alfanuméricos (letras maiúsculas e minúsculas são consideradas iguais) e pode conter espaços e caracteres especiais.

Output

A saída deve ser um valor booleano: True se a string de entrada é um palíndromo e False caso contrário.

Example

Input

"A man, a plan, a canal, Panama!"

Output

true
