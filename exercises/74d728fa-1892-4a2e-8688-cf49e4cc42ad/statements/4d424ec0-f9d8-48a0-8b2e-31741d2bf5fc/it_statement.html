Verificatore di Palindrome

Palindrome è una parola, frase o numero che si legge allo stesso modo in avanti e all'indietro. Questo può essere utile in diversi campi come la crittografia, la letteratura o semplicemente come sfida divertente. In questo esercizio, dovresti creare una funzione che riceve una stringa in input e verifica se è una palindromo o meno.

Task

Crea una funzione che riceve una stringa e verifica se è un palindromo. La funzione dovrebbe restituire True se la stringa è un palindromo e False altrimenti.

Input

L'input è una stringa di caratteri alfanumerici (lettere maiuscole e minuscole sono considerate uguali) e può contenere spazi e caratteri speciali.

Output

L'output dovrebbe essere un valore booleano: True se la stringa in input è un palindromo e False altrimenti.

Example

Input

"A man, a plan, a canal, Panama!"

Output

true
