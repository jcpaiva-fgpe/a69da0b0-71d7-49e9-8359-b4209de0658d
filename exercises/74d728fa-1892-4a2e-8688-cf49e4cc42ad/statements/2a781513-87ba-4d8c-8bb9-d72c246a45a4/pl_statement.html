Sprawdzacz Palindromów

Palindrom to słowo, fraza lub liczba, która czyta się tak samo od tyłu jak od przodu. Może to być przydatne w różnych dziedzinach, takich jak kryptografia, literatura lub po prostu jako zabawne wyzwanie. W tym zadaniu należy utworzyć funkcję, która otrzymuje ciąg znaków jako dane wejściowe i sprawdza, czy jest to palindrom.

Task

Utwórz funkcję, która otrzymuje ciąg znaków i sprawdza, czy jest to palindrom. Funkcja powinna zwrócić True, jeśli łańcuch jest palindromem, a False w przeciwnym razie.

Input

Wejście to ciąg znaków alfanumerycznych (duże i małe litery są uważane za takie same) i może zawierać spacje oraz znaki specjalne.

Output

Wyjście powinno być wartością typu boolean: True, jeśli ciąg wejściowy jest palindromem, a False w przeciwnym przypadku.

Example

Input

"A man, a plan, a canal, Panama!"

Output

true
