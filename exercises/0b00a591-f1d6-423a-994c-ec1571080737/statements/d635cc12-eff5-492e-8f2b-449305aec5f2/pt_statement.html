Maior Substring Sem Caracteres Repetidos

Dada uma string, encontre o comprimento da maior substring sem caracteres repetidos.

Task

Crie uma função que receba uma string e retorne o comprimento da maior substring sem caracteres repetidos.

Input

A entrada será uma string contendo qualquer caractere ASCII.

Output

A saída deve ser um único inteiro que representa o comprimento da maior substring sem caracteres repetidos.

Example

Input

"abcabcbb"

Output

3
