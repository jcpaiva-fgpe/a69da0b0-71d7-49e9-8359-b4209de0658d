Longest Substring Without Repeating Characters

You are given a string, find the length of the longest substring without repeating characters.

Task

Create a function that takes in a string and returns the length of the longest substring without repeating characters.

Input

The input will be a string containing any ASCII character.

Output

The output should be a single integer representing the length of the longest substring without repeating characters.

Example

Input

"abcabcbb"

Output

3
