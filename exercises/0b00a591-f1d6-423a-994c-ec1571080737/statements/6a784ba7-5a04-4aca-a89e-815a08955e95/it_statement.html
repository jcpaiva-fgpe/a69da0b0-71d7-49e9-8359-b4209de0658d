Sottostringa Più Lunga Senza Caratteri Ripetuti

Ti è stata data una stringa, trova la lunghezza della sottostringa più lunga senza caratteri ripetuti.

Task

Crea una funzione che prenda in input una stringa e restituisca la lunghezza della sottostringa più lunga senza caratteri ripetuti.

Input

L'input sarà una stringa contenente qualsiasi carattere ASCII.

Output

L'output dovrebbe essere un singolo intero che rappresenta la lunghezza della sottostringa più lunga senza caratteri ripetuti.

Example

Input

"abcabcbb"

Output

3
