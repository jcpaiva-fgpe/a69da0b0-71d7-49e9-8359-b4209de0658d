Ordinamento di stringhe

Il tuo compito è creare una funzione che ordina una lista di stringhe in ordine alfabetico. Questa funzione dovrebbe essere in grado di gestire stringhe che contengono lettere maiuscole e minuscole.

Task

Crea una funzione che prende una lista di stringhe in input e restituisce la stessa lista, ma ordinata in ordine alfabetico. La classificazione dovrebbe avvenire senza distinguere tra lettere maiuscole e minuscole.

Input

L'input è una lista di stringhe, dove ogni stringa ha lunghezza massima 100.

Output

L'output è la stessa lista di stringhe, ma ordinata in ordine alfabetico, senza caratteri maiuscoli o minuscoli.

Example

Input

["Apple","banana","waterMelon"]

Output

["Apple","banana","waterMelon"]
