Sortowanie ciągów znaków

Twoim zadaniem jest stworzenie funkcji która sortuje liste ciągów znaków w kolejności alfabetycznej. Ta funkcja powinna radzić sobie ze stringami zawierającymi zarówno małe jak i wielkie litery.

Task

Stwórz funkcję która przyjmuje listę ciągów znaków jako parametr i zwraca tę samą listę, ale posortowaną w kolejności alfabetycznej. Sortowanie powinno ignorować wielkość liter.

Input

Parametrem wejściowym jest lista ciągów znaków gdzie każdy ciąg ma maksymalnie długość 100.

Output

Wynik to ta sama lista ciągów znaków, ale posortowana w kolejności alfabetycznej, ignorując wielkość liter.

Example

Input

["Apple","banana","waterMelon"]

Output

["Apple","banana","waterMelon"]
