Sorting strings

Your task is to create a function that sorts a list of strings in alphabetical order. This function should be able to handle strings that contain uppercase and lowercase letters.

Task

Create a function that takes a list of strings as input and returns the same list, but sorted in alphabetical order. The sorting should be case-insensitive.

Input

The input is a list of strings, where each string has at most length 100.

Output

The output is the same list of strings, but sorted in alphabetical order, with case-insensitivity.

Example

Input

["Apple","banana","waterMelon"]

Output

["Apple","banana","waterMelon"]
