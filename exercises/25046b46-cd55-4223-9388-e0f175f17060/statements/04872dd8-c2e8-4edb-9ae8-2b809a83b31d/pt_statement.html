Ordenando strings

Sua tarefa é criar uma função que ordena uma lista de strings em ordem alfabética. Esta função deve ser capaz de lidar com strings que contêm letras maiúsculas e minúsculas.

Task

Crie uma função que recebe uma lista de strings como entrada e retorna a mesma lista, mas ordenada em ordem alfabética. A ordenação deve ser insensível à caixa.

Input

A entrada é uma lista de strings, onde cada string tem no máximo comprimento 100.

Output

A saída é a mesma lista de strings, mas ordenada em ordem alfabética, com insensibilidade à caixa.

Example

Input

["Apple","banana","waterMelon"]

Output

["Apple","banana","waterMelon"]
