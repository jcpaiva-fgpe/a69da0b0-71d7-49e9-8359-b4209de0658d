Birthday Paradox

The birthday paradox is a probability puzzle that involves finding the probability that, in a set of n randomly chosen people, some pair of them will have the same birthday. This puzzle is counterintuitive because our intuition says that the probability should be very low even when the number of people is large. But the actual probability is quite high even for relatively small numbers of people.

Task

Write a function that takes an integer n (1 ≤ n ≤ 365) as input and returns the probability that, in a set of n randomly chosen people, some pair of them will have the same birthday. The probability should be rounded to two decimal places.

Input

The input is an integer n (1 ≤ n ≤ 365) representing the number of people in the set.

Output

The output is a float number rounded to two decimal places representing the probability that some pair of people in the set will have the same birthday.

Example

Input

"23"

Output

"0.51"
