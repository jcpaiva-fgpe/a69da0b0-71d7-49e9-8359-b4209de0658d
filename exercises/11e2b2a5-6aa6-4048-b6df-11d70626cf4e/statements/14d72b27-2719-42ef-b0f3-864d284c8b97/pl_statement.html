Paradoks urodzinowy

Paradoks urodzinowy to zagadka probabilistyczna, która polega na znalezieniu prawdopodobieństwa, że w zestawie n losowo wybranych osób, jakaś para będzie miała tę samą urodzinę. Zagadka ta jest nietypowa, ponieważ intuicja podpowiada, że prawdopodobieństwo powinno być bardzo niskie, nawet gdy liczba ludzi jest duża. Ale rzeczywiste prawdopodobieństwo jest całkiem wysokie nawet dla stosunkowo niewielkich liczb ludzi.

Task

Napisz funkcję, która przyjmuje jako wejście liczbę całkowitą n (1 ≤ n ≤ 365) i zwraca prawdopodobieństwo, że w zestawie n losowo wybranych osób, jakaś para będzie miała tę samą urodzinę. Prawdopodobieństwo powinno być zaokrąglone do dwóch miejsc po przecinku.

Input

Wejście to liczba całkowita n (1 ≤ n ≤ 365), która reprezentuje liczbę osób w zestawie.

Output

Wyjście to liczba zmiennoprzecinkowa zaokrąglona do dwóch miejsc po przecinku, reprezentująca prawdopodobieństwo, że jakaś para osób w zestawie będzie miała taką samą urodzinę.

Example

Input

"23"

Output

"0.51"
