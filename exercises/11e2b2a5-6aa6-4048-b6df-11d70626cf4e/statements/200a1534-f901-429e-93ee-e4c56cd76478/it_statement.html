Paradosso del compleanno

Il paradosso del compleanno è un rompicapo sulla probabilità che riguarda la probabilità che in un gruppo di n persone scelte a caso, almeno due avranno lo stesso compleanno. Questo rompicapo è controintuitivo perché la nostra intuizione dice che la probabilità dovrebbe essere molto bassa anche quando il numero di persone è grande. Ma la probabilità effettiva è abbastanza alta anche per numeri relativamente piccoli di persone.

Task

Scrivi una funzione che prenda in input un intero n (1 ≤ n ≤ 365) e restituisca la probabilità che, in un set di n persone scelte a caso, almeno due avranno lo stesso compleanno. La probabilità dovrebbe essere arrotondata a due decimali.

Input

L'input è un intero n (1 ≤ n ≤ 365) che rappresenta il numero di persone nel set.

Output

L'output è un numero decimale arrotondato a due decimali che rappresenta la probabilità che qualche coppia di persone nel set abbia lo stesso compleanno.

Example

Input

"23"

Output

"0.51"
