Paradoxo do Aniversário

O paradoxo do aniversário é um quebra-cabeça de probabilidade que envolve encontrar a probabilidade de, em um conjunto de n pessoas escolhidas aleatoriamente, alguma delas terá o mesmo aniversário. Esse quebra-cabeça é contra-intuitivo porque nossa intuição diz que a probabilidade deve ser muito baixa mesmo quando o número de pessoas é grande. Mas a probabilidade real é bastante alta mesmo para números relativamente pequenos de pessoas.

Task

Escreva uma função que receba um inteiro n (1 ≤ n ≤ 365) como entrada e retorne a probabilidade de, em um conjunto de n pessoas escolhidas aleatoriamente, alguma delas terá o mesmo aniversário. A probabilidade deve ser arredondada para duas casas decimais.

Input

A entrada é um inteiro n (1 ≤ n ≤ 365) que representa o número de pessoas no conjunto.

Output

A saída é um número decimal arredondado para duas casas decimais que representa a probabilidade de que alguma dupla de pessoas no conjunto terá o mesmo aniversário.

Example

Input

"23"

Output

"0.51"
