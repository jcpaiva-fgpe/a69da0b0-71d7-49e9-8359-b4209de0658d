Suma tablicy

Masz daną tablicę liczb całkowitych i twoim zadaniem jest znalezienie sumy wszystkich elementów w tablicy.

Task

Napisz funkcję, która przyjmuje tablicę liczb całkowitych i zwraca sumę wszystkich elementów w tablicy.

Input

Wejściem będzie tablica liczb całkowitych.

Output

Wyjściem będzie pojedyncza liczba całkowita reprezentująca sumę wszystkich elementów w tablicy.

Example

Input

[1,2,3,4,5]

Output

15
