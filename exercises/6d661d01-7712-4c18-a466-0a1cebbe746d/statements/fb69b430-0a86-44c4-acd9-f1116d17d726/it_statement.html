Somma di un Array

Viene fornito un array di interi e il tuo compito è trovare la somma di tutti gli elementi nell'array.

Task

Scrivi una funzione che prenda un array di interi e restituisca la somma di tutti gli elementi dell'array.

Input

L'input sarà un array di interi.

Output

L'output sarà un singolo intero che rappresenta la somma di tutti gli elementi dell'array.

Example

Input

[1,2,3,4,5]

Output

15
