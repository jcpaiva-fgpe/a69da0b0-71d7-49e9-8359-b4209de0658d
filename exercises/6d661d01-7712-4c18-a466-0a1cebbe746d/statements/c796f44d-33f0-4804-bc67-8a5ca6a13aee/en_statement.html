Array Sum

You are given an array of integers and your task is to find the sum of all the elements in the array.

Task

Write a function that takes in an array of integers and returns the sum of all the elements in the array.

Input

The input will be an array of integers.

Output

The output will be a single integer representing the sum of all the elements in the array.

Example

Input

[1,2,3,4,5]

Output

15
